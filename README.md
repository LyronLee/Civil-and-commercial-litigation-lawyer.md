# 04 执行程序：生效法律文书的实现

> 🎯 **程序定位**：将生效判决、裁定、调解书、仲裁裁决等法律文书确定的权利，通过法院强制执行变为现实。
>
> 执行是诉讼的"最后一公里"，直接关系到债权人能否实际拿到钱、实现权益。

---

## 什么时候进入执行程序

| 情形 | 说明 |
|------|------|
| 一审判决生效，对方不履行 | ✅ 申请执行 |
| 二审判决生效，对方不履行 | ✅ 申请执行 |
| 调解书生效，对方不履行 | ✅ 申请执行 |
| 仲裁裁决生效，对方不履行 | ✅ 申请执行 |
| 判决尚未生效 | ❌ 需等待生效或先上诉 |

## 与相邻程序的衔接

### 从前序程序进入

| 前序程序 | 进入执行的触发条件 |
|---------|-------------------|
| 01-一审程序 | 一审判决生效，对方在履行期内未履行 |
| 02-二审程序 | 二审判决生效，对方在履行期内未履行 |
| 03-再审程序 | 再审判决生效，对方在履行期内未履行 |

### 执行中的救济路径

| 问题 | 救济路径 |
|------|---------|
| 对执行行为有异议（超标的查封等） | 02-执行审查程序 → 执行行为异议 → 10日内复议 |
| 案外人对执行标的有实体权利 | 02-执行审查程序 → 案外人异议 → 15日内异议之诉 |
| 不服追加/变更被执行人 | 02-执行审查程序 → 追加异议 → 15日内异议之诉 |
| 执行和解协议不履行 | 01-执行实施主线 → 恢复执行 或 就和解协议起诉（互斥） |
| 多个债权人执行同一被执行人 | 04-执行衍生与特殊救济 → 参与分配 |

## 关键期限

| 事项 | 期限 | 说明 |
|------|------|------|
| **申请执行** | **2年** | 判决生效/履行期届满后起算 |
| 执行立案 | 7日 | 法院收到申请后 |
| 执行通知 | 10日 | 立案后向被执行人发出 |
| 查封不动产 | 3年 | 到期前申请续封 |
| 查封动产 | 2年 | 到期前申请续封 |
| 冻结存款 | 1年 | 到期前申请续冻 |
| 执行异议审查 | 15日 | 法院审查并作出裁定 |
| 异议之诉起诉 | 15日 | 收到异议裁定后 |
| 执行复议申请 | 10日 | 收到异议裁定后 |
| 执行和解恢复执行 | 2年 | 和解协议约定履行期届满后起算 |

## 快速开始

### 第一步：准备执行依据

将生效法律文书放入 `01-执行实施主线/01-执行依据/`。

### 第二步：收集财产线索（如有）

将掌握的被执行人财产线索放入 `01-执行实施主线/02-财产线索/`。

### 第三步：唤起 AI

> **"我要申请强制执行，请帮我起草执行申请。执行依据在01-执行依据里。"**

AI 会：
1. 读取生效法律文书
2. 起草强制执行申请书
3. 建立执行进程跟踪表
4. 提示关键期限

## ❓ 常见问题

**Q1: 执行和解协议和调解书有什么区别？**
> - **调解书**：法院出具的具有强制执行力的法律文书，对方不履行可直接申请执行
> - **执行和解协议**：双方自行达成的协议，不具有强制执行力，对方不履行需申请恢复执行原生效文书或就和解协议起诉

**Q2: 终本后还能恢复执行吗？**
> 可以。终本只是暂时终结本次执行程序，发现被执行人有财产的，可随时申请恢复执行，不受2年申请执行期限限制。

**Q3: 被执行人没有财产怎么办？**
> 1. 申请悬赏执行、限制消费、纳入失信名单
> 2. 调查被执行人的到期债权、知识产权、网络财产
> 3. 申请追加/变更被执行人
> 4. 终本后持续跟踪
