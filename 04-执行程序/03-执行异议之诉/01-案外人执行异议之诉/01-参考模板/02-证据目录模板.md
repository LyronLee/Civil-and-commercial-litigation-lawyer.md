# 案外人执行异议之诉 · 证据目录

> 案外人执行异议之诉的核心是证明案外人对执行标的享有足以排除强制执行的实体权利。

---

## 证据目录

| 序号 | 证据名称 | 证明目的 | 页码 |
|------|---------|---------|------|
| 1 | 执行异议裁定书 | 前置程序已完成（案外人异议被驳回） | |
| 2 | 查封/扣押裁定书 | 执行标的查封时间 | |
| 3 | 【权属证明】如房产证/买卖合同/股权登记等 | 案外人对执行标的享有实体权利 | |
| 4 | 【时间证据】如合同签订日期、付款日期等 | 权利形成时间早于查封时间 | |
| 5 | 【占有证据】如水电费单据、物业费单据、入住证明 | 实际占有/使用执行标的 | |
| 6 | 【付款凭证】如银行转账记录、收据 | 已支付价款 | |
| 7 | 【未过户原因说明】如开发商未办证证明 | 非因买受人原因未过户 | |
| 8 | 【其他】 | | |

---

> ⚠️ **关键提示**：
> - 证据 3-7 是核心证据，必须充分
> - 权利形成时间必须早于查封时间
> - 所有证据需准备原件供庭审核对
