# 申请执行人执行异议之诉 · 证据目录

> 申请执行人执行异议之诉的核心是证明案外人的权利不足以排除执行。

---

## 证据目录

| 序号 | 证据名称 | 证明目的 | 页码 |
|------|---------|---------|------|
| 1 | 执行依据（生效判决/裁定） | 申请执行人对被执行人享有债权 | |
| 2 | 查封/扣押裁定书 | 执行标的已被合法查封 | |
| 3 | 案外人异议申请书及证据 | 案外人主张的权利及证据（用于针对性反驳） | |
| 4 | 中止执行裁定书 | 前置程序已完成（案外人异议成立） | |
| 5 | 【执行标的权属查询】 | 执行标的实际权属状况 | |
| 6 | 【反驳证据】如合同倒签鉴定、资金回流证据 | 案外人权利不真实/不足以排除执行 | |
| 7 | 【其他】 | | |

---

> ⚠️ **关键提示**：
> - 重点收集否定案外人权利的证据
> - 攻击方向：权利真实性、权利形成时间、权利充分性
