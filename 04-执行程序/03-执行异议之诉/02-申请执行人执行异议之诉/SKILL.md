# 申请执行人执行异议之诉 SKILL

> ⚠️ 依据2023年修正《民事诉讼法》（2024年1月1日施行）及《最高人民法院关于人民法院办理执行异议和复议案件若干问题的规定》。

---

## 触发条件

用户表达以下意图时触发：
- "案外人异议成立了，我要起诉继续执行"
- "申请执行人不服异议裁定"
- "准备申请执行人执行异议之诉"
- "对方阻止执行，我要恢复执行"

---

## 前置依赖

触发本SKILL前必须确认：
1. 案外人异议已被执行法院审查并裁定**成立**（中止执行）
2. 申请执行人已收到中止执行的书面裁定
3. 起诉期限未届满（收到裁定之日起**15日内**）
4. 用户身份为申请执行人

> ⚠️ **15日起诉期限是不可变期间**，逾期不起诉，异议裁定即生效，执行标的解除执行。

## Step 1：战略分析（阶段一）

### 1.1 前置程序确认

读取 `05-前置材料/` 中的执行异议裁定书，确认：
- 裁定结果是否为"案外人异议成立/中止执行"
- 裁定日期 → 计算起诉期限（收到之日起15日）
- 如期限≤3日，立即提醒用户优先完成起诉状

### 1.2 案外人权利分析

读取案外人异议申请书及证据，分析案外人主张的权利：

| 案外人可能主张的权利 | 攻击要点 |
|---------------------|---------|
| 物权期待权（第28条） | 是否满足全部四要件（合同、占有、付款、未过户原因） |
| 消费者购房人（第29条） | 是否用于居住、名下是否有其他住房、付款是否超50% |
| 担保物权（第30条） | 抵押登记是否真实、是否覆盖执行标的 |
| 租赁权 | 租赁合同是否真实、是否在查封前签订并占有 |
| 其他权利 | 权利真实性、合法性、充分性 |

### 1.3 攻击点识别

1. **权利真实性攻击**：
   - 权属证明是否虚假/伪造
   - 合同是否倒签
   - 付款凭证是否真实
   - 占有证据是否充分

2. **权利形成时间攻击**：
   - 权利形成时间是否晚于查封时间
   - 合同签订时间是否真实
   - 占有时间是否在查封前

3. **权利充分性攻击**：
   - 权利是否"足以排除执行"
   - 是否满足司法解释的全部要件

### 1.4 策略分析

1. **证据收集**：收集否定案外人权利的证据
2. **法律依据检索**：检索否定案外人权利的司法解释和案例
3. **风险评估**：胜诉概率评估
4. **改判方案**：如胜诉，恢复执行；如败诉，寻找其他财产线索

### 1.5 生成《申请执行人执行异议之诉策略方案书》

输出到 `06-阶段产出/01-策略方案书.md`，包含：
- 案外人权利类型分析
- 攻击点清单及优先级
- 证据收集计划
- 争议焦点预判
- 胜诉概率评估
- 后续路径建议

## Step 2：材料制备（阶段二）

### 2.1 起诉状

参考 `01-参考模板/00-起诉状模板.md`，起草起诉状：

- **原告**：申请执行人
- **被告**：案外人
- **共同被告/第三人**：被执行人
- **管辖**：执行法院（专属管辖）
- **诉讼请求**：准许执行【执行标的描述】

> ⚠️ **必须包含前置程序说明**：案外人提出异议，执行法院于【日期】作出【案号】裁定，中止对【标的】的执行。

### 2.2 证据目录

参考 `01-参考模板/02-证据目录模板.md`（如有），整理证据：

| 证据类别 | 证明目的 | 优先级 |
|---------|---------|--------|
| 执行依据 | 申请执行人对被执行人享有债权 | ★★★ |
| 查封文书 | 执行标的已被合法查封 | ★★★ |
| 案外人异议材料 | 案外人主张的权利及证据 | ★★☆ |
| 反驳证据 | 案外人权利不真实/不足以排除执行 | ★★★ |
| 中止执行裁定 | 前置程序已完成 | ★☆☆ |

### 2.3 反驳证据重点

| 攻击方向 | 反驳证据 |
|---------|---------|
| 合同虚假 | 笔迹鉴定、合同备案记录、付款流水真实性 |
| 占有不实 | 水电费记录、物业费记录、实际居住证据 |
| 付款虚假 | 银行流水、资金来源、是否存在回流 |
| 恶意串通 | 案外人与被执行人的关系、交易价格是否公允 |
| 未满足要件 | 逐项论证不满足司法解释的全部要件 |

## Step 3：庭审交锋（阶段三）

### 3.1 开庭提纲

参考 `01-参考模板/03-开庭提纲模板.md`（如有），生成开庭提纲：

1. **程序核对**：管辖、当事人地位
2. **前置程序确认**：案外人异议成立、执行被中止
3. **攻击方案**：
   - 第一步：攻击案外人权利的真实性
   - 第二步：攻击权利形成时间
   - 第三步：论证权利不足以排除执行
4. **争议焦点预判**：
   - 案外人权利是否真实
   - 权利形成时间是否早于查封
   - 权利是否"足以排除执行"
5. **法官发问预判**：
   - "你认为案外人的权利有哪些漏洞？"
   - "有哪些证据证明案外人与被执行人串通？"
   - "查封时间是什么时候？"

### 3.2 质证意见

针对案外人提交的证据，逐项质证：
- **权属证明**：真实性、合法性、关联性质证
- **合同**：是否倒签、是否虚假、是否网签备案
- **付款凭证**：银行流水是否真实、是否存在资金回流
- **占有证据**：水电费、物业费、实际使用证据

### 3.3 庭审纪要

开庭后24小时内生成庭审纪要，记录：
- 程序事项
- 法官关注重点（特别关注法官对案外人权利的态度）
- 案外人新观点和证据
- 法官口头要求

### 3.4 代理词

参考 `01-参考模板/04-代理词模板.md`（如有），提交书面代理词：
- 围绕"案外人的权利不足以排除强制执行"展开
- 逐项论证案外人权利不满足司法解释的要件
- 如存在恶意串通证据，重点论证

---

## Step 4：收官（阶段四）

### 4.1 判决分析

- **胜诉**：恢复对执行标的的执行
- **败诉**：执行标的解除执行，需寻找其他财产线索或追加被执行人

### 4.2 后续路径

| 结果 | 后续路径 | 期限 |
|------|---------|------|
| 胜诉 | 恢复执行 | — |
| 败诉 | 上诉 | 判决15日/裁定10日 |
| 败诉+不上诉 | 寻找其他财产线索或追加被执行人 | — |

## 质量检查清单

生成全部文件后，逐项核对：

- [ ] 前置程序是否已完成（案外人异议成立、收到中止执行裁定）
- [ ] 起诉期限是否在收到裁定之日起15日内
- [ ] 管辖法院是否为执行法院（专属管辖）
- [ ] 当事人地位是否正确（原告：申请执行人，被告：案外人）
- [ ] 诉讼请求是否为"准许执行"
- [ ] 起诉状是否包含前置程序说明
- [ ] 反驳证据是否围绕案外人权利的真实性、时间、充分性展开
- [ ] 是否收集了否定案外人权利的关键证据
- [ ] 法条引用是否准确（《执行异议复议规定》第28-31条）
- [ ] 全部文件编码为UTF-8，中文显示正常
- [ ] 所有文件生成后标注"初稿，需律师审核后使用"

- 所有文书按申请执行人执行异议之诉的独立结构生成
