# 前置材料

本文件夹用于存放提起申请执行人执行异议之诉前的前置程序材料。

## 应放入的材料

| 材料名称 | 说明 | 必要性 |
|---------|------|--------|
| **案外人执行异议申请书** | 案外人向执行法院提交的异议申请书及证据 | 必须 |
| **执行异议裁定书（成立）** | 执行法院作出的支持案外人异议、中止执行的裁定 | 必须 |
| **执行依据** | 生效判决书/裁定书/调解书/仲裁裁决等 | 必须 |
| **查封/扣押/冻结裁定** | 执行法院对执行标的作出的查封裁定 | 必须 |
| **执行通知书** | 执行法院向被执行人发出的执行通知书 | 建议 |
| **执行标的权属材料** | 查封时掌握的执行标的权属信息 | 建议 |

## 使用方式

1. 在唤起 AI 前，将上述材料按顺序放入本文件夹
2. AI 会读取全部前置材料，确认前置程序已完成
3. 重点核对：执行异议裁定是否为"支持案外人异议/中止执行"裁定

## 注意事项

- 提起执行异议之诉的前提：**案外人异议已被裁定成立，执行被中止**
- 起诉期限：收到中止执行裁定之日起 **15 日内**（不可变期间）
- 管辖法院：执行法院（专属管辖）
