# 前置材料

本文件夹用于存放提起追加/变更被执行人异议之诉前的前置程序材料。

## 应放入的材料

| 材料名称 | 说明 | 必要性 |
|---------|------|--------|
| **追加/变更被执行人申请书** | 申请执行人提交的追加/变更申请书及证据 | 必须 |
| **追加/变更被执行人裁定书** | 执行法院作出的追加或驳回追加的裁定 | 必须 |
| **执行依据** | 生效判决书/裁定书/调解书/仲裁裁决等 | 必须 |
| **终本裁定** | 终结本次执行程序的裁定（如主张加速到期） | 视情形 |
| **公司章程/工商登记** | 被追加公司的章程、股东信息 | 视情形 |
| **财务账册** | 一人公司财务账册（如主张人格否认） | 视情形 |

## 使用方式

1. 在唤起 AI 前，将上述材料按顺序放入本文件夹
2. AI 会读取全部前置材料，确认前置程序已完成
3. 明确角色：是被追加人不服追加，还是申请执行人不服驳回追加

## 注意事项

- 起诉期限：收到追加/驳回追加裁定之日起 **15 日内**（不可变期间）
- 管辖法院：执行法院（专属管辖）
- 追加事由严格限定于《变更追加当事人规定》第 14-24 条，不能随意扩大
