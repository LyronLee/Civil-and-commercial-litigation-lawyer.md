# 追加/变更被执行人异议之诉 SKILL

> ⚠️ 依据2023年修正《民事诉讼法》（2024年1月1日施行）及《最高人民法院关于民事执行中变更、追加当事人若干问题的规定》。

---

## 触发条件

用户表达以下意图时触发：
- "不服追加被执行人的裁定"
- "申请追加被执行人被驳回了"
- "准备追加被执行人异议之诉"
- "被追加为被执行人，要起诉"

---

## 前置依赖

触发本SKILL前必须确认：
1. 已收到追加/变更被执行人的书面裁定（追加或驳回）
2. 起诉期限未届满（收到裁定之日起**15日内**）
3. 已明确用户角色：
   - **被追加人**：不服被追加为被执行人
   - **申请执行人**：不服驳回追加申请

> ⚠️ **15日起诉期限是不可变期间**，逾期不起诉，追加/变更裁定即生效。

## Step 1：战略分析（阶段一）

### 1.1 角色确认与追加事由

读取 `05-前置材料/` 中的追加/变更裁定，确认：
- 用户角色：被追加人 或 申请执行人
- 追加/变更的法律依据（《变更追加当事人规定》第14-24条的哪一条）
- 裁定日期 → 计算起诉期限（收到之日起15日）

### 1.2 法定追加事由速查

| 条款 | 追加情形 | 核心要件 |
|------|---------|---------|
| 第13条 | 被执行人死亡/注销 | 继承人/权利义务承受人 |
| 第14条 | 个人独资企业 | 不能清偿债务 → 追加出资人 |
| 第15条 | 合伙企业 | 不能清偿债务 → 追加普通合伙人 |
| 第16条 | 法人分支机构 | 不能清偿债务 → 追加设立该分支机构的法人 |
| 第17条 | 有限责任公司 | 财产不足以清偿 → 追加未缴纳/未足额缴纳出资的股东 |
| 第18条 | 有限责任公司 | 财产不足以清偿 → 追加抽逃出资的股东 |
| 第19条 | 一人公司 | 财产不足以清偿 → 追加股东（举证责任倒置） |
| 第20条 | 公司未经清算即注销 | 追加股东/董事/控股股东为被执行人 |
| 第21条 | 公司解散后无偿接受财产 | 追加接受财产的股东/主管部门 |
| 第22条 | 第三人书面承诺 | 第三人自愿代履行 |

> ⚠️ **关键判断**：追加/变更被执行人的法定事由**严格限定**于上述条款，不能随意扩大。

### 1.3 攻防策略（按角色）

**如用户为被追加人（防守方）**：
1. **出资情况论证**：已实际出资、出资期限未届满（非破产情形）
2. **财务独立性论证**（一人公司）：财务账册完整、财产独立
3. **程序抗辩**：追加程序是否合法、是否经过听证
4. **证据收集**：出资凭证、验资报告、财务审计报告

**如用户为申请执行人（进攻方）**：
1. **追加事由论证**：证明符合法定追加情形
2. **证据收集**：工商登记、出资记录、财务混同证据
3. **攻击点**：股东抽逃出资、人格混同、恶意转移财产

### 1.4 风险评估

- 被追加人胜诉概率：取决于出资是否真实、财务是否独立
- 申请执行人胜诉概率：取决于证据是否充分、追加事由是否成立

### 1.5 生成《追加/变更被执行人异议之诉策略方案书》

输出到 `06-阶段产出/01-策略方案书.md`，包含：
- 角色定位和追加事由
- 攻防策略（按角色）
- 证据收集计划
- 争议焦点预判
- 胜诉概率评估
- 后续路径建议

## Step 2：材料制备（阶段二）

### 2.1 起诉状

参考 `01-参考模板/00-起诉状模板.md`，起草起诉状：

**如被追加人不服追加**：
- **原告**：被追加人
- **被告**：申请执行人
- **诉讼请求**：不得追加为被执行人

**如申请执行人不服驳回追加**：
- **原告**：申请执行人
- **被告**：被追加人
- **诉讼请求**：追加【被追加人】为被执行人，在【范围】内承担责任

- **管辖**：执行法院（专属管辖）

> ⚠️ **必须包含前置程序说明**：收到追加/驳回追加的裁定（案号、日期）。

### 2.2 证据目录

参考 `01-参考模板/02-证据目录模板.md`（如有），整理证据：

**被追加人角度**：

| 证据类别 | 证明目的 |
|---------|---------|
| 出资凭证 | 已实际缴纳出资 |
| 验资报告 | 出资经第三方验证 |
| 财务账册/审计报告 | 一人公司财产独立 |
| 公司章程 | 出资期限尚未届满 |
| 追加裁定 | 前置程序已完成 |

**申请执行人角度**：

| 证据类别 | 证明目的 |
|---------|---------|
| 工商登记 | 股东身份、出资额、出资期限 |
| 银行流水 | 抽逃出资的资金流向 |
| 财务混同证据 | 人格混同、财产不独立 |
| 终本裁定 | 被执行人财产不足以清偿 |
| 驳回追加裁定 | 前置程序已完成 |

## Step 3：庭审交锋（阶段三）

### 3.1 开庭提纲

参考 `01-参考模板/03-开庭提纲模板.md`（如有），生成开庭提纲：

**被追加人角度**：
1. **程序核对**
2. **出资事实陈述**：已实际出资/出资期限未届满
3. **财务独立性陈述**（一人公司）：财务账册完整、审计报告
4. **争议焦点预判**：
   - 是否已实际出资
   - 出资期限是否届满
   - 一人公司财产是否独立
5. **法官发问预判**：
   - "实际出资多少？"
   - "出资方式是什么？"
   - "财务账册是否完整？"

**申请执行人角度**：
1. **追加事由陈述**：符合法定追加情形
2. **证据出示**：工商登记、资金流向、财务混同
3. **争议焦点预判**：
   - 是否存在抽逃出资
   - 是否存在人格混同
   - 是否满足加速到期条件

### 3.2 质证意见

**被追加人**：攻击对方证据的真实性、完整性
**申请执行人**：强调对方证据不足以证明其主张

### 3.3 庭审纪要

开庭后24小时内生成庭审纪要。

### 3.4 代理词

参考 `01-参考模板/04-代理词模板.md`（如有），提交书面代理词：
- 围绕追加事由是否成立展开
- 引用《变更追加当事人规定》具体条款

---

## Step 4：收官（阶段四）

### 4.1 判决分析

| 角色 | 胜诉 | 败诉 |
|------|------|------|
| 被追加人 | 不被追加，执行继续针对原被执行人 | 被追加为被执行人，在裁定范围内承担责任 |
| 申请执行人 | 追加成功，执行范围扩大 | 追加失败，需寻找其他救济路径 |

### 4.2 后续路径

| 结果 | 后续路径 | 期限 |
|------|---------|------|
| 胜诉 | 按判决结果执行 | — |
| 败诉 | 上诉 | 判决15日/裁定10日 |

## 质量检查清单

生成全部文件后，逐项核对：

- [ ] 前置程序是否已完成（收到追加/驳回追加的裁定）
- [ ] 起诉期限是否在收到裁定之日起15日内

- [ ] 诉讼请求是否明确（不得追加 或 追加并承担责任）

- [ ] 追加事由是否对应《变更追加当事人规定》第14-24条
- [ ] 证据是否围绕追加事由/抗辩展开
- [ ] 出资凭证、财务账册等关键证据是否详细列明
- [ ] 法条引用是否准确（《变更追加当事人规定》）
- [ ] 全部文件编码为UTF-8，中文显示正常
- [ ] 所有文件生成后标注"初稿，需律师审核后使用"
