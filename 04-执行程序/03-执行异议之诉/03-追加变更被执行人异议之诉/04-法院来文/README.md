# 法院来文

本文件夹用于存放追加/变更被执行人异议之诉过程中，法院发出的各类文书。

## 应放入的材料

| 材料名称 | 说明 |
|---------|------|
| **受理通知书** | 法院受理执行异议之诉的通知 |
| **传票** | 开庭传票（含开庭时间、地点） |
| **举证通知书** | 法院指定的举证期限 |
| **缴费通知书** | 诉讼费缴纳通知（如有） |
| **判决书/裁定书** | 执行异议之诉的裁判文书 |
| **生效证明** | 判决生效证明 |

## 使用方式

1. 收到法院文书后，及时存入本文件夹
2. AI 会读取法院来文中的关键期限（举证期限、开庭时间等）
3. 收到判决书后，根据结果确定后续路径（继续执行/追加成功/上诉）

## 注意事项

- 追加/变更被执行人异议之诉按一审程序审理，对判决不服可在 **15 日内**上诉
- 对裁定不服可在 **10 日内**上诉
- 追加成功后，执行范围扩大至被追加人名下财产
