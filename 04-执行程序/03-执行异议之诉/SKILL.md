# 03 执行异议之诉：实体权利的诉讼救济

> 核心理念：执行异议之诉是执行程序中产生的独立诉讼类型。虽诉讼结构与一审类似，但在管辖（执行法院专属管辖）、前置程序（必须先经执行异议）、起诉期限（15日不变期间）、当事人地位、诉讼请求、争议焦点等方面均有显著差异。各子类型 SKILL.md 包含独立完整的四阶段工作流，不再简单引用一审框架。

## 前置检查

| 检查项 | 合格标准 | 不满足时的替代路径 |
|--------|---------|-------------------|
| 是否已收到执行异议裁定 | 已收到书面裁定 | 未收到 → 提示先申请执行异议 |
| 起诉期限是否届满 | 收到裁定之日起 **15 日内** | 超期 → 裁定生效，告知其他救济路径 |
| 是否属于四种类型之一 | 已识别为案外人/申请执行人/追加变更/分配方案异议之诉 | 不确定 → 协助识别类型 |

> ⚠️ **15 日起诉期限是不可变期间**，逾期不起诉，异议裁定即生效，丧失执行异议之诉的起诉权。

## 阶段路由

```
收到执行异议裁定
    │
    ├─ 起诉期限已届满（>15日）───────→ 裁定生效，告知其他救济路径
    │
    ├─ 案外人异议被驳回 ─────────────→ 01-案外人执行异议之诉/
    │                                      （原告：案外人，被告：申请执行人）
    │
    ├─ 案外人异议成立，申请执行人不服 ─→ 02-申请执行人执行异议之诉/
    │                                      （原告：申请执行人，被告：案外人）
    │
    ├─ 不服追加/变更被执行人裁定 ──────→ 03-追加变更被执行人异议之诉/
    │                                      （原告/被告因情形而异）
    │
    └─ 对执行分配方案异议被反对 ───────→ 04-执行分配方案异议之诉/
                                           （原告：异议人，被告：反对人）
```

## 四种子类型速查

| 子类型 | 原告 | 被告 | 核心争议 | 前置程序 | 关键法条 |
|--------|------|------|---------|---------|---------|
| **案外人执行异议之诉** | 案外人 | 申请执行人 | 案外人是否享有足以排除执行的实体权利 | 案外人异议被驳回 | 《执行异议复议规定》第 28-31 条 |
| **申请执行人执行异议之诉** | 申请执行人 | 案外人 | 案外人的权利是否不足以排除执行 | 案外人异议成立 | 同上 |
| **追加/变更被执行人异议之诉** | 被追加人/申请人 | 对方 | 追加/变更是否符合法定情形 | 追加/变更裁定 | 《变更追加当事人规定》第 14-24 条 |
| **执行分配方案异议之诉** | 异议人 | 反对人 | 分配方案是否正确 | 对分配方案异议被反对 | 《民事诉讼法解释》第 506-510 条 |

## 通用工作流

### Step 1：异议之诉类型识别

通过案件情况判断属于四种类型中的哪一种（参见上表）。

### Step 2：前置程序检查

确认：
1. 已提出执行异议（执行行为异议/案外人异议/追加异议/分配方案异议）
2. 已收到执行法院作出的裁定
3. 起诉期限：收到裁定之日起 **15 日内**
4. 如期限≤3日，立即提醒用户优先完成起诉状

### Step 3：互斥规则检查

检查是否存在以下互斥情形：
- 案外人同时考虑提起第三人撤销之诉？
- 如存在，提醒用户根据《民事诉讼法解释》第 301 条只能择一启动
- 协助用户评估两种路径的优劣，选择最优方案

### Step 4：进入子类型工作流

根据识别的类型，进入对应子类型文件夹：

| 子类型 | 路径 | 核心任务 |
|--------|------|---------|
| 案外人执行异议之诉 | `01-案外人执行异议之诉/SKILL.md` | 证明案外人对执行标的享有足以排除执行的实体权利 |
| 申请执行人执行异议之诉 | `02-申请执行人执行异议之诉/SKILL.md` | 证明案外人对执行标的不享有足以排除执行的实体权利 |
| 追加/变更被执行人异议之诉 | `03-追加变更被执行人异议之诉/SKILL.md` | 证明是否应当追加/变更被执行人 |
| 执行分配方案异议之诉 | `04-执行分配方案异议之诉/SKILL.md` | 证明分配方案不公平 |

## 与一审程序的核心差异（参照适用要点）

执行异议之诉虽与一审诉讼结构类似，但以下方面存在显著差异，各子类型 SKILL.md 已据此设计独立工作流：

| 差异项 | 执行异议之诉 | 一审程序 |
|--------|-------------|---------|
| 管辖 | **执行法院专属管辖** | 被告住所地/合同履行地等 |
| 前置程序 | **必须先经执行异议程序** | 无 |
| 起诉期限 | **收到裁定之日起 15 日** | 3 年诉讼时效 |
| 诉讼请求 | 围绕排除执行/准许执行/追加/分配 | 给付/确认/形成 |
| 争议焦点 | 实体权利是否足以排除执行 | 原纠纷的事实和法律 |
| 执行关系 | 审理期间**原则上不停止执行** | 独立诉讼 |

## 质量检查清单

生成全部文件后，逐项核对：

- [ ] 前置程序是否已完成（已提出异议并收到裁定）
- [ ] 起诉期限是否在收到裁定之日起 15 日内
- [ ] 管辖法院是否为执行法院（专属管辖）
- [ ] 当事人地位是否正确（按异议之诉类型调整原告/被告/第三人）
- [ ] 诉讼请求是否围绕"足以排除强制执行的民事权益"展开
- [ ] 起诉状是否包含前置程序说明（已提出异议并收到裁定）
- [ ] 是否排除了互斥情形（未同时启动第三人撤销之诉，《民事诉讼法解释》第 301 条）
- [ ] 证据是否围绕实体权利展开（权利取得时间、权利性质、占有/登记状态等）
- [ ] 法条引用是否准确（2023 年版《民事诉讼法》及执行异议复议规定第 28-31 条）
- [ ] 全部文件编码为 UTF-8，中文显示正常
- [ ] 所有文件生成后标注"初稿，需律师审核后使用"
