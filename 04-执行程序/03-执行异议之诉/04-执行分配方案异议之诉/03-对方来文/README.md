# 对方来文

本文件夹用于存放执行分配方案异议之诉中，对方当事人（反对人）提交的材料。

## 应放入的材料

| 材料名称 | 说明 |
|---------|------|
| **答辩状** | 反对人提交的答辩状 |
| **债权证据** | 反对人提交的债权数额和性质证明 |
| **优先权证据** | 反对人主张的优先受偿权证明 |
| **代理词** | 对方的代理词（如有） |
| **补充意见** | 庭后补充的书面意见 |

## 使用方式

1. 收到对方材料后，及时扫描/复印并存入本文件夹
2. AI 在准备质证意见和庭审材料时会读取本文件夹内容
3. 重点分析对方债权的真实性、数额计算、优先权成立时间

## 注意事项

- 可能涉及多个债权人之间的利益冲突
- 分配方案的计算必须逐项核对，数字准确
- 注意优先受偿权的成立时间和范围
