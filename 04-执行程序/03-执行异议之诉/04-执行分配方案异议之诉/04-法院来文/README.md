# 法院来文

本文件夹用于存放执行分配方案异议之诉过程中，法院发出的各类文书。

## 应放入的材料

| 材料名称 | 说明 |
|---------|------|
| **受理通知书** | 法院受理执行异议之诉的通知 |
| **传票** | 开庭传票（含开庭时间、地点） |
| **举证通知书** | 法院指定的举证期限 |
| **缴费通知书** | 诉讼费缴纳通知（如有） |
| **判决书** | 执行分配方案异议之诉的判决书 |
| **重新分配方案** | 法院根据判决重新制作的分配方案 |

## 使用方式

1. 收到法院文书后，及时存入本文件夹
2. AI 会读取法院来文中的关键期限（举证期限、开庭时间等）
3. 收到判决书后，按判决结果执行分配

## 注意事项

- 执行分配方案异议之诉**一审终审**，判决**不可上诉**
- 判决生效后，执行法院根据判决结果重新制作分配方案
- 所有债权人按新的分配方案受偿
