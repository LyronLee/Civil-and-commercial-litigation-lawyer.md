## 🚀 执行实施主线的完整流程

### 阶段一：申请执行

**申请执行期限**：
- 判决/裁定：生效后 **2 年内**
- 如分期履行：每期履行期限届满后 **2 年内**
- 如未规定履行期：生效后 **2 年内**

> ⏰ **注意**：超过2年申请执行的，法院不予受理。

**申请执行需提交的材料**：
1. 强制执行申请书
2. 生效法律文书原件/复印件
3. 申请人身份证明
4. 被执行人财产线索（如有）
5. 委托手续（如委托律师）

### 阶段二：财产调查与控制

法院依职权调查 + 申请人提供线索：

| 调查措施 | 适用情形 |
|---------|---------|
| 网络查控 | 法院通过执行查控系统查询银行存款、房产、车辆、股权等 |
| 现场调查 | 到被执行人住所地、经营场所调查 |
| 审计调查 | 被执行人为企业，需查清财产状况时 |
| 悬赏执行 | 财产线索难以查找时 |

**财产控制措施**：

| 措施 | 适用对象 | 期限 |
|------|---------|------|
| 查封 | 不动产、动产 | 不动产3年，动产2年 |
| 扣押 | 动产 | 同查封 |
| 冻结 | 存款、股权、债权 | 存款1年，股权3年 |

### 阶段三：财产处置

控制财产后，进入处置阶段：

| 处置方式 | 适用情形 | 程序 |
|---------|---------|------|
| 拍卖 | 查封/扣押的财产 | 网络司法拍卖优先 |
| 变卖 | 拍卖流拍或不宜拍卖 | 双方协商价格或询价 |
| 以物抵债 | 双方同意 | 不经拍卖直接抵债 |
| 直接扣划 | 冻结的存款 | 直接扣划至法院账户 |

### 阶段四：执行完毕或终结

| 结果类型 | 适用情形 |
|---------|---------|
| **执行完毕** | 全部执行到位 |
| **终结本次执行** | 未发现可供执行财产，申请人同意或法院依职权 |
| **终结执行** | 据以执行的法律文书被撤销、被执行人死亡无遗产等 |
| **执行和解** | 双方达成和解协议 |

> 💡 **终本 ≠ 终结**：终结本次执行后，发现被执行人有财产的，可随时申请恢复执行。

### 执行和解成功后的处理

如双方达成执行和解协议并**履行完毕**：
1. 执行法院出具执行完毕证明
2. 执行程序终结
3. 进入执行结案流程

如双方达成执行和解协议但**尚未履行完毕**：
- 持续跟踪履行进度
- 在 `05-阶段产出/04-执行进程跟踪表.md` 中标注"执行和解履行中"
- 如对方不履行 → 在和解协议约定履行期届满后 2 年内申请恢复执行

> ⚠️ **执行和解协议不具有强制执行力**。如担心对方不履行，建议：
> - 将和解协议转化为法院调解书（经执行法院确认）
> - 或就和解协议办理公证债权文书
> - 或在和解协议中设置担保条款

## 📬 执行中的期限管理

| 期限事项 | 法定期限 | 备注 |
|---------|---------|------|
| 申请执行 | 2年 | 从法律文书规定履行期的最后一日起算 |
| 执行立案 | 7日内 | 法院收到申请后7日内立案 |
| 执行通知 | 10日内 | 立案后10日内向被执行人发出执行通知 |
| 查封不动产 | 3年 | 到期前申请续封 |
| 查封动产 | 2年 | 到期前申请续封 |
| 冻结存款 | 1年 | 到期前申请续冻 |
| 执行和解履行 | 按协议约定 | 如不履行，申请恢复执行（2年内） |

## ❓ 常见问题

**Q1: 被执行人没有财产怎么办？**
> 1. 申请法院采取悬赏执行、限制消费、纳入失信名单等措施
> 2. 调查被执行人的到期债权、知识产权、网络财产等
> 3. 申请追加/变更被执行人（如股东出资不实、人格混同等）
> 4. 终本后持续跟踪，发现财产立即申请恢复执行

**Q2: 执行和解协议有什么风险？**
> - 执行和解不具有强制执行力，如对方不履行，需申请恢复执行原生效法律文书
> - 建议将和解协议转化为调解书（经法院确认）或公证债权文书，赋予强制执行力
> - 和解协议中可约定担保条款，增加履约保障

**Q3: 终本后还能恢复执行吗？**
> 可以。终本只是暂时终结本次执行程序，不是执行完毕。发现被执行人有可供执行财产的，可随时向法院申请恢复执行，不受2年申请执行期限限制。

**Q4: 网络查控能查到哪些财产？**
> 法院执行查控系统通常可查询：银行存款、不动产、车辆、股权、证券、网络资金（支付宝/微信等）。但部分财产（如境外资产、隐名财产）可能无法通过网络查控发现，需要申请人提供线索。
