# 04-法院来文

本文件夹用于存放执行过程中法院送达的各类文书。

## 应放入的材料

| 材料名称 | 说明 |
|---------|------|
| 受理通知书 | 法院受理执行申请的通知 |
| 执行通知书 | 向被执行人发出的履行通知 |
| 财产查控结果 | 法院网络查控系统反馈的财产信息 |
| 查封/冻结/扣押裁定 | 法院作出的财产控制裁定 |
| 拍卖/变卖裁定 | 法院作出的财产处置裁定 |
| 执行款发放凭证 | 执行款发放的相关凭证 |
| 执行完毕/终本裁定 | 执行程序终结的裁定 |

## 使用方式

1. 收到法院文书后，及时存入本文件夹
2. 在文件名中标注收到日期
3. 唤起 AI 时，AI 会读取本文件夹内容并跟踪执行进度

## 注意事项

- 收到**执行通知书**后，跟踪被执行人是否按期履行
- 收到**财产查控结果**后，评估是否需要补充财产线索
- 收到**查封/冻结裁定**后，注意查封期限（不动产3年/动产2年/存款1年），到期前及时申请续封
- 收到**终本裁定**后，了解恢复执行的路径和条件
