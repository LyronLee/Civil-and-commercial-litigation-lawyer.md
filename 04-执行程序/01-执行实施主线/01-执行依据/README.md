# 01-执行依据

本文件夹用于存放申请强制执行所依据的生效法律文书。

## 应放入的材料

| 材料类型 | 说明 | 必要性 |
|---------|------|--------|
| 判决书 | 一审/二审/再审判决书（已生效） | 必须 |
| 调解书 | 法院出具的调解书（已生效） | 必须 |
| 仲裁裁决书 | 仲裁机构作出的裁决（已生效） | 必须 |
| 公证债权文书 | 经公证的具有强制执行效力的债权文书 | 必须 |
| 生效证明 | 证明法律文书已生效的材料（如送达回证） | 建议 |

## 使用方式

1. 将生效法律文书原件/复印件放入本文件夹
2. 确认法律文书确定的履行期限已届满
3. 唤起 AI 时，AI 会读取本文件夹中的文书，确认执行依据和管辖法院

## 注意事项

- 执行依据**必须已生效**，未生效的法律文书不能申请执行
- 注意申请执行期限：**2 年**（从法律文书规定履行期间的最后一日起算）
- 不同类型的执行依据可能有不同的管辖规则（仲裁裁决由中级法院管辖）
