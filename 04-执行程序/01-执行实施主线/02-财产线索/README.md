# 02-财产线索

本文件夹用于存放申请人掌握的被执行人财产线索。

## 应放入的材料

| 财产类型 | 线索示例 | 价值 |
|---------|---------|------|
| 银行存款 | 开户行、账号、余额信息 | 可直接扣划 |
| 房产 | 房产证号、坐落地址、面积 | 可查封拍卖 |
| 车辆 | 车牌号、车辆型号、停放地点 | 可查封扣押 |
| 股权 | 持股公司名称、持股比例 | 可冻结拍卖 |
| 到期债权 | 债务人对第三人的债权信息 | 可冻结提取 |
| 知识产权 | 专利号、商标、著作权 | 可冻结评估 |
| 网络财产 | 支付宝/微信/网络店铺信息 | 可冻结扣划 |

## 使用方式

1. 将已知的财产线索材料放入本文件夹
2. 线索形式不限：照片、截图、合同、银行流水、工商登记信息等
3. 唤起 AI 时，AI 会评估线索价值并协助起草财产查控措施申请书

## 注意事项

- 财产线索**越详细越好**，包括财产类型、所在地、权属状况、是否已被查封
- 如财产已被其他法院查封，需了解查封顺位
- 法院网络查控系统可自动查询部分财产，但申请人提供的线索能提高执行效率
- 发现新的财产线索后，随时补充到本文件夹并申请查控
