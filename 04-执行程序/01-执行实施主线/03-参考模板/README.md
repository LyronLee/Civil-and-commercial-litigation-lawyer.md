# 参考模板

本文件夹存放执行实施主线的参考模板，供 AI 生成阶段产出时参照使用。

## 模板清单

| 序号 | 模板名称 | 用途 |
|------|---------|------|
| 00-执行申请书模板.md | 申请法院强制执行 | 申请事项、执行依据、被执行人信息、财产线索 |
| 01-财产线索清单模板.md | 整理被执行人财产信息 | 财产类型、所在地、权属状况、查控措施建议 |
| 02-查控措施申请模板.md | 申请查封/冻结/扣押财产 | 财产描述、控制措施、法律依据 |
| 03-限制消费申请书模板.md | 申请对被执行人限制消费 | 适用条件、限制内容、法律依据 |
| 04-纳入失信被执行人名单申请书模板.md | 申请将被执行人纳入失信名单 | 6种适用条件、惩戒措施 |
| 05-执行和解协议模板.md | 双方达成执行和解 | 履行金额/期限/方式、担保、违约责任、风险提示 |

## 使用方式

1. 本文件夹的模板为**参照模板**，AI 在执行 `SKILL.md` 工作流时会自动调用
2. 用户无需手动填写模板，将生效法律文书放入 `01-执行依据/` 后，唤起 AI 即可
3. 如需了解模板格式，可直接打开查看

## 工作流分支

| 触发条件 | 生成分支 |
|---------|---------|
| 需申请强制执行 | 分支A：执行申请书 |
| 有财产线索需查控 | 分支B：财产查控措施申请 |
| 双方有意和解 | 分支C：执行和解协议 |
| 需信用惩戒 | 分支D：限制消费/纳入失信申请 |
| 被执行人下落不明 | 分支E：悬赏执行申请 |

## 产出位置

AI 根据本模块 SKILL.md 生成的文件保存在 `05-阶段产出/` 中。
