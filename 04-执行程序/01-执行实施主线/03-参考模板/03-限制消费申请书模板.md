```
事实与理由：

申请执行人与被执行人【案由】纠纷一案，业经【法院名称】审理，于【日期】作出（【案号】）民事判决/调解书，判决/确认【判决主文内容】。

该判决/调解书已于【日期】生效，被执行人应于【日期】前履行【义务内容】。申请执行人已于【日期】向贵院申请强制执行，贵院已依法立案受理（执行案号：【案号】）。

执行过程中，贵院已向被执行人发出执行通知书，责令其在指定期限内履行义务。但被执行人至今未履行/未完全履行生效法律文书确定的给付义务。

根据《中华人民共和国民事诉讼法》第二百六十六条及《最高人民法院关于限制被执行人高消费及有关消费的若干规定》第一条之规定，被执行人未按执行通知书指定的期间履行生效法律文书确定的给付义务的，人民法院可以对其采取限制消费措施。

为维护申请执行人的合法权益，促使被执行人尽快履行义务，特申请贵院依法对被执行人及其相关责任人员采取限制消费措施。

此致

【执行法院名称】

                                    申请执行人：【签章】

                                    日期：

附：
1. 申请执行人身份证明/营业执照复印件
2. 生效法律文书复印件
3. 执行案件受理通知书/执行裁定书复印件
```

## 限制消费措施内容

根据《最高人民法院关于限制被执行人高消费及有关消费的若干规定》第 3 条，被限制消费后不得有以下行为：

| 类别 | 限制内容 |
|------|---------|
| 交通 | 乘坐飞机、列车软卧、轮船二等以上舱位 |
| 住宿 | 在星级以上宾馆、酒店、夜总会、高尔夫球场等场所进行高消费 |
| 购房 | 购买不动产或者新建、扩建、高档装修房屋 |
| 租赁 | 租赁高档写字楼、宾馆、公寓等场所办公 |
| 购车 | 购买非经营必需车辆 |
| 旅游 | 旅游、度假 |
| 教育 | 子女就读高收费私立学校 |
| 理财 | 支付高额保费购买保险理财产品 |
| 其他 | 乘坐 G 字头动车组列车全部座位、其他动车组列车一等以上座位 |

---

## 重要提示

1. **申请主体**：申请执行人申请，或法院依职权采取
2. **适用条件**：被执行人未按执行通知书指定的期间履行生效法律文书确定的给付义务
3. **限制对象**：被执行人；单位被执行人的法定代表人、主要负责人、影响债务履行的直接责任人员、实际控制人
4. **解除条件**：被执行人履行完毕义务 / 提供确实有效的担保 / 申请执行人同意解除
5. **违反后果**：违反限制消费令的，可予以罚款、拘留；构成犯罪的，依法追究刑事责任

## 关联法条

- 《民事诉讼法》第 266 条：被执行人不履行法律文书确定的义务的，人民法院可以对其采取限制出境、罚款、拘留等措施
- 《最高人民法院关于限制被执行人高消费及有关消费的若干规定》第 1-11 条
