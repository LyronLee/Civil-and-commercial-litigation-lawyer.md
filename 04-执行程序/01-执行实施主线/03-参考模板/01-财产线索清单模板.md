# 财产线索清单

> 申请执行时或执行过程中向法院提供的被执行人财产线索。

| 序号 | 财产类型 | 财产描述 | 具体信息 | 预估价值 | 线索来源 |
|------|---------|---------|---------|---------|---------|
| 1 | 银行存款 | 开户行+账号 | | | |
| 2 | 不动产 | 房屋坐落 | | | |
| 3 | 动产 | 车辆/设备等 | | | |
| 4 | 股权 | 公司名称+持股比例 | | | |
| 5 | 应收账款 | 债务人+金额 | | | |
| 6 | 其他 | | | | |

---

## 特别说明

- **银行存款**：尽量提供具体开户行和账号
- **不动产**：提供坐落地址、产权证号
- **车辆**：提供车牌号、车辆登记信息
- **股权**：提供目标公司工商登记信息
- **应收账款**：提供合同、发票等证明材料

---

## 调查申请

如申请人无法获取被执行人财产信息，可申请法院依职权调查：
- 银行账户及流水
- 不动产登记信息
- 车辆登记信息
- 工商登记及股权信息
- 社保/公积金缴纳信息
