# 强制执行申请书

```
申请执行人：【姓名/名称】
住所地：
法定代表人/负责人：

被执行人：【姓名/名称】
住所地：
法定代表人/负责人：

申请事项：
1. 强制执行【法院名称】（【案号】）民事判决/调解书确定的义务；
2. 被执行人向申请执行人支付【具体金额或行为】；
3. 被执行人加倍支付迟延履行期间的债务利息；
4. 被执行人承担本案执行费用。

事实与理由：
申请执行人与被执行人【案由】纠纷一案，业经【法院名称】审理，于【日期】作出（【案号】）民事判决/调解书，判决/确认【判决主文内容】。

该判决/调解书已于【日期】生效，被执行人应于【日期】前履行【义务内容】，但被执行人至今未履行/未完全履行。

为维护申请执行人的合法权益，依据《中华人民共和国民事诉讼法》第247条之规定，特申请强制执行。

此致
【执行法院名称】

申请执行人：【签章】
日期：

附：
1. 申请执行人身份证明
2. 生效法律文书副本
3. 法律文书生效证明
4. 被执行人财产线索（如有）
```

---

## 撰写要点

1. **执行依据必须明确**：写明案号、判决日期、生效日期
2. **申请事项具体**：金额要明确，行为要可执行
3. **注意2年时效**：从法律文书规定履行期间的最后一日起计算
4. **财产线索一并提交**：有财产线索的，同时提交《财产线索清单》
5. **优先选择一审法院**：一般向一审法院或被执行财产所在地法院申请
