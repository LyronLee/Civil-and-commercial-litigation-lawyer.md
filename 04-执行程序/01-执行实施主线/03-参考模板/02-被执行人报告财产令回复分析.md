> 法院向被执行人发出《报告财产令》后，分析被执行人的财产申报是否真实、完整。

---

## 二、真实性核查

| 核查项 | 核查方式 | 核查结果 | 是否属实 |
|--------|---------|---------|---------|
| 银行存款 | 申请法院查询流水 | | |
| 不动产 | 不动产登记中心查询 | | |
| 车辆 | 车管所查询 | | |
| 股权 | 工商登记查询 | | |

---

## 三、完整性核查

- [ ] 是否存在已知的财产未申报？
- [ ] 是否存在财产转移迹象？
- [ ] 申报的财产是否已被查封/冻结？

---

## 四、分析结论

### 如实申报
- 财产足以清偿 → 推动执行
- 财产不足以清偿 → 考虑追加被执行人/参与分配

### 虚假申报/拒绝申报
- 申请法院对被执行人采取罚款、拘留措施
- 考虑移送拒执罪
