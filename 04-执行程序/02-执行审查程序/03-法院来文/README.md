# 03-法院来文

本文件夹用于存放执行审查程序中法院送达的各类文书。

## 应放入的材料

| 材料名称 | 说明 |
|---------|------|
| 异议受理通知书 | 法院受理异议申请的通知 |
| 听证通知书 | 法院组织异议听证的通知 |
| 异议裁定书 | 法院作出的异议审查裁定 |
| 复议受理通知书 | 上一级法院受理复议申请的通知 |
| 复议裁定书 | 上一级法院作出的复议裁定 |

## 使用方式

1. 收到法院文书后，及时存入本文件夹
2. 在文件名中标注收到日期
3. 唤起 AI 时，AI 会读取本文件夹内容并分析后续路径

## 注意事项

- 收到**异议裁定书**后，立即确认救济路径：
  - 执行行为异议裁定 → 不服可 **10 日内** 向上一级法院申请**复议**
  - 案外人异议裁定 → 不服可 **15 日内** 向执行法院提起**异议之诉**
  - 追加/变更异议裁定 → 不服可 **15 日内** 向执行法院提起**异议之诉**
- 复议/异议之诉期限均为**不变期间**，逾期丧失权利
- 异议审查期间，一般不停止执行（案外人提供充分有效担保的除外）
