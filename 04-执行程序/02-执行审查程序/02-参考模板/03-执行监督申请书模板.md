# 执行监督申请书

> 当事人、利害关系人认为执行法院的执行活动存在违法情形，可向人民检察院申请执行监督。

```
申请人：【姓名/名称】
住所地：

被监督机关：【执行法院名称】

申请事项：
请求【检察院名称】依法对【执行法院】在（【执行案号】）案件中的【具体执行行为】进行执行监督。

事实与理由：
【简述案件基本情况】

申请人认为，【执行法院】在执行过程中存在以下违法情形：

一、【违法情形1】
【具体说明违反的法律规定及造成的后果】

二、【违法情形2】
【具体说明】

申请人已【穷尽救济途径的说明：如已提出执行异议、复议、执行监督申请等及结果】。

综上所述，【执行法院】的执行活动存在违法情形，损害了申请人的合法权益，恳请贵院依法进行执行监督。

此致
【检察院名称】

申请人：【签章】
日期：

附：
1. 身份证明
2. 相关法律文书
3. 证据材料
```

---

## 重要提示

1. **监督机关**：人民检察院
2. **前置条件**：一般应先穷尽法院内部救济途径（异议、复议等）
3. **受理情形**：法院执行活动违法、执行人员职务犯罪等
4. **效力**：检察院可向执行法院提出检察建议，或提请上一级检察院抗诉
