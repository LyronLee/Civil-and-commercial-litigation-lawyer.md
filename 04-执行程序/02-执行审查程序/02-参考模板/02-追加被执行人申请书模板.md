# 追加/变更被执行人申请书（参考模板）

> ⚠️ 本模板仅供参考，需根据案件实际情况调整。生成后标注"初稿，需律师审核后使用"。
> ⚠️ 追加/变更被执行人的法定事由严格限定，必须有明确的法律依据，不能随意扩大。

---

## 文书信息

| 项目 | 内容 |
|------|------|
| 文书名称 | 追加/变更被执行人申请书 |
| 文书类型 | 申请书 |
| 法律依据 | 《最高人民法院关于民事执行中变更、追加当事人若干问题的规定》（2020年修正） |
| 适用场景 | 执行程序中，申请追加/变更符合法定条件的主体为被执行人 |

---

## 文书正文

```
追加/变更被执行人申请书

申请执行人：【姓名/名称】
住所地：
法定代表人/负责人：
联系电话：

被执行人：【姓名/名称】
住所地：

申请追加/变更被执行人：【姓名/名称】
住所地：
法定代表人/负责人：

申请事项：

请求贵院依法追加/变更【姓名/名称】为本案被执行人，对【原被执行人名称】在（【案号】）民事判决/调解书确定的债务承担【清偿/连带清偿/补充赔偿责任】。

事实与理由：

申请执行人与被执行人【案由】纠纷一案，业经【法院名称】审理，作出（【案号】）民事判决/调解书，判决/确认【判决主文内容】。

该判决/调解书已生效，申请执行人已依法向贵院申请强制执行（执行案号：【案号】）。执行过程中，【简述执行情况及发现的追加事由】。

经查，【申请追加/变更被执行人名称】存在以下情形：

【根据具体追加事由选择填写】

情形一：股东未缴纳或未足额缴纳出资
【申请追加/变更被执行人名称】为被执行人【原被执行人名称】的股东，认缴出资【金额】元，实缴出资【金额】元，未缴纳出资【金额】元。根据《最高人民法院关于民事执行中变更、追加当事人若干问题的规定》第十七条之规定，作为被执行人的企业法人，财产不足以清偿生效法律文书确定的债务，申请执行人申请变更、追加未缴纳或未足额缴纳出资的股东、出资人或依公司法规定对该出资承担连带责任的发起人为被执行人，在尚未缴纳出资的范围内依法承担责任的，人民法院应予支持。

情形二：股东抽逃出资
【申请追加/变更被执行人名称】为被执行人【原被执行人名称】的股东，存在抽逃出资【金额】元的行为【简述抽逃方式，如：将出资款转入公司账户验资后又转出】。根据《最高人民法院关于民事执行中变更、追加当事人若干问题的规定》第十八条之规定……

情形三：一人公司财产混同
【申请追加/变更被执行人名称】为被执行人【原被执行人名称】的唯一股东，被执行人系一人有限责任公司。根据《最高人民法院关于民事执行中变更、追加当事人若干问题的规定》第二十条之规定，作为被执行人的一人有限责任公司，财产不足以清偿生效法律文书确定的债务，股东不能证明公司财产独立于自己的财产，申请执行人申请变更、追加该股东为被执行人，对公司债务承担连带责任的，人民法院应予支持。

情形四：公司未经清算即注销
被执行人【原被执行人名称】未经依法清算即办理注销登记，【申请追加/变更被执行人名称】作为被执行人的股东/出资人/主管部门，在办理注销登记时书面承诺对被执行人的债务承担清偿责任。根据《最高人民法院关于民事执行中变更、追加当事人若干问题的规定》第二十一条、第二十三条之规定……

情形五：自愿提供执行担保
【申请追加/变更被执行人名称】于【日期】向贵院出具《执行担保书》，自愿为被执行人【原被执行人名称】提供担保，承诺如被执行人在【日期】前不履行义务，由【申请追加/变更被执行人名称】承担清偿责任。现被执行人未按期履行，根据《最高人民法院关于民事执行中变更、追加当事人若干问题的规定》第二十四条之规定……

情形六：合伙企业合伙人（无限连带责任）
被执行人【原被执行人名称】为合伙企业，【申请追加/变更被执行人名称】为被执行人的普通合伙人。根据《最高人民法院关于民事执行中变更、追加当事人若干问题的规定》第十四条第一款之规定……

情形七：分支机构
被执行人【原被执行人名称】为【申请追加/变更被执行人名称】的分支机构，不能清偿生效法律文书确定的债务。根据《最高人民法院关于民事执行中变更、追加当事人若干问题的规定》第十五条第一款之规定……

情形八：无偿接受财产
被执行人【原被执行人名称】的股东/出资人/主管部门【申请追加/变更被执行人名称】无偿接受被执行人财产【金额/财产描述】，致使被执行人无遗留财产或遗留财产不足以清偿债务。根据《最高人民法院关于民事执行中变更、追加当事人若干问题的规定》第二十二条之规定……

【选择一种或多种情形填写，删除未适用的情形】

综上所述，【申请追加/变更被执行人名称】符合追加/变更被执行人的法定条件，恳请贵院依法追加/变更其为被执行人。

此致

【执行法院名称】

                                    申请执行人：【签章】

                                    日期：

附：
1. 申请执行人身份证明/营业执照复印件
2. 生效法律文书复印件
3. 执行案件受理通知书/执行裁定书复印件
4. 证明追加事由的证据材料（公司章程、验资报告、工商档案、担保书等）
```

---

## 重要提示

1. **举证责任**：申请执行人需证明追加事由成立（如股东未出资、抽逃出资等）
2. **一人公司特殊规则**：一人公司财产混同的举证责任**倒置**——由股东证明公司财产独立于个人财产
3. **被追加人的救济**：被追加人不服的，可在收到裁定之日起 **15 日内** 提起**追加/变更被执行人异议之诉**
4. **不得重复追加**：已被裁定追加/变更的，不得就同一事由再次申请
5. **审查期限**：法院应在收到申请后 **60 日内** 作出裁定（复杂案件可延长）
