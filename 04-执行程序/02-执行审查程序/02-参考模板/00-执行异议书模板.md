# 执行异议书（行为异议）

> 依据《民事诉讼法》第236条，当事人、利害关系人认为执行行为违反法律规定的，可向执行法院提出书面异议。

```
异议人：【姓名/名称】
住所地：

申请执行人：【姓名/名称】

被执行人：【姓名/名称】

异议事项：
请求【执行法院名称】依法【具体请求，如：中止对XX财产的执行/撤销XX执行行为】。

事实与理由：
【简述执行案件基本情况】

贵院在执行过程中【具体执行行为描述】，该行为存在以下违法之处：

一、【违法点1】
【具体说明违反的法律规定】

二、【违法点2】
【具体说明违反的法律规定】

综上所述，贵院的上述执行行为违反了《民事诉讼法》及相关司法解释的规定，恳请贵院依法支持异议人的请求。

此致
【执行法院名称】

异议人：【签章】
日期：

附：
1. 本异议书副本【】份
2. 相关证据材料
```

---

## 撰写要点

1. **期限**：执行程序终结前提出（法律未规定具体期限，但应在知道或应当知道执行行为之日起合理期限内提出，建议尽早）
2. **针对性**：必须针对具体的执行行为，不能是对执行依据（判决本身）的异议
3. **法律引用**：明确引用违反的具体法律条文
4. **复议权**：对裁定不服的，可自裁定送达之日起10日内向上一级法院申请复议
