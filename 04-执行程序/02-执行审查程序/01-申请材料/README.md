# 01-申请材料

本文件夹用于存放与执行异议相关的证据和申请材料。

## 应放入的材料

| 材料类型 | 说明 | 必要性 |
|---------|------|--------|
| 执行依据 | 生效判决书/调解书等 | 必须 |
| 执行法院文书 | 执行通知书、裁定书等 | 必须 |
| 异议相关证据 | 支持异议事由的证据材料 | 必须 |
| 身份证明 | 异议人身份证/营业执照 | 必须 |

### 按异议类型补充材料

| 异议类型 | 补充材料 |
|---------|---------|
| 执行行为异议 | 证明执行行为违法的证据（如超标的查封评估报告） |
| 案外人异议 | 证明对执行标的享有实体权利的证据（如买卖合同、付款凭证、占有证明） |
| 追加/变更异议 | 证明追加事由不成立的证据（如出资凭证、财务独立证明） |

## 使用方式

1. 将与异议相关的全部材料放入本文件夹
2. 唤起 AI 时，AI 会读取本文件夹内容并协助起草异议申请书

## 注意事项

- 异议申请应**针对具体执行行为或标的**，避免笼统表述
- 案外人异议需证明权利取得时间**早于查封时间**
- 执行行为异议应在**执行程序终结前**提出
- 案外人异议应在**执行标的执行终结前**提出
