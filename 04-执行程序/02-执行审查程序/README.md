# 02-执行审查程序：异议与救济

> 🎯 **阶段定位**：执行实施过程中，当事人或案外人对执行行为、执行标的、追加被执行人有异议时，通过审查程序寻求救济。
>
> 执行审查程序是执行实施主线的"纠偏机制"，确保执行行为合法、执行标的不侵害案外人权益。

## 🚀 五种审查程序的使用指南

### 程序一：执行行为异议

**适用情形**：认为执行行为违反法律规定（如超标的查封、错误扣划、程序违法等）

**提出主体**：当事人、利害关系人

**程序流程**：
```
提出异议 → 执行法院15日内裁定 → 不服裁定 → 10日内向上一级法院申请复议
```

**期限**：在执行程序终结前提出

---

### 程序二：案外人异议

**适用情形**：案外人对执行标的主张所有权或其他足以排除执行的实体权利

**提出主体**：案外人

**程序流程**：
```
提出异议 → 执行法院15日内审查 → 
  ├─ 裁定中止执行 → 申请执行人可提起异议之诉
  └─ 裁定驳回 → 案外人可提起异议之诉
```

**期限**：在执行标的执行终结前提出

> ⚠️ **关键区分**：案外人异议（审查程序） vs. 案外人执行异议之诉（诉讼程序）。异议是前置程序，对异议裁定不服才能提起异议之诉。

---

### 程序三：追加/变更被执行人异议

**适用情形**：不服法院追加/变更被执行人的裁定

**提出主体**：被追加/变更的被执行人或申请执行人

**程序流程**：
```
法院作出追加/变更裁定 → 被追加人不服 → 15日内向执行法院提起异议之诉
```

---

### 程序四：执行复议

**适用情形**：对执行行为异议裁定不服

**提出主体**：执行行为异议的申请人

**程序流程**：
```
收到异议裁定 → 10日内向上一级法院申请复议 → 上一级法院30日内作出复议裁定
```

**期限**：收到裁定之日起 **10日**

> ⏰ **注意**：案外人异议的裁定不能复议，只能提起异议之诉。

---

### 程序五：执行监督

**适用情形**：执行程序存在违法情形，且通过异议、复议无法救济

**提出主体**：当事人、利害关系人

**受理机关**：上一级法院或同级/上级人民检察院

**适用情形**：
- 执行行为违法且异议、复议被驳回
- 执行人员存在违法行为
- 长期拖延执行
- 其他需要监督的情形

---

## 五种程序的对比速查

| 程序 | 提出主体 | 针对对象 | 期限 | 救济路径 |
|------|---------|---------|------|---------|
| 执行行为异议 | 当事人、利害关系人 | 执行行为违法 | 执行终结前 | 不服 → 10日复议 |
| 案外人异议 | 案外人 | 执行标的权属 | 执行终结前 | 不服 → 15日异议之诉 |
| 追加/变更异议 | 被追加人 | 追加/变更裁定 | 裁定后 | 不服 → 15日异议之诉 |
| 执行复议 | 异议申请人 | 异议裁定 | 收到裁定10日 | 复议裁定为终局 |
| 执行监督 | 当事人、利害关系人 | 执行程序违法 | 无明确期限 | 法院/检察院监督 |

## ❓ 常见问题

**Q1: 执行行为异议和案外人异议有什么区别？**
> - **执行行为异议**：认为执行"行为"违法（如超标的查封、错误扣划），不涉及实体权利
> - **案外人异议**：对执行"标的"主张实体权利（如"这房子是我的，不是被执行人的"）
> - 区分关键：异议的是"行为"还是"标的"

**Q2: 对异议裁定不服，什么时候复议、什么时候起诉？**
> - **执行行为异议**裁定 → **复议**（10日内向上一级法院）
> - **案外人异议**裁定 → **异议之诉**（15日内向执行法院）
> - **追加/变更被执行人异议**裁定 → **异议之诉**（15日内向执行法院）

**Q3: 执行复议和异议之诉有什么区别？**
> - **执行复议**：审查执行行为是否合法，程序性救济
> - **执行异议之诉**：审理实体权利争议，一审诉讼程序
> - 复议不服只能接受；异议之诉不服还可上诉
