# 阶段产出

本文件夹用于存放执行救助阶段 AI 生成的工作成果文件。

## 产出文件清单

| 序号 | 文件名 | 说明 |
|------|--------|------|
| 01 | 执行救助条件分析报告.md | 分析是否符合执行救助条件 |
| 02 | 执行救助申请书.md | 提交法院的申请书 |
| 03 | 材料清单.md | 需准备的支持材料清单 |
| 04 | 跟进记录.md | 救助申请进度跟踪记录 |

## 使用方式

AI 根据 `SKILL.md` 工作流生成的文件自动保存于此。用户可直接使用或按需修改。

## 注意事项

- 所有文件生成后标注"初稿，需律师审核后使用"
- 救助金额和标准因地而异，以受理法院所在地政策为准
