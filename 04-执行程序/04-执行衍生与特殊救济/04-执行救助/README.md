# 04-执行救助

被执行人确无财产可供执行，申请执行人因权利无法实现而生活困难，向法院申请执行救助金时在此处理。

## 适用情形

- 执行案件已终本（终结本次执行程序）
- 被执行人确无财产可供执行
- 申请执行人生活困难
- 符合当地执行救助政策

直接阅读并执行 `SKILL.md` 文件。

## 说明

- 执行救助是**国家司法救助**，不是执行程序的延续
- 救助金额和标准因地而异，以受理法院所在地政策为准
- 获得救助不影响继续追偿被执行人的权利
