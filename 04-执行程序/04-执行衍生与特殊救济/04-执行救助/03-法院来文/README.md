# 法院来文

本文件夹用于存放执行救助申请过程中法院发出的各类文书。

## 应放入的材料

| 材料名称 | 说明 |
|---------|------|
| 受理通知书 | 法院受理执行救助申请的通知 |
| 救助决定 | 法院司法救助委员会作出的救助决定 |
| 救助金发放凭证 | 救助金发放的相关凭证 |
| 补充材料通知 | 法院要求补充材料的通知 |

## 使用方式

1. 收到法院文书后，及时存入本文件夹
2. AI 在跟进救助申请进度时会读取本文件夹内容

## 注意事项

- 执行救助由法院司法救助委员会审批，周期因地而异
- 如救助申请被驳回，可了解驳回理由并评估是否补充材料重新申请
- 获得救助后，如发现被执行人有财产，仍应申请恢复执行
