# 指定执行申请书

```
申请人：【姓名/名称】，住所地：【地址】

申请事项：
请求【上级人民法院】依法指定【被指定法院名称】执行【原执行法院】（【案号】）执行案件。

事实与理由：
申请人与被执行人【案由】纠纷一案，业经【原审法院】审理，作出（【案号】）民事判决/调解书，已发生法律效力。

该案由【原执行法院】立案执行（执行案号：【案号】），但目前存在以下执行冲突情形：

【具体描述执行冲突情形，如：
1. 同一财产被多个法院查封，【原执行法院】与【另一法院】对【财产描述】的查封存在冲突；
2. 【原执行法院】的执行行为影响【另一法院】的执行；
3. 其他需要上级法院协调或指定执行的情形。】

上述情形导致执行程序难以顺利推进，为统一执行、避免冲突，依据《中华人民共和国民事诉讼法》及相关执行协调规定，特申请由贵院指定【被指定法院名称】执行该案。

此致
【上级人民法院名称】

申请人：【签章】
日期：【年月日】
```

## 使用说明

1. 本模板适用于两个以上法院执行冲突时，向上级法院申请指定某一法院执行的情形。
2. 应附以下材料：
   - 各执行案件的执行依据
   - 冲突财产的相关证明
   - 各法院查封/冻结裁定（如有）
3. 指定执行申请一般由当事人向执行法院提出，由执行法院报请上级法院决定。
