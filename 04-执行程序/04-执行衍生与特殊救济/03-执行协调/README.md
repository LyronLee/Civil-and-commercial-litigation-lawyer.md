# 03-执行协调

两个以上法院对同一被执行人或关联财产的执行发生冲突，需要报请上级法院协调时在此处理。

## 适用情形

- 同一财产被多个法院查封
- 执行标的涉及其他法院审理的案件
- 执行行为影响其他法院执行
- 管辖争议

## 文件结构

```
03-执行协调/
├── README.md          # 本文件

```

## 工作流入口

直接阅读并执行 `SKILL.md` 文件。

## 说明

执行协调是法院内部程序，一般由执行法院依职权或依当事人申请报请上级法院协调。
