# 法院来文

本文件夹用于存放执行协调过程中各级法院发出的各类文书。

## 应放入的材料

| 材料名称 | 说明 |
|---------|------|
| 协调通知 | 上级法院发出的协调会议通知 |
| 指定执行裁定 | 上级法院指定某一法院执行的裁定 |
| 合并执行裁定 | 上级法院裁定合并执行的文书 |
| 协调纪要 | 协调会议形成的纪要 |
| 各法院回复 | 相关法院对协调的回复意见 |

## 使用方式

1. 收到法院文书后，及时存入本文件夹
2. AI 在跟进协调进展时会读取本文件夹内容

## 注意事项

- 执行协调不设立案程序，由法院内部依职权或依申请启动
- 指定执行后，原执行法院应将执行材料移送给被指定法院
