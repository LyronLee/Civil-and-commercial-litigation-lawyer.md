# 01-执行回转

执行完毕后，据以执行的法律文书被撤销或变更，需要执行回转时在此处理。

## 适用情形

- 原执行程序已完毕
- 执行依据（判决书/裁定书/调解书）被撤销、变更或裁定不予执行
- 原被执行人申请返还已执行的财产

## 文件结构

```
01-执行回转/
├── README.md          # 本文件
├── SKILL.md           # 执行回转工作流
├── 01-参考模板/       # 执行回转申请书模板等
├── 02-申请材料/       # 原执行依据、撤销/变更文书等
├── 03-法院来文/       # 法院文书
└── 04-阶段产出/       # AI生成的工作成果
```

## 工作流入口

直接阅读并执行 `SKILL.md` 文件。

## 法律依据

《中华人民共和国民事诉讼法》（2023年修正）第244条。
