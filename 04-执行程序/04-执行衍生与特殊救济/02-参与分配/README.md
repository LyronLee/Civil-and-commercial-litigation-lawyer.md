# 02-参与分配

被执行人为公民或其他组织，其财产不足以清偿全部债务，已取得执行依据的债权人申请参与分配时在此处理。

## 适用情形

- 被执行人为公民或其他组织（非企业法人）
- 财产不足以清偿全部债务
- 已取得生效法律文书
- 多个债权人对同一被执行人执行
