# 法院来文

本文件夹用于存放参与分配过程中法院发出的各类文书。

## 应放入的材料

| 材料名称 | 说明 |
|---------|------|
| 受理通知书 | 法院受理参与分配申请的通知 |
| 分配方案 | 法院制作的财产分配方案 |
| 执行裁定 | 财产处置、分配裁定 |
| 异议通知 | 法院将异议通知其他债权人的文书 |
| 反对意见 | 其他债权人对异议的反对意见（如有） |

## 使用方式

1. 收到法院文书后，及时存入本文件夹
2. 收到分配方案后，及时评估是否提出异议
3. 如有人反对异议，15日内可提起分配方案异议之诉

## 注意事项

- 收到分配方案后应仔细核对债权数额和分配比例
- 如对分配方案有异议，应在法定期间内提出
- 分配方案异议之诉一审终审，判决不可上诉
