# 阶段产出

本文件夹用于存放参与分配阶段 AI 生成的工作成果文件。

## 产出文件清单

| 序号 | 文件名 | 说明 |
|------|--------|------|
| 01 | 参与分配条件分析报告.md | 分析是否符合参与分配条件 |
| 02 | 参与分配申请书.md | 提交法院的申请书 |
| 03 | 分配方案分析报告.md | 对法院分配方案的分析意见 |
| 04 | 跟进记录.md | 参与分配进度跟踪记录 |

## 使用方式

AI 根据 `SKILL.md` 工作流生成的文件自动保存于此。用户可直接使用或按需修改。

## 注意事项

- 所有文件生成后标注"初稿，需律师审核后使用"
- 如对分配方案有异议，15日内可提起分配方案异议之诉
