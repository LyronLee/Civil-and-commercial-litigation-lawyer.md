# 04 执行程序：生效法律文书的实现

> 核心理念：执行是诉讼的"最后一公里"。本程序涵盖申请执行→财产查控→处置变现→执行完毕/终本的完整主线，以及执行审查、执行异议之诉、执行衍生救济三大支线。

## 前置检查

| 检查项 | 合格标准 | 不满足时的替代路径 |
|--------|---------|-------------------|
| 是否有生效法律文书 | 判决书/调解书/仲裁裁决等已生效 | 未生效 → 等待生效或先上诉 |
| 履行期限是否届满 | 已超过判决确定的履行期间 | 未届满 → 提示等待履行期满 |
| 是否在申请执行期限内 | 2年内（从法律文书规定履行期间的最后一日起算） | 超期 → 告知可能丧失执行权 |
| 执行标的是否明确 | 给付内容具体、可执行 | 不明确 → 可能需要补正判决 |

> ⚠️ **申请执行期限**：2年，适用中止、中断规则（与再审的除斥期间不同）。

## 阶段路由

```
条件确认完成
    │
    ├─ 未生效法律文书 ────────────────→ 等待生效或 02-二审程序/
    │
    ├─ 履行期未届满 ──────────────────→ 提示等待
    │
    ├─ 超2年申请执行期限 ─────────────→ 告知可能丧失执行权
    │
    ├─ 需申请执行 ────────────────────→ 01-执行实施主线/
    │    （申请执行、财产查控、信用惩戒、悬赏执行、执行和解）
    │
    ├─ 对执行行为有异议 ──────────────→ 02-执行审查程序/ 分支A
    ├─ 案外人对标的有异议 ────────────→ 02-执行审查程序/ 分支B
    ├─ 不服追加/变更被执行人 ─────────→ 02-执行审查程序/ 分支C
    ├─ 对异议裁定不服（行为异议）─────→ 02-执行审查程序/ 分支D（复议）
    ├─ 对异议裁定不服（案外人异议）───→ 03-执行异议之诉/
    │
    ├─ 执行依据被撤销 ────────────────→ 04-执行衍生与特殊救济/ 分支A（回转）
    ├─ 多个债权人执行同一被执行人 ────→ 04-执行衍生与特殊救济/ 分支B（参与分配）
    ├─ 不同法院执行冲突 ──────────────→ 04-执行衍生与特殊救济/ 分支C（协调）
    └─ 执行不能且生活困难 ────────────→ 04-执行衍生与特殊救济/ 分支D（救助）
```

| 02-执行审查程序 | 执行行为异议、案外人异议、追加异议、复议、执行监督 | `02-执行审查程序/SKILL.md` |

| 执行立案 | 7 日 | 法院收到申请后 | 法定期间 | |
| 执行通知 | 10 日 | 立案后 | 法定期间 | |

## 通用原则

1. **执行依据必须生效且明确**：执行的前提是生效法律文书，内容必须具体可执行
2. **期限不可错过**：申请执行 2 年、异议之诉起诉 15 日、复议申请 10 日，逾期丧失权利
3. **程序转换衔接**：执行异议之诉 → 进入 `03-执行异议之诉/`，按独立工作流执行；执行回转 → 进入 `04-执行衍生与特殊救济/01-执行回转/`，按独立工作流执行
4. **AI 边界声明**：所有生成文书标注"初稿，需律师审核后使用"
5. **严禁编造**：引用法条、案例必须准确
