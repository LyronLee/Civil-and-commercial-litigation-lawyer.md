# 对方来文

本文件夹用于存放对方当事人提交的诉讼文书和证据材料。

## 应放入的材料

| 材料类型 | 说明 | 触发作用 |
|---------|------|---------|
| 起诉状（原告提交） | 原告向法院提交的起诉文书 | 触发被告模式，生成答辩状 |
| 答辩状（被告提交） | 被告对原告起诉的答辩 | 了解对方抗辩方向 |
| 证据目录及证据材料 | 对方提交的证据清单和证据册 | 阶段三质证意见的直接输入 |
| 补充证据 | 庭审前或庭审中对方补充的材料 | 更新质证意见 |
| 反诉状 | 被告提起的反诉文书 | 触发反诉应对流程 |
| 管辖权异议申请书 | 对方对管辖提出的异议 | 评估管辖争议 |

## 使用方式

1. 收到对方材料后，将原件/复印件扫描件放入本文件夹
2. 在阶段二中，如检测到起诉状，AI 自动切换为被告模式，生成答辩状
3. 在阶段三中，AI 读取本文件夹全部材料，撰写质证意见

## 注意事项

- 来文管理贯穿阶段二至四，形成完整的案件时间线
- 收到对方新材料后，应及时更新本文件夹，并通知 AI 重新生成质证意见
- 对方当庭提交的新证据，在庭审纪要中记录后，也应补充到本文件夹
