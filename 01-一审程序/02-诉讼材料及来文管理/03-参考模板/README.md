# 参考模板

本文件夹存放一审程序阶段二（诉讼材料及来文管理）的参考模板，供 AI 生成阶段产出时参照使用。

## 模板清单

| 序号 | 模板名称 | 用途 |
|------|---------|------|
| 00-起诉状模板.md | 原告起诉文书格式 | 首部、案由、诉讼请求、事实与理由、尾部 |
| 01-答辩状模板.md | 被告答辩文书格式 | 首部、答辩请求、事实答辩、法律答辩、程序性抗辩 |
| 02-证据目录模板.md | 证据编目表格 | 4列格式（序号/名称/证明目的/页码），按争议焦点分组 |
| 03-证据说明（自用版）模板.md | 内部工作文件格式 | 证据形式、来源、内容摘要、证明力评估、预判质证 |
| 证据说明（提交法院版）模板.md | 正式提交版本格式 | 段落式排版，客观陈述证据事实，便于书记员复制 |
| 04-财产保全申请书模板.md | 申请财产保全 | 保全标的、担保方式、紧急情况说明 |
| 05-调查取证申请书模板.md | 申请法院调查取证 | 需调取的证据名称、持有方、无法自行收集的理由 |

## 特殊案由适配

如案件属于特殊案由（代位权/撤销权/人格否认/股东出资/执行异议之诉），AI 会自动调用 `特殊案由适配/` 子文件夹中的起诉状差异标注，调整当事人地位、管辖、诉讼请求等关键参数。

## 使用方式

1. 本文件夹的模板为**参照模板**，AI 在执行 `SKILL.md` 工作流时会自动调用
2. 用户无需手动填写模板，将阶段一产出放入 `01-上一阶段产出/` 后，唤起 AI 即可
3. 如需了解模板格式，可直接打开查看

## 角色分流

| 角色 | 主要调用模板 |
|------|-------------|
| 原告（Builder） | 起诉状模板、证据目录模板、证据说明模板、诉讼费用计算表 |
| 被告（Breaker） | 答辩状模板、证据目录模板、证据说明模板 |

## 产出位置

AI 根据本阶段 SKILL.md 生成的正式文件保存在 `06-阶段产出/` 中。
