## 分组建议（根据案由调整）

| 常见案由 | 建议分组 |
|---------|---------|
| 合同纠纷 | 合同订立证据 / 合同履行证据 / 违约事实证据 / 损失金额证据 |
| 侵权纠纷 | 权属证据 / 侵权事实证据 / 损害后果证据 / 因果关系证据 |
| 借贷纠纷 | 借贷合意证据 / 款项交付证据 / 还款情况证据 / 催告证据 |
| 劳动争议 | 劳动关系证据 / 工资标准证据 / 解除事实证据 / 损失证据 |

---

> 💡 **填写提示**：
> - "证据名称"填写材料的全称，如"《商品房买卖合同》（张三）"
> - "证明目的"用一句话概括该证据要证明什么，如"证明地下停车场未计入住宅公摊面积"
> - "页码"填写该证据在证据册中的起止页码
> - 同一证据可列入不同组别（在页码处注明"详见证据X-Y"）
