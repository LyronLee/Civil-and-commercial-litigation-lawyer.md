# 02 诉讼材料及来文管理：从策略到文书的转化

> 🎯 **阶段定位**：阶段一已经理清了案情、定好了策略。这个阶段的核心任务是把策略转化为可直接提交法院的**正式法律文书**，同时建立**来文管理机制**。
>
> 一句话概括：**阶段一想清楚，阶段二写清楚**。

## 🚀 三步使用流程

### 第一步：复制上一阶段产出 → `01-上一阶段产出/`

把阶段一（`01-战略把脉/03-阶段产出/`）中的全部文件，**复制**到本阶段的 `01-上一阶段产出/` 文件夹中。

> ⚠️ **是复制，不是剪切**。阶段一的原文件保持不动，这里放的是副本。

### 第二步：填写问卷 → `02-客户情况说明/`

打开 `02-客户情况说明/诉讼意图情况说明.md`，根据你的实际情况填写：

| 必填项 | 为什么重要 |
|--------|-----------|
| 你的诉讼角色（原告/被告） | 决定AI生成起诉状还是答辩状 |
| 是否申请财产保全 | 决定AI是否生成保全申请书 |
| 是否需要法院调查取证 | 决定AI是否生成调查取证申请书 |
| 是否接受调解 | 影响起诉状/答辩状的措辞 |
| 诉讼请求标的额 | 影响诉讼费用计算 |

> 💡 **技巧**：即使你不确定某些选项，也可以先勾选"待定"，AI会在生成过程中询问你。

### 第三步：唤AI → "帮我写诉讼文书"

打开你的AI助手，告诉ta：

> **"我在做02诉讼材料及来文管理阶段，请帮我起草文书。上一阶段产出在01-上一阶段产出文件夹里，客户情况说明已填好。"**

AI会按以下步骤帮你：

1. **确认角色和意图** — 读取你的情况说明，确认是原告还是被告
2. **起草起诉状/答辩状** — 基于阶段一的策略方案书和材料索引
3. **制作证据目录** — 按争议焦点分组，4列格式
4. **撰写证据说明** — 为每份证据准备庭审说明
5. **生成辅助文书** — 诉讼费用计算表、保全申请书等（按需）
6. **列出待补充清单** — 阶段二视角下还缺什么信息

> ⚠️ **重要提醒**：AI生成的是**文书初稿**，必须经过律师审核修改后才能提交法院。

## 📬 来文管理指南

### 收到对方文书后

把对方的以下材料放入 `04-对方来文/`：
- 起诉状/答辩状/反诉状
- 对方提交的证据材料
- 律师函
- 其他书面意见

然后可以唤起AI：
> **"对方提交了新材料在04-对方来文里，帮我写质证意见。"**

AI会：
1. 读取对方材料
2. 对比你方的起诉状/答辩状和证据目录
3. 逐份出具三性质证意见（真实性、合法性、关联性）
4. 生成质证意见文件

### 收到法院文书后

把法院的以下材料放入 `05-法院来文/`：
- 传票（注意开庭时间！）
- 举证通知书（注意举证期限！）
- 应诉通知书
- 裁定书
- 判决书

> 💡 **提示**：建议按收到日期+文书类型命名文件，如"20260420-传票.pdf"，方便后续查找。

> ⏰ **期限提醒**：AI会自动提取法院来文中的关键期限（答辩期、举证期限、上诉期等），生成《关键期限跟踪表.md》。**法院指定的期限优先于法定期限**。

## ❓ 常见问题

**Q1: 我是原告，阶段二要准备哪些文书？**
> 核心必做：起诉状 + 证据目录 + 证据说明 + 诉讼费用计算表  
> 按需生成：财产保全申请书、调查取证申请书  
> 来文管理：收到被告答辩状后放入 `04-对方来文/`，可生成质证意见

**Q2: 我是被告，阶段二要准备哪些文书？**
> 核心必做：答辩状 + 证据目录 + 证据说明  
> 额外需要：把原告的起诉状放入 `04-对方来文/`，AI据此起草答辩状  
> 按需生成：管辖异议申请书（如需）、反诉状（如需）、调查取证申请书  
> 来文管理：持续更新 `04-对方来文/` 和 `05-法院来文/`

**Q3: 证据说明和证据目录有什么区别？**
> - **证据目录**：提交给法院的正式文件，4列表格（序号、证据名称、证明目的、页码）
> - **证据说明（自用版）**：律师庭审准备的自用工作文件，详细分析每份证据的内容、关联性、证明力、可能的质证点。**不提交法院**
> - **证据说明（提交法院版）**：可提交法院的正式版本，根据各份证据内容客观陈述事实和证据所在页码，不含内部评估

**Q4: 为什么阶段二不直接生成质证意见？**
> 没有对方实际材料时，杜撰质证意见没有实战价值。当你收到对方材料后放入 `04-对方来文/`，AI会作为独立skill帮你逐份审阅并生成针对性的质证意见。

**Q5: 诉讼费用计算表有什么用？**
> 起诉时需要向法院交纳案件受理费（败诉方最终承担）。计算表帮你：
> - 精确计算应交纳的诉讼费金额
> - 预判保全费、公告费等其他费用
> - 为客户做费用预算

### 调解/和解成功

如在阶段二（诉讼材料制备期间）达成和解：

**路径一：诉前和解（尚未立案）**
- 签订和解协议 → 直接进入阶段四的结案复盘
- 如担心对方反悔，可共同申请法院出具调解书（需先立案）

**路径二：立案后调解**
- 向法院申请调解
- 法院出具调解书（具有强制执行力）
- 阶段二提前结束，直接进入阶段四的结案复盘

> ⚠️ **和解协议 vs. 调解书**：和解协议不具有强制执行力，调解书具有强制执行力。如选择和解协议，建议附加违约条款或担保条款。

---

**Q6: AI生成的起诉状能直接提交法院吗？**
> **不能**。AI生成的是初稿，你必须：
> 1. 检查是否遗漏诉讼请求
> 2. 核对事实陈述是否与证据一致
> 3. 按受理法院的具体格式要求调整（不同法院可能有特殊要素要求）
> 4. 经律师审核签字盖章后提交

---

## ⚠️ 重要提示

1. **文书审核**：AI生成的所有文书均为初稿，提交法院前必须经律师审核。
2. **法院格式**：不同法院对文书格式可能有不同要求（如某些法院要求特殊要素），请按受理法院要求调整。
3. **举证期限**：收到法院举证通知书后，注意举证期限，逾期提交的证据可能不被采纳。
4. **来文及时归档**：收到对方或法院的任何文书，第一时间放入对应文件夹，避免遗漏。
5. **随时可回退**：如发现新证据或策略调整，可回到阶段一补充，再重新生成阶段二文书。
