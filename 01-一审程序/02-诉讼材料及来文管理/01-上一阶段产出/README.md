# 上一阶段产出

本文件夹用于存放阶段一（战略把脉）的全部产出文件，作为阶段二（诉讼材料及来文管理）的核心输入。

## 应放入的文件

将 `01-一审程序/01-战略把脉/03-阶段产出/` 中的全部文件复制到本文件夹：

| 来源文件 | 在本阶段的作用 |
|---------|--------------|
| 01-材料索引与分类.md | 筛选⭐级证据，制作证据目录 |
| 02-案件事实时间轴.md | 撰写起诉状/答辩状的事实依据 |
| 03-类案检索报告.md | 法律论证和类案支撑 |
| 04-法条汇编.md | 起诉状/答辩状的法律依据 |
| 05-诉讼策略方案书.md | 确定角色、诉讼请求/抗辩方向、争议焦点 |
| 06-待补充材料清单.md | 阶段二继续跟踪补充进度 |
| 00-定制化材料清单.md | 如为特殊案由，确认案由适配规则 |

## 使用方式

1. 复制阶段一全部产出到本文件夹（不要移动，阶段三仍需引用阶段二卷宗）
2. 确保 `05-诉讼策略方案书.md` 完整可读，AI 将从中提取代理方向
3. 如策略方案书中标注了特殊案由，AI 会自动调用对应适配模板

## 注意事项

- 本文件夹是阶段二的**唯一输入源**，不跨目录读取其他阶段文件
- 如缺少核心文件（如策略方案书），AI 会提示用户补充
- 阶段二生成的起诉状/答辩状直接引用本文件夹中的材料索引和时间轴
