# 客户情况说明

本文件夹用于存放客户填写的诉讼意图情况说明，是阶段二（诉讼材料及来文管理）的重要输入之一。

## 核心文件

| 文件名 | 用途 |
|--------|------|
| 诉讼意图情况说明.md | 客户填写的诉讼意向问卷 | 帮助 AI 确认角色、诉讼请求、保全意向、调解意向等 |

## 填写指引

`诉讼意图情况说明.md` 包含以下关键信息：

| 信息项 | 说明 |
|--------|------|
| 本方角色 | 原告 / 被告 / 第三人 |
| 是否接受调解 | 是 / 否 / 视条件而定 |
| 是否申请财产保全 | 是 / 否 |
| 是否需要法院调查取证 | 是 / 否 |
| 是否提起反诉（被告） | 是 / 否 |
| 核心诉求/抗辩要点 | 客户最关注的结果 |
| 可接受的底线条件 | 如接受调解，最低可接受的条件 |

## 使用方式

1. 让客户在阶段二开始前填写 `诉讼意图情况说明.md`
2. AI 读取该文件，结合阶段一产出，确认角色和诉讼方向
3. 如关键项空白，AI 会向用户询问必要信息

## 注意事项

- 客户情况说明是**辅助输入**，如与策略方案书冲突，以用户最新确认为准
- 如客户未填写，AI 仍可基于策略方案书完成阶段二，但可能需要额外确认
- 涉及隐私的信息（身份证号、银行账户等）建议脱敏处理
