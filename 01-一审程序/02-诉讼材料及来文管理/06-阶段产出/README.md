# 阶段产出

本文件夹存放一审程序阶段二（诉讼材料及来文管理）的 AI 生成工作成果。

## 预期产出文件

### 核心文书

| 序号 | 文件名 | 内容说明 |
|------|--------|---------|
| 01-起诉状.md / 01-答辩状.md | 正式诉讼文书 | 原告模式→起诉状；被告模式→答辩状 |
| 02-证据目录.md | 证据编目表格 | 4列格式（序号/名称/证明目的/页码），按争议焦点分组 |
| 03-证据说明（自用版）.md | 内部工作文件 | 含证据形式、来源、内容摘要、证明力评估、预判质证 |
| 03-证据说明（提交法院版）.md | 正式提交版本 | 段落式排版，客观陈述证据事实，便于书记员复制 |

### 辅助文书（按需生成）

| 序号 | 文件名 | 触发条件 |
|------|--------|---------|
| 04-诉讼费用计算表.md | 原告模式必填 | 按标的额分段计算案件受理费、保全费等 |
| 05-财产保全申请书.md | 客户勾选"申请保全" | 申请法院查封、冻结对方财产 |
| 06-调查取证申请书.md | 客户勾选"需调查取证" | 申请法院调取特定证据 |
| 07-待补充材料清单.md | 阶段二视角 | 起诉状/答辩状中发现的信息缺口 |
| 08-关键期限跟踪表.md | 收到法院来文后 | 答辩期、举证期、开庭日、上诉期等 |

## 使用方式

1. 本文件夹文件由 AI 根据 `SKILL.md` 工作流自动生成
2. 生成后需由**执业律师审核**，确认诉讼请求/抗辩意见准确、证据目录完整、格式规范
3. 审核通过后，打印/盖章/提交法院

## 注意事项

- 所有产出文件均标注"初稿，需律师审核后使用"
- 证据说明（自用版）**不提交法院**，仅用于律师庭审准备
- 证据说明（提交法院版）采用段落式排版，关键处用 `>` 引用格式强调
- 内部材料（案件背景说明、律师工作笔记等）**不纳入证据目录**
