# 法院来文

本文件夹用于存放法院送达的各类程序性文书。

## 应放入的材料

| 材料类型 | 说明 | AI 提取的关键期限 |
|---------|------|------------------|
| 受理通知书 | 法院立案后送达 | — |
| 应诉通知书 | 告知被告应诉 | **答辩期**（通常15日） |
| 举证通知书 | 告知举证要求和期限 | **举证期限**（通常≥30日） |
| 传票 | 开庭时间、地点 | **开庭日期** |
| 证据交换通知 | 证据交换的时间和地点 | 证据交换日期 |
| 财产保全裁定 | 保全申请的结果 | 保全期限 |
| 判决书/裁定书 | 一审裁判结果 | **上诉期**（判决15日/裁定10日） |
| 缴费通知书 | 诉讼费预缴通知 | **缴费期限** |

## 使用方式

1. 收到法院文书后，将原件/复印件扫描件放入本文件夹
2. AI 自动读取全部法院来文，提取关键期限并写入《关键期限跟踪表.md》
3. 法院指定期限优先于法定期限，AI 会在期限到达前3日提醒用户

## 注意事项

- ⚠️ **关键期限不可错过**：答辩期、举证期、上诉期等超期可能导致失权
- 来文管理贯穿阶段二至四，形成完整的案件时间线
- 如收到电子送达的文书，建议打印后放入本文件夹存档
- 法院指定的期限与法定期限不一致时，以法院指定为准
