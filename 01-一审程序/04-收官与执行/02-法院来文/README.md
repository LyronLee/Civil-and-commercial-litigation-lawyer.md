# 法院来文

本文件夹用于存放阶段四（收官与执行）期间法院送达的裁判文书和程序性文书。

## 应放入的材料

| 材料类型 | 说明 | AI 提取的关键信息 |
|---------|------|------------------|
| 判决书 | 一审裁判结果 | 胜败结果、上诉期（15日）、履行期、诉讼费承担 |
| 裁定书 | 程序性裁定 | 裁定类型、上诉期（10日） |
| 调解书 | 调解结案文书 | 履行内容、履行期限 |
| 生效证明 | 判决生效证明 | 判决生效日期（计算申请执行期限起点） |
| 撤诉裁定 | 准予撤诉裁定 | — |
| 其他法院文书 | 补充通知、更正裁定等 | 视内容提取期限 |

## 使用方式

1. 收到裁判文书后，将原件/复印件扫描件放入本文件夹
2. AI 读取判决书/裁定书，确认结果、期限和后续路径
3. 根据判决结果分流：败诉→上诉状；胜诉→强制执行申请书；调解→跟踪履行

## 与阶段二法院来文的区别

| 对比项 | 阶段二法院来文 | 阶段四法院来文 |
|--------|--------------|--------------|
| 位置 | `02-诉讼材料及来文管理/05-法院来文/` | `04-收官与执行/02-法院来文/` |
| 内容 | 受理通知、举证通知、传票等程序文书 | 判决书、裁定书、调解书、生效证明 |
| 期限 | 答辩期、举证期、开庭日 | 上诉期、履行期、申请执行期 |

## 注意事项

- ⚠️ **上诉期限不可错过**：判决书送达之日起**15日**内、裁定书送达之日起**10日**内
- ⚠️ **申请执行期限**：判决生效后**2年**内，从履行期最后一日起算
- 判决生效证明是申请执行的重要依据，务必妥善保存
