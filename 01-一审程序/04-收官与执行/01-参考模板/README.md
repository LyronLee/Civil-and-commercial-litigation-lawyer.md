# 参考模板

本文件夹存放一审程序阶段四（收官与执行）的参考模板，供 AI 生成阶段产出时参照使用。

## 模板清单

| 序号 | 模板名称 | 用途 |
|------|---------|------|
| 00-代理词模板.md | 庭后提交法院的书面意见 | 基于庭审纪要，回应争议焦点，强化法律论证 |
| 01-上诉状模板.md | 不服一审判决的上诉文书 | 请求二审法院撤销/改判/发回重审 |
| 02-强制执行申请书模板.md | 胜诉后对方不履行时申请执行 | 申请法院强制执行生效判决 |
| 03-退费申请书模板.md | 判决生效后退诉讼费 | 申请退还预交的诉讼费 |
| 04-结案报告模板.md | 案件终局后的归档文书 | 案件经过、结果、文书清单、后续事项 |

## 使用方式

1. 本文件夹的模板为**参照模板**，AI 在执行 `SKILL.md` 工作流时会自动调用
2. 用户无需手动填写模板，按四个时点独立触发，唤起 AI 即可
3. 如需了解模板格式，可直接打开查看

## 工作流分支

| 触发时机 | 生成分支 | 输出文件 |
|---------|---------|---------|
| 开完庭后 | 代理词 | `04-阶段产出/01-代理词.md` |
| 收到判决书后（败诉） | 上诉状 | `04-阶段产出/02-上诉状.md` |
| 收到判决书后（胜诉） | 强制执行申请书 | `04-阶段产出/02-强制执行申请书.md` |
| 判决生效后 | 退费申请书 | `04-阶段产出/03-退费申请书.md` |
| 案件终局后 | 结案报告 + 复盘总结 | `04-阶段产出/04-结案报告.md` + `05-复盘总结.md` |

## 后续程序衔接

| 判决结果 | 后续程序 |
|---------|---------|
| 败诉/部分败诉，决定上诉 | → **02-二审程序/01-战略分析/** |
| 胜诉，对方不履行 | → **04-执行程序/01-执行实施主线/** |
| 发回重审 | → **01-一审程序/** 重新审理 |

## 产出位置

AI 根据本阶段 SKILL.md 生成的文件保存在 `04-阶段产出/` 中。
