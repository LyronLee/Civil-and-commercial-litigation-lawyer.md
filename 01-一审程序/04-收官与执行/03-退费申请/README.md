# 退费申请

本文件夹用于存放诉讼费退费相关的申请材料。

## 适用条件

- 判决确定诉讼费由对方承担（或部分承担）
- 原告已预交全部诉讼费
- 判决已生效

## 应放入的材料

| 材料类型 | 说明 |
|---------|------|
| 原缴费凭证 | 诉讼费交纳通知书/收据/银行转账记录 |
| 判决书/裁定书 | 证明诉讼费承担方式 |
| 收款账户信息 | 开户名、开户行、账号（需用户补充） |
| 退费申请书 | AI 生成的退费申请文书 |

## 使用方式

1. 确认判决生效后，将判决书和原缴费凭证放入本文件夹
2. 提供收款账户信息（如未在客户情况说明中填写）
3. 唤起 AI 生成退费申请书
4. 按受理法院的具体退费流程提交申请

## 注意事项

- ⚠️ 退费通常在**判决生效后**办理，刚收到判决书时不要急于退费
- 各地法院退费流程不同，具体以受理法院要求为准
- 退费金额 = 预交金额 - 本方应承担金额
- 如对方上诉，需等待二审结果确定后再办理退费
