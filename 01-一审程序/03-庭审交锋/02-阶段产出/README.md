# 阶段产出

本文件夹存放一审程序阶段三（庭审交锋）的 AI 生成工作成果。

## 预期产出文件

| 序号 | 文件名 | 内容说明 | 提交法院 |
|------|--------|---------|---------|
| 01-质证意见.md | 对对方证据的逐一质证 | 四性分析 + 策略性评价 | ✅ 是 |
| 02-开庭提纲.md | 庭审前内部准备文件 | 程序核对、发问设计、争议焦点、辩论要点、突发应对 | ❌ 否 |
| 03-庭审纪要.md | 庭审记录 | 基本信息、程序事项、法庭调查、法庭辩论、其他安排 | ❌ 否 |

## 使用方式

1. 本文件夹文件由 AI 根据 `SKILL.md` 工作流自动生成
2. **质证意见**审核后提交法院
3. **开庭提纲**打印出来庭审时参考，不提交法院
4. **庭审纪要**如尚未开庭，输出空模板；庭后按实际填写

## 注意事项

- 所有产出文件均标注"初稿，需律师审核后使用"
- 开庭提纲和庭审纪要已标注"内部使用，不提交法院"
- 对方提交新材料后，应重新生成质证意见
- 多次开庭时，庭审纪要按时间顺序分节记录
