# 阶段产出

本文件夹存放一审程序阶段一（战略把脉）的 AI 生成工作成果。

## 预期产出文件

| 序号 | 文件名 | 内容说明 |
|------|--------|---------|
| 01-材料索引与分类.md | 原始材料编目 | 按类别分类统计，每份材料标注编号、名称、页码、性质（有利/不利/待核实） |
| 02-案件事实时间轴.md | 按时间顺序梳理的关键事件 | 每个事件标注日期、内容、相关证据、性质 |
| 03-类案检索报告.md | 相关指导性案例、公报案例、上级法院裁判 | 裁判要旨提炼、对本案的参考价值 |
| 04-法条汇编.md | 实体法、司法解释、程序法依据 | 标注适用要点和条文内容 |
| 05-诉讼策略方案书.md | 核心策略文件 | 代理方向、核心主张、请求权/抗辩权基础、风险提示、Plan B |
| 06-待补充材料清单.md | 缺失材料及获取方法 | 优先级、获取方式、负责方、截止日期 |
| 00-定制化材料清单.md | 特殊案由的定制化材料清单 | 如检测到特殊案由，生成定制化清单 |

## 使用方式

1. 本文件夹文件由 AI 根据 `SKILL.md` 工作流自动生成
2. 生成后需由**执业律师审核**，确认事实梳理准确、法律检索完整、策略方向可行
3. 审核通过后，将本文件夹全部文件复制到 `02-诉讼材料及来文管理/01-上一阶段产出/`，继续阶段二工作

## 注意事项

- 所有产出文件均标注"初稿，需律师审核后使用"
- AI 检索的案例和法条有限，律师需在专业数据库中深度补充
- 策略方案书是后续全部阶段的核心指引，务必仔细审核
