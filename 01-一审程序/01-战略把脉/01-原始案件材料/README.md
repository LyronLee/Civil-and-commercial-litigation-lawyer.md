# 原始案件材料

本文件夹用于存放当事人提供的全部原始案件材料，是阶段一（战略把脉）的唯一输入源。

## 应放入的材料

| 类别 | 具体材料 | 优先级 |
|------|---------|--------|
| 主体身份材料 | 身份证/营业执照、法定代表人身份证明、授权委托书 | 🔴 必备 |
| 合同/协议 | 买卖合同、借款合同、租赁合同、建设工程合同等 | 🔴 必备 |
| 往来函件 | 催款函、律师函、沟通邮件、微信/短信记录 | 🟡 重要 |
| 付款凭证 | 银行转账记录、收据、发票、对账单 | 🔴 必备 |
| 履约凭证 | 发货单、签收单、验收报告、施工记录 | 🟡 重要 |
| 侵权证据 | 现场照片、视频、鉴定报告、医疗记录 | 视案由 |
| 沟通记录 | 会议纪要、通话录音（附文字整理稿） | 🟡 重要 |
| 其他书证 | 借条、欠条、担保函、抵押登记材料 | 视案由 |
| 关联案件材料 | 在先诉讼/执行/仲裁的裁判文书、证据材料 | 如有则放 |

## 使用方式

1. 将当事人提供的所有材料（无论格式、无论是否整洁）直接放入本文件夹
2. 无需预先整理或分类，AI 会在阶段一自动盘点、编目、评估完整性
3. 如为特殊案由，同时填写本文件夹内的 `特殊情况说明.md`

## 注意事项

- ⚠️ **原始材料仅复制整理，不移动、不修改、不删除原件**
- 材料越完整，AI 分析越准确。即使自认为"无关"的材料也建议放入
- 电子证据（微信记录、邮件等）建议导出为 PDF 或截图后放入
- 录音证据需同时提供文字整理稿
- 如材料涉及隐私，上传 AI 前建议对身份证号、银行账户等进行脱敏处理
