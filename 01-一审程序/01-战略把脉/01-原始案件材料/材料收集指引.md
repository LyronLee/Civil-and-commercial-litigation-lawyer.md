# 材料收集指引

> 请把与本案相关的全部材料放入本文件夹。
> 你提供的材料越完整，AI 分析越精准，策略制定越到位。

---

## 一、必备材料（打勾式清单）

### □ 1. 主体身份材料
- [ ] 我方身份证/营业执照复印件
- [ ] 我方法定代表人身份证明（如我方是法人或其他组织）
- [ ] 对方身份证/营业执照复印件（或国家企业信用信息公示报告）
- [ ] 对方法定代表人/负责人身份信息（如有）
- [ ] 委托代理手续（授权委托书、律师执业证、所函——如已委托律师）

### □ 2. 基础法律关系材料
- [ ] 合同/协议/订单/借条等确立法律关系的文件
- [ ] 补充协议、变更协议、备忘录、会议纪要
- [ ] 招投标文件、中标通知书（如适用）
- [ ] 章程/合伙协议/股东协议（公司类纠纷）

### □ 3. 履行过程材料
- [ ] 付款凭证（银行转账记录、汇票、本票、支票、发票、收据）
- [ ] 交付/验收凭证（送货单、签收单、验收报告、入库单）
- [ ] 工程进度资料（施工日志、监理报告、签证单——建设工程适用）
- [ ] 往来函件（催款函、违约通知、工作联系单、沟通邮件）

### □ 4. 争议发生材料
- [ ] 对方违约/侵权的证据（照片、视频、录音、鉴定报告、报警记录）
- [ ] 我方遭受损失的证据（费用单据、维修发票、评估报告、医院病历）
- [ ] 双方沟通记录（微信、短信、邮件截图——需保留原始载体）
- [ ] 第三方见证材料（证人证言、公证文书、媒体报道）

### □ 5. 维权过程材料（如有在先程序）
- [ ] 律师函、催告函、和解协议
- [ ] 仲裁裁决书、调解书
- [ ] 在先诉讼的起诉状、判决书、裁定书
- [ ] 执行案件材料（执行申请书、执行裁定、终本裁定）

---

## 二、建议补充材料（加分项）

- [ ] **对方财产线索**（银行账户、房产地址、车辆信息、股权结构）——有助于保全和执行
- [ ] **行业惯例/交易习惯证据**（行业协会文件、既往交易记录）
- [ ] **类案裁判文书**（你检索到的类似案件判决，供 AI 参考）
- [ ] **专业报告**（审计报告、资产评估报告、工程造价鉴定）

---

## 三、材料整理规范

| 项目 | 建议 |
|------|------|
| **文件命名** | `日期-材料类型-内容摘要` 例：`20240115-合同-买卖合同（甲乙双方）.pdf` |
| **格式优先** | PDF > 清晰照片（能辨认文字）> Word/Excel |
| **排序建议** | 按时间顺序排列，先旧后新；同一日期的按"先程序、后实体"排列 |
| **页码** | 如有纸质材料扫描件，建议按时间顺序连续编页码 |
| **份数** | 如已知法院立案要求，起诉状和证据材料准备"被告人数 + 1 份" |

---

## 四、隐私保护提示

- 如通过在线方式传输材料，建议对**身份证号、银行账户**做脱敏处理（用 `****` 代替部分数字），或另附脱敏说明
- 如某些材料暂时无法提供，请在文件名中标注`【暂缺】`，AI 会评估影响并告诉你如何补充

---

## 五、如果你的案子比较特殊

以上清单是**通用版**，适用于大多数合同纠纷、侵权纠纷、借贷纠纷等常见案件。

如果你的案子属于以下情形，通用清单可能不够用：
- 建设工程、房地产、知识产权、海事海商、证券期货等专业领域
- 涉及涉外主体、上市公司、金融机构、政府机关
- 需要特殊鉴定、评估、行政审批
- 证据形式特殊（如大量电子数据、区块链存证、公证债权文书）

**请在本文件夹中创建 `特殊情况说明.md`，按模板填写你的案件特殊性。**
**AI 读取后会自动检索该类案由的专业材料清单，为你定制一份更精准的材料收集方案。**
