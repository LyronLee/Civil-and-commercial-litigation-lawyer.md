# 损害公司债权人利益责任纠纷（人格否认）· 材料收集清单（差异版）

> 通用清单已覆盖的基础材料不再重复。以下仅为人格否认之诉的**额外/特殊要求**。
> 完整规则见 `00-通用基础设施/案由适配索引.md` 第三条。

---

## 一、必须补充的材料（通用清单未覆盖）

### 1. 公司人格混同证据（五维混同）

**人员混同：**
- [ ] 法定代表人、董事、高管交叉任职的证明（工商登记、任命文件）
- [ ] 共用财务人员、共用劳动合同
- [ ] 同一地址办公、共用联系电话

**业务混同：**
- [ ] 经营范围重合的证明
- [ ] 业务资源无偿转移的证据（客户名单、供应商名单共用）
- [ ] 对外宣传混同（共用官网、共用品牌、统一对外表述）

**财务混同：**
- [ ] **公司账簿与股东账簿不分的证据**（核心）
- [ ] 股东无偿使用公司资金/财产，不作财务记载
- [ ] 公司资金偿还股东债务，或供关联公司无偿使用
- [ ] 公司财产记载于股东名下
- [ ] 股东自身收益与公司盈利不加区分

**组织机构混同：**
- [ ] 共用办公地址、共用印章
- [ ] 无独立决策程序（股东一人决定公司重大事项）

**财产边界消失：**
- [ ] 关联公司间资产随意调拨且无合理对价
- [ ] 无独立的财务账簿或财务账簿无法区分

### 2. 控制权滥用证据
- [ ] 股东/实际控制人未经决策程序转移重大资产的证据
- [ ] 关联交易无书面合同或价格不公允
- [ ] 公司沦为股东逃避债务工具的证据（债务产生后突击转移资产）

### 3. 因果关系证据
- [ ] 公司偿债能力因股东行为而严重降低的证据（资产负债表、审计报告、终本裁定）

### 4. 一人公司特殊证据（如适用）
- [ ] 公司财务会计报告（股东需自证财产独立，否则举证责任倒置）

## 二、材料收集的特殊要求

| 项目 | 要求 |
|------|------|
| 诉讼地位 | 债权已确认→股东被告/公司第三人；一并提起→共同被告；未经确认直接诉股东→法院释明追加公司 |
| 管辖 | 侵权行为地（含原告住所地）或被告住所地，不适用公司诉讼特殊管辖 |
| 举证责任 | 一般：债权人举证；一人公司：股东自证财产独立 |
| 关联关系时点 | 关联关系认定的时间节点应是**债务形成时**开始，而非诉讼形成时 |

---

## 三、调取建议

- 公司工商内档（含股东信息、变更记录）→ 工商登记机关
- 公司财务账簿、银行流水 → 申请法院调查令（公司通常不配合）
- 关联公司股权结构 → 企查查/天眼查/企业信用信息公示系统
- 一人公司财务报告 → 公司自行提供或申请法院调取
