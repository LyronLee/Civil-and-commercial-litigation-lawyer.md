# 债权人撤销权纠纷 · 材料收集清单（差异版）

> 通用清单已覆盖的基础材料不再重复。以下仅为撤销权之诉的**额外/特殊要求**。
> 完整规则见 `00-通用基础设施/案由适配索引.md` 第二条。

---

## 一、必须补充的材料（通用清单未覆盖）

### 1. 债务人处分财产行为的完整证据
- [ ] **转让合同/赠与协议/股权转让协议**（核心证据）
- [ ] **工商变更登记**（如为股权转让、法人变更）
- [ ] **不动产登记信息**（如为房产、土地转让）
- [ ] **放弃债权声明/债权债务抵销协议**
- [ ] **担保设立文件**（如为他人债务提供担保）
- [ ] **无偿转让的辅助证据**：资金流向（如转让给亲属但无对价）

### 2. 相对人/受益人信息（用于恶意认定）
- [ ] 相对人身份证明
- [ ] 债务人与相对人的**关联关系证据**：亲属关系证明、股东关系、实际控制人关系
- [ ] 相对人支付对价的证据（银行流水、付款凭证）——用于判断"明显不合理低价"

### 3. 诈害行为的损害证据
- [ ] 债务人处分财产前后的**资产负债表对比**
- [ ] 债务人**无其他财产清偿债务**的证据（终本裁定、执行不能证明）
- [ ] 债务人**明知负债仍处分财产**的证据（处分行为发生在债务确认后）

### 4. 除斥期间核查
- [ ] 债权人**知道或应当知道撤销事由之日**的证据（如：何时发现转让行为）
- [ ] 债务人行为发生之日（用于计算 5 年最长保护期）

---

## 二、材料收集的特殊要求

| 项目 | 要求 |
|------|------|
| 被告范围 | 债务人和相对人为**共同被告**，不能只起诉一方 |
| 诉讼请求 | 必须同时请求"撤销"+"恢复原状/返还财产/折价补偿"，遗漏将导致另诉 |
| 有偿行为 | 如为有偿转让，需重点收集相对人**恶意**的证据（关联关系、知情证据） |
| 连环转让 | 如财产已被再次转让，需收集次相对人的信息及善意/恶意证据 |

---

## 三、调取建议

- 不动产登记信息 → 不动产登记中心查询
- 工商登记信息 → 企业信用信息公示系统
- 银行流水/资金流向 → 申请法院调查令
- 亲属关系证明 → 户口本、结婚证、亲属声明
