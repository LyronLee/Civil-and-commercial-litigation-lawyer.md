# 股东出资纠纷（加速到期/追缴）· 材料收集清单（差异版）

> 通用清单已覆盖的基础材料不再重复。以下仅为股东出资纠纷的**额外/特殊要求**。
> 完整规则见 `00-通用基础设施/案由适配索引.md` 第四条。

---

## 一、必须补充的材料（通用清单未覆盖）

### 1. 出资约定与实缴证据
- [ ] **公司章程**：认缴出资额、出资方式、出资期限
- [ ] **股东会决议**：关于出资、增资、减资的决议
- [ ] **验资报告**（如有实缴）
- [ ] **银行流水**：股东向公司转账记录，标注"出资款"或"投资款"
- [ ] **工商登记信息**：实缴出资公示信息

### 2. 加速到期的触发证据（择一）
- [ ] **公司作为被执行人的终本裁定**（穷尽执行措施无财产可供执行）
- [ ] **债务产生后延长出资期限的股东会决议**
- [ ] **公司资不抵债的证据**：资产负债表、审计报告、破产受理裁定

### 3. 抽逃出资证据（如主张抽逃）
- [ ] **资金流向**：股东出资后的公司账户流水，追踪资金转出路径
- [ ] **关联交易**：股东与公司之间的交易合同、资金往来
- [ ] **虚假合同**：用于转出资金的虚假交易合同
- [ ] **第三方代收**：资金最终流入股东或关联方的证据

### 4. 股权转让证据（如涉及原股东/受让股东）
- [ ] **股权转让协议**
- [ ] **股权转让时的出资状态**（转让时是否已出资）
- [ ] **受让股东是否知晓出资瑕疵**

---

## 二、材料收集的特殊要求

| 项目 | 要求 |
|------|------|
| 管辖 | 存在争议（公司住所地 vs 被告住所地），需提前确认受诉法院立场 |
| 案由区分 | 注意与"损害公司债权人利益责任纠纷"（人格否认）区分：出资纠纷重"出资义务违反"，人格否认重"滥用控制权" |
| 被告范围 | 未出资/抽逃出资的股东；股权转让中的原股东和受让股东 |
| 一人公司 | 如为一人公司，注意举证责任倒置规则 |

---

## 三、调取建议

- 公司工商内档（章程、股东信息、变更记录）→ 工商登记机关
- 公司银行账户流水 → 申请法院调查令
- 股东个人银行账户流水 → 申请法院调查令（需证明与抽逃出资相关）
- 公司财务会计账簿 → 申请法院调取或公司自行提供
