# 执行异议之诉 · 材料收集清单（差异版）

> 通用清单已覆盖的基础材料不再重复。以下仅为执行异议之诉的**额外/特殊要求**。
> 完整规则见 `00-通用基础设施/案由适配索引.md` 第五条。
> 注意：执行异议之诉是执行程序中产生的独立诉讼类型，与一审诉讼结构类似但在管辖、前置程序、起诉期限等方面存在显著差异，材料范围有特殊要求。

---

## 一、必须补充的材料（通用清单未覆盖）

### 1. 前置程序材料（必备）
- [ ] **执行异议裁定书**（执行异议之诉的前置条件）
- [ ] **执行依据**：生效判决书/裁定书/调解书/仲裁裁决/公证债权文书
- [ ] **执行案件受理通知书、执行裁定书**

### 2. 执行标的权属材料（核心）

**如为不动产：**
- [ ] 房产证/不动产权证
- [ ] 买卖合同/赠与合同/继承文书
- [ ] 付款凭证（银行流水、收据）
- [ ] 占有使用证据（物业费、水电费缴纳记录、入住证明）
- [ ] 未过户的原因说明及证据（如开发商原因、查封原因）

**如为动产：**
- [ ] 购买凭证（发票、合同）
- [ ] 占有使用证据（照片、视频、使用记录）
- [ ] 权属登记证明（车辆登记证、船舶所有权证书）

**如为股权：**
- [ ] 股权转让协议
- [ ] 股东名册/工商登记
- [ ] 出资证明/付款凭证

**如为银行存款：**
- [ ] 账户归属证明（开户资料、账户使用记录）
- [ ] 资金往来说明（如账户内资金为案外人所有）

### 3. 权利形成时间证据
- [ ] 证明权利形成时间**早于查封时间**的证据（关键！）
- [ ] 查封时间的证明（执行裁定书载明的查封日期、查封登记表）

### 4. 权利性质证据
- [ ] 权利为物权/物权期待权/租赁权/所有权的证据
- [ ] 权利足以排除执行的法律依据和事实依据

---

## 二、材料收集的特殊要求

| 项目 | 要求 |
|------|------|
| 前置程序 | 必须先经过执行异议程序，对裁定不服才能起诉 |
| 起诉期限 | 收到执行异议裁定之日起 **15 日内** |
| 专属管辖 | 执行法院管辖，不得约定其他法院 |
| 权利形成时间 | 核心抗辩点：权利形成时间必须早于查封时间 |
| 权利性质 | 必须证明享有"足以排除强制执行的民事权益" |

---

## 三、四类执行异议之诉的举证侧重

| 类型 | 举证侧重 |
|------|---------|
| 案外人执行异议之诉 | 案外人享有所有权/物权期待权等实体权利，且足以排除执行 |
| 申请执行人执行异议之诉 | 执行标的属于被执行人责任财产，案外人不享有排除执行的权利 |
| 追加/变更被执行人异议之诉 | 股东出资情况、公司财产独立性（一人公司）、清算程序合法性 |
| 执行分配方案异议之诉 | 各债权的真实性、数额、优先权顺位、分配比例计算 |

---

## 四、调取建议

- 执行案件卷宗（查封时间、查封顺序）→ 执行法院档案室
- 不动产登记信息 → 不动产登记中心
- 车辆登记信息 → 车辆管理所
- 银行账户信息 → 申请法院调查令
- 股权登记信息 → 工商登记机关/企业信用信息公示系统
