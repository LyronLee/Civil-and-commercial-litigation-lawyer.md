# 参考模板

本文件夹存放一审程序阶段一（战略把脉）的参考模板，供 AI 生成阶段产出时参照使用。

## 模板清单

| 序号 | 模板名称 | 用途 |
|------|---------|------|
| 00-材料收集清单.md | 通用材料收集指引 | 逐项检查案件材料的完整性，标注缺失项 |
| 01-案件事实时间轴模板.md | 事实梳理框架 | 按时间顺序排列关键事件，标注证据和性质 |
| 02-类案检索报告模板.md | 类案检索格式 | 检索目标、检索策略、案例列表、裁判要旨、对本案参考价值 |
| 03-法条汇编模板.md | 法律依据整理格式 | 实体法、司法解释、程序法条文及适用要点 |
| 04-诉讼策略方案书模板.md | 策略制定框架 | 代理方向、核心主张、请求权/抗辩权基础、风险提示、Plan B |
| 05-类案检索报告模板.md | （备用/扩展模板） | — |

## 特殊案由适配

如案件属于特殊案由（代位权/撤销权/人格否认/股东出资/执行异议之诉），AI 会自动调用 `特殊案由适配/` 子文件夹中的专属材料收集清单。

## 使用方式

1. 本文件夹的模板为**参照模板**，AI 在执行 `SKILL.md` 工作流时会自动调用
2. 用户无需手动填写模板，将原始材料放入 `01-原始案件材料/` 后，唤起 AI 即可
3. 如需了解模板格式，可直接打开查看

## 产出位置

AI 根据本阶段 SKILL.md 生成的文件保存在 `03-阶段产出/` 中。
