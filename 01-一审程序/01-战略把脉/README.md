# 01 战略把脉：案件启动指南

> 🎯 **阶段定位**：这是诉讼流程的第一步，核心任务是"理清材料、理顺事实、搞清法律、定好方案"。
>
> 拿到案件后，你可能有一堆杂乱的合同、聊天记录、转账截图……不知道从何下手。这个阶段就是帮你把混乱变成清晰。

这个文件夹里有几类东西，先搞清楚它们是干嘛的，你才不会迷路：

|--------|--------|-----------|-----------|

## 🚀 三步使用流程

### 第一步：扔材料 → `01-原始案件材料/`

把你手头的**所有**案件相关材料，全部放进 `01-原始案件材料/` 文件夹：

| 材料类型 | 举例 |
|---------|------|
| 身份材料 | 身份证、营业执照、法定代表人证明 |
| 合同协议 | 书面合同、补充协议、订单、备忘录、借条欠条 |
| 沟通记录 | 微信/短信/QQ/邮件截图、通话录音 |
| 款项凭证 | 转账记录、银行流水、支付宝/微信支付截图、发票、收据 |
| 履行证据 | 送货单、验收单、物流单据、签收单、交房记录 |
| 违约/损害 | 催告函、违约通知、损害照片、医疗记录、维修单据 |
| 其他 | 证人信息、鉴定报告、公证书、报警记录、事故认定书 |

> 💡 **原则**：宁可多放，不要少放。你觉得"可能有关"的材料，都放进去。AI会帮你判断哪些有用。

> ⚠️ **特殊案件提醒**：如果你的案子属于以下类型，请在 `01-原始案件材料/` 中额外填写 `特殊情况说明.md`（模板已预置在文件夹内），AI会据此生成定制化材料清单：
> - 债权人代位权纠纷
> - 债权人撤销权纠纷
> - 损害公司债权人利益责任纠纷（人格否认）
> - 股东出资纠纷（加速到期/抽逃出资）
> - 执行异议之诉

---

### 第二步：唤AI → "帮我分析这个案子"

打开你的AI助手，告诉ta：

> **"我在做01战略把脉阶段，请帮我分析这个案子。材料在原始案件材料文件夹里。"**

AI会按以下步骤引导你：

1. **盘点材料** — 读取你放的材料，列出清单，告诉你还缺什么、怎么补
2. **整理材料索引** — 把杂乱的原始材料按类别编目，生成索引表（方便后续查找）
3. **梳理时间轴** — 按时间线整理案件事实，标注有利/不利/待核实
4. **检索法律** — 帮你找到相关法律条文和类似判例
5. **制定策略** — 形成整体诉讼/应诉方案

> 💡 **技巧**：你可以随时打断AI，补充新材料或纠正理解偏差。材料可以随时补放到 `01-原始案件材料/` 里。

---

### 第三步：产出文书

这个阶段最终要产出 **5份核心文件**，全部保存在 `03-阶段产出/` 文件夹内：

| 产出文件 | 作用 |
|---------|------|
| 📋 01-材料索引与分类 | 把原始材料按类别编目（盘点已有材料），下一阶段的"导航地图" |
| 📋 06-待补充材料清单 | 缺什么材料、为什么缺、怎么补、谁去补、什么时候补完——"行动清单" |
| 📄 02-案件事实时间轴 | 帮你理清案件来龙去脉，发现逻辑漏洞 |
| 📄 03-类案检索报告 | 找到支持你观点的法院判例，预判裁判倾向（⚠️ AI无法直接检索，需你自己去数据库查，AI帮你分析） |
| 📄 04-法条汇编 | 整理你的"法律武器库"，明确依据 |
| 📄 05-诉讼策略方案书 | 整个诉讼的"作战地图"，下一阶段的行动纲领 |

> 💡 **重要**：`01-原始案件材料/` 里的文件**保持原样不动**，AI不会去修改或删除它们。所有整理和生成的新内容都放在 `03-阶段产出/` 里。这样既安全又清晰。

## 🤝 和解与调解

诉讼不是唯一路径。在阶段一，就应该评估和解/调解的可行性：

### 什么时候考虑和解/调解

| 情形 | 建议 |
|------|------|
| 证据对我方有利但不充分 | 可考虑适度让步达成和解，避免败诉风险 |
| 对方有履行意愿但暂时困难 | 可协商分期履行或延期履行方案 |
| 案件标的不大，诉讼成本过高 | 和解可能更经济 |
| 双方有长期合作关系 | 和解有助于维护关系 |
| 时间成本高，希望快速解决 | 调解通常比诉讼更快 |

### 调解策略纳入方案书

AI 在生成 `05-诉讼策略方案书.md` 时，会同步评估：
- **调解可行性**：基于证据强度和对方态度
- **调解底线**：我方可以接受的最低条件
- **最优调解方案**：比判决更有利的和解条件
- **调解时机**：哪个阶段调解最有利（诉前/庭前/庭中）

> 💡 **调解 ≠ 让步**：好的调解方案是基于证据和策略的主动设计，不是被动妥协。

### 阶段一达成和解的快捷路径

如在阶段一（战略把脉）就决定和解：
1. 由律师与对方沟通和解方案
2. 达成一致后，可选择：
   - **诉前和解**：不起诉，直接签订和解协议 → 进入阶段四的结案复盘
   - **立案后调解**：立案后申请法院调解 → 法院出具调解书 → 进入阶段四的结案复盘

## ❓ 常见问题

**Q1: 我不知道我的案子属于什么案由？**
> 没关系，先把材料全部扔进去，AI看了之后会帮你判断案由（比如是借贷纠纷、买卖合同纠纷还是劳动争议）。

**Q2: 我手上没有某些材料怎么办？**
> AI会评估：这个材料是"必须有"还是"有了更好"。对于关键缺失材料，AI会告诉你补救办法，比如：
> - 自己再找找（银行APP查流水、微信导出聊天记录）
> - 申请法院调取（对方身份信息、银行流水等）
> - 补充取证（录音、拍照、公证等）

**Q3: 我是被告，刚收到法院传票，也适用这个阶段吗？**
> **完全适用。** 被告更需要战略把脉。你需要：
> - 分析原告的起诉状和证据（材料里放上去）
> - 寻找原告诉求中的漏洞
> - 制定抗辩策略（是否否认、是否反诉、是否提管辖异议）

**Q4: 这个阶段要写起诉状/答辩状吗？**
> **不需要。** 起诉状和答辩状是"阶段二：弹药筹备"的事。阶段一的任务是"想清楚"，阶段二的任务才是"写清楚"。

**Q5: 为什么有些材料AI读不了？**
> 目前AI最擅长读取：PDF、Word、TXT、Markdown、图片（文字类）。
> 如果材料是：手写纸条、模糊照片、语音录音 → 建议你先整理成文字版再放进去。

**Q6: AI能帮我找法院判例吗？**
> **能初步检索，但不能替代专业数据库。**
> - ✅ AI**能**检索：法院官网公开的典型案例、指导性案例、法官文章、部分公开裁判文书
> - ❌ AI**不能**保证：检索到所有相关案例、访问付费数据库完整内容
> - ❌ AI**不能**编造案例（这是严重违规的）
> - ✅ AI**能**帮你：确定检索什么、用什么关键词、去哪个平台查
> - ✅ AI**能**帮你：分析你找到的裁判文书，提炼要点，写成报告
>
> **正确做法**：
> 1. AI先帮你初步检索公开的典型案例
> 2. 你自己去 MetaLaw / 中国裁判文书网 / 北大法宝 进行深度检索补充
> 3. 把查到的裁判文书发给AI，AI帮你分析写报告
> 4. 所有案例必须标明来源，提交前复核真实性

## ⚠️ 重要提示

1. **材料真实性**：你提供的材料必须真实。虚假材料会导致败诉甚至法律责任。
2. **隐私保护**：`01-原始案件材料/` 里的内容只在本地处理，不会上传到外部。
3. **AI是助手不是律师**：AI帮你整理思路、生成文书草稿，但重大决策（是否起诉、是否和解、是否上诉）需要你自己判断，复杂案件建议咨询专业律师。
4. **随时可回退**：诉讼过程中发现新材料，随时可以回到这个阶段补充时间轴和策略。
