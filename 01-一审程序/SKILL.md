# 01 一审程序 SKILL

> 核心理念：一审是民事诉讼的起点。本程序配备独立完整的四阶段工作流，覆盖从混乱的材料堆到清晰的诉讼策略，再到完整的法律文书和庭审应对的全部过程。

---

## 触发条件

用户表达以下意图时触发：
- "我要起诉"
- "刚接到一个案子，不知道怎么开始"
- "收到法院传票了，我是被告"
- "帮我整理一下这些材料"
- "起草起诉状/答辩状"
- "准备开庭"

---

## 前置依赖

触发本SKILL前必须确认：
1. 案件属于民事诉讼范围（非刑事、行政）
2. 如为原告起诉：确认诉讼时效未届满
3. 如为被告应诉：确认已收到起诉状副本和应诉通知书

---

## Step 0：条件确认与角色判定

### 0.1 向用户确认

- 我方是原告还是被告？
- 案件基本类型（合同/侵权/婚姻家庭/继承/物权等）
- 是否有特殊案由（代位权/撤销权/人格否认/股东出资/执行异议之诉）
- 是否已有法院文书（传票、应诉通知书等）

### 0.2 角色判定

| 角色 | 工作流特点 | 进入阶段 |
|------|-----------|---------|
| 原告 | 建设型：收集材料→制定策略→起草起诉状→主动推进 | 01-战略把脉/ |
| 被告 | 拆弹型：分析对方材料→寻找漏洞→起草答辩状→防守反击 | 01-战略把脉/ |

> 💡 角色判定后，各阶段 SKILL.md 自动切换对应工作流。

---

## Step 1：阶段路由

```
接案/收到传票
    │
    ├─ 材料收集与整理 ───────────────→ 01-战略把脉/
    │    （索引、时间轴、类案检索、法条汇编、策略方案书）
    │
    ├─ 文书制备与来文管理 ───────────→ 02-诉讼材料及来文管理/
    │    （起诉状/答辩状、证据目录、财产保全、来文跟踪）
    │
    ├─ 庭审准备 ─────────────────────→ 03-庭审交锋/
    │    （质证意见、开庭提纲、庭审纪要）
    │
    └─ 庭后代理与闭环 ───────────────→ 04-收官与执行/
         （代理词、判决应对、退费、结案复盘）
```

---

## Step 2：特殊案由适配

如案件属于以下特殊案由，阶段一自动调用适配规则：

| 案由 | 适配内容 |
|------|---------|
| 债权人代位权纠纷 | 被告=次债务人、管辖=次债务人住所地、诉讼请求限定 |
| 债权人撤销权纠纷 | 共同被告=债务人和相对人、双重请求（撤销+返还）、除斥期间 |
| 损害公司债权人利益责任纠纷 | 三种当事人组合、连带责任、人格混同证据 |
| 股东出资纠纷 | 加速到期触发条件、补充赔偿责任、出资瑕疵证据 |
| 执行异议之诉 | 四类当事人组合、专属管辖、前置程序 |

> 💡 适配规则详见 `00-通用基础设施/案由适配索引.md`。

---

## Step 3：与相邻程序的衔接

### 3.1 一审结束后的路径选择

| 一审结果 | 我方角色 | 后续程序 |
|---------|---------|---------|
| 胜诉，对方履行 | 原告 | 结案复盘（无需进入其他程序） |
| 胜诉，对方不履行 | 原告 | → **04-执行程序** |
| 败诉 | 原告/被告 | → **02-二审程序**（判决15日内） |
| 部分胜诉 | 任何一方 | 综合评估 → 02-二审 或 04-执行 |
| 调解结案 | 任何一方 | 调解书生效 → 对方不履行 → **04-执行程序** |
| 撤诉 | 原告 | 如需要可重新起诉 |

### 3.2 从前序程序进入

| 前序程序 | 进入一审的触发条件 |
|---------|-------------------|
| 二审发回重审 | 收到二审发回重审裁定 → 回到一审重新审理 |
| 再审发回重审 | 收到再审发回重审裁定 → 回到一审重新审理 |

## 质量检查清单（一审程序通用版）

| 序号 | 检查项 | 合格标准 |
|------|--------|---------|
| 1.1 | 诉讼时效 | 未届满（通常3年，特殊情形除外） |
| 1.2 | 管辖法院 | 正确（级别管辖+地域管辖） |
| 1.3 | 当事人信息 | 完整、准确（姓名/名称、住所、联系方式） |
| 1.4 | 诉讼请求 | 明确、具体、可执行 |
| 1.5 | 事实与理由 | 围绕诉讼请求展开，有证据支撑 |
| 1.6 | 证据材料 | 真实、合法、与待证事实有关联 |
| 1.7 | 举证期限 | 在法院指定的举证期限内提交 |
| 1.8 | 答辩期 | 被告在收到起诉状副本后15日内答辩 |
| 1.9 | 上诉期 | 判决15日/裁定10日（送达之日起算） |
| 1.10 | 所有文件编码 | UTF-8，中文显示正常 |
| 1.11 | AI产出标注 | 所有提交法院的文书标注"初稿，需律师审核后使用" |

---

## 关键期限提醒

> 🔴 **诉讼时效**：通常 **3 年**（自知道或应当知道权利受到损害之日起）
>
> 🔴 **答辩期**：收到起诉状副本之日起 **15 日**
>
> 🔴 **举证期限**：法院指定（通常 ≥30 日）
>
> 🔴 **上诉期**：判决 **15 日** / 裁定 **10 日**（送达之日起算）
>
> 🔴 **申请执行期**：判决生效后 **2 年**
>
> ⚠️ **法院指定的期限优先于法定期限**。收到法院文书后，务必以文书载明的期限为准。
