# 执行分配方案异议之诉 SKILL

> ⚠️ 依据2023年修正《民事诉讼法》（2024年1月1日施行）及《最高人民法院关于适用〈中华人民共和国民事诉讼法〉的解释》第506-510条。

---

## 触发条件

用户表达以下意图时触发：
- "对执行分配方案有异议"
- "有人反对我的分配方案异议"
- "准备执行分配方案异议之诉"
- "分配方案不公平，要起诉"

---

## 前置依赖

触发本SKILL前必须确认：
1. 执行法院已制作并送达执行分配方案
2. 已向法院提出书面异议
3. 有人（其他债权人或被执行人）反对该异议
4. 起诉期限未届满（收到反对意见之日起**15日内**）

> ⚠️ **15日起诉期限是不可变期间**，逾期不起诉，按原分配方案执行。

## Step 1：战略分析（阶段一）

### 1.1 前置程序确认

读取 `05-前置材料/`，确认：
- 执行分配方案已制作并送达
- 已提出书面异议
- 有人反对该异议
- 反对意见日期 → 计算起诉期限（收到之日起15日）

### 1.2 分配方案分析

逐项核对分配方案的计算：

```
分配顺序：执行费用 → 优先受偿债权 → 普通债权按比例分配
```

| 核对项 | 核对内容 |
|--------|---------|
| 执行费用 | 执行费、评估费、拍卖费等是否计算准确 |
| 优先债权 | 担保物权、工程款优先权等是否优先扣除 |
| 普通债权 | 各债权人债权数额是否正确 |
| 分配比例 | 普通债权的分配比例计算是否准确 |
| 预留份额 | 是否遗漏了应预留的份额（如未到期债权） |

### 1.3 异议事由确认

明确对分配方案的哪些内容有异议：

| 常见异议事由 | 说明 |
|-------------|------|
| 债权数额错误 | 某债权人的债权数额计算有误 |
| 优先权不成立 | 某债权人主张的优先权不符合法律规定 |
| 分配顺序错误 | 优先债权和普通债权的顺序有误 |
| 分配比例错误 | 普通债权的分配比例计算有误 |
| 遗漏债权 | 某合法债权未列入分配方案 |
| 执行费用过高 | 执行费用的计算或扣除不合理 |

### 1.4 策略分析

1. **法律依据检索**：检索分配顺序、比例、优先权等相关规定
2. **数字核对**：逐项核对分配方案的计算
3. **利益平衡**：评估诉讼成本与可能获得的分配利益
4. **多方关系**：注意多个债权人之间的利益冲突

### 1.5 生成《执行分配方案异议之诉策略方案书》

输出到 `06-阶段产出/01-策略方案书.md`，包含：
- 分配方案逐项核对结果
- 异议事由及法律依据
- 数字差异计算
- 争议焦点预判
- 利益评估
- 后续路径建议

## Step 2：材料制备（阶段二）

### 2.1 起诉状

参考 `01-参考模板/00-起诉状模板.md`，起草起诉状：

- **原告**：异议人（对分配方案有异议的债权人/被执行人）
- **被告**：反对人（反对该异议的债权人/被执行人）
- **管辖**：执行法院（专属管辖）
- **诉讼请求**：
  1. 撤销原执行分配方案
  2. 重新制作分配方案（明确具体的修改内容）

> ⚠️ **必须包含前置程序说明**：已提出书面异议，【反对人】于【日期】提出反对意见。

### 2.2 证据目录

参考 `01-参考模板/02-证据目录模板.md`（如有），整理证据：

| 证据类别 | 证明目的 |
|---------|---------|
| 分配方案 | 原分配方案的内容及错误之处 |
| 债权凭证 | 各债权人的债权数额和性质 |
| 优先权证明 | 优先受偿权的依据（或否定优先权的依据） |
| 执行费用凭证 | 费用扣除的合理性 |
| 异议及反对材料 | 前置程序已完成 |
| 计算对比表 | 原方案 vs 正确方案的逐项对比 |

### 2.3 计算对比表

制作详细的计算对比表，逐项说明原方案的错误和正确的计算方式：

| 项目 | 原方案 | 正确计算 | 差额 | 依据 |
|------|--------|---------|------|------|

| 优先债权A | | | | |
| 普通债权分配 | | | | |
| 我方应得 | | | | |

## Step 3：庭审交锋（阶段三）

### 3.1 开庭提纲

参考 `01-参考模板/03-开庭提纲模板.md`（如有），生成开庭提纲：

1. **程序核对**
2. **分配方案错误陈述**：逐项说明原方案的错误
3. **正确计算展示**：展示正确的分配计算
4. **争议焦点预判**：
   - 债权数额是否正确
   - 优先权是否成立
   - 分配比例是否准确
5. **法官发问预判**：
   - "债权数额的依据是什么？"
   - "优先权成立的时间和范围？"
   - "执行费用的计算依据？"
6. **多方当事人注意**：多个原告/被告之间的利益冲突

### 3.2 质证意见

针对各债权人的债权凭证和优先权证据，逐项质证：
- 债权凭证的真实性和数额
- 优先权的成立时间和范围
- 执行费用的合理性
- 分配比例计算的准确性

### 3.3 庭审纪要

开庭后24小时内生成庭审纪要。

### 3.4 代理词

参考 `01-参考模板/04-代理词模板.md`（如有），提交书面代理词：
- 围绕分配方案的错误展开
- 引用《民事诉讼法解释》第506-510条
- 数字计算必须逐项核对，准确无误

---

## Step 4：收官（阶段四）

### 4.1 判决分析

- **胜诉**：按判决重新制作分配方案并分配
- **败诉**：按原分配方案执行分配

### 4.2 后续路径

> ⚠️ **重要**：执行分配方案异议之诉**一审即为终审**，判决**不可上诉**。

无论胜败，执行法院根据判决结果完成分配。

## 质量检查清单

生成全部文件后，逐项核对：

- [ ] 前置程序是否已完成（已提出异议且被反对）
- [ ] 起诉期限是否在收到反对意见之日起15日内
- [ ] 管辖法院是否为执行法院（专属管辖）
- [ ] 原告是否为异议人，被告是否为反对人
- [ ] 诉讼请求是否为"撤销分配方案+重新分配"
- [ ] 起诉状是否包含前置程序说明
- [ ] 分配方案的计算是否逐项核对，数字准确
- [ ] 涉及多个债权人的，债权数额和分配比例是否清晰列明
- [ ] 优先权的成立时间和范围是否准确
- [ ] 法条引用是否准确（《民事诉讼法解释》第506-510条）
- [ ] 是否已提示一审终审（不可上诉）
- [ ] 全部文件编码为UTF-8，中文显示正常
- [ ] 所有文件生成后标注"初稿，需律师审核后使用"

## 输出规范

- 所有文书按执行分配方案异议之诉的独立结构生成
- 分配方案的计算必须逐项核对，数字准确
- 涉及多个债权人的，债权数额和分配比例必须清晰列明
- 起诉状中必须说明前置程序（已提出异议且被反对）
- 管辖法院必须是执行法院（专属管辖）
- **特别提醒用户**：执行分配方案异议之诉一审终审，判决不可上诉
- 所有期限提醒以法院指定为准，法院未指定的适用法定期限
- 所有文件生成后标注"初稿，需律师审核"
