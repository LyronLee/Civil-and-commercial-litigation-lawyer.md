# 前置材料

本文件夹用于存放提起执行分配方案异议之诉前的前置程序材料。

## 应放入的材料

| 材料名称 | 说明 | 必要性 |
|---------|------|--------|
| **执行分配方案** | 执行法院制作的财产分配方案 | 必须 |
| **分配方案异议书** | 异议人向法院提交的书面异议 | 必须 |
| **反对意见书** | 其他债权人/被执行人对异议的反对意见 | 必须 |
| **执行依据** | 生效判决书/裁定书/调解书等 | 必须 |
| **债权凭证** | 各债权人的债权数额和性质证明 | 建议 |
| **优先权证明** | 担保物权、工程款优先权等证明 | 视情形 |
| **执行费用凭证** | 执行费用的支出凭证 | 建议 |

## 使用方式

1. 在唤起 AI 前，将上述材料按顺序放入本文件夹
2. AI 会读取全部前置材料，确认前置程序已完成
3. 重点核对：是否已提出书面异议且有人反对（缺一不可）

## 注意事项

- 起诉期限：收到反对意见之日起 **15 日内**（不可变期间）
- 管辖法院：执行法院（专属管辖）
- **一审终审**：执行分配方案异议之诉的判决不可上诉
