# 执行异议之诉 · 起诉状制作指引

> 执行异议之诉是执行程序中产生的独立诉讼类型。起诉状制作需遵循以下特殊规则，不再引用一审程序模板。

---

## 一、通用规则（适用于全部四类型）

| 项目 | 调整内容 |
|------|---------|
| 前置程序 | 必须先经过执行异议程序，收到执行异议裁定后方可起诉 |
| 起诉期限 | 收到执行异议裁定之日起 **15 日内**（不可变期间） |
| 管辖 | **执行法院专属管辖**，不得约定其他法院 |
| 审理期间 | **原则上不停止执行**（案外人提供担保可申请中止） |

---

## 二、四类执行异议之诉的当事人地位

### 2.1 案外人执行异议之诉

- **原告**：案外人
- **被告**：申请执行人
- **共同被告/第三人**：被执行人反对案外人异议的 → 共同被告；不反对的 → 第三人

### 2.2 申请执行人执行异议之诉

- **原告**：申请执行人
- **被告**：案外人
- **共同被告/第三人**：被执行人反对申请执行人主张的 → 共同被告；不反对的 → 第三人

### 2.3 追加/变更被执行人异议之诉

- **被追加人不服裁定**：原告：被追加人 / 被告：申请执行人 / 第三人：被执行人
- **申请人不服驳回裁定**：原告：申请执行人 / 被告：被追加人 / 第三人：被执行人

### 2.4 执行分配方案异议之诉

- **原告**：对分配方案提出异议后被反对的债权人/被执行人
- **被告**：反对该异议的其他债权人/被执行人

---

## 三、诉讼请求（按类型调整）

### 3.1 案外人执行异议之诉

```
诉讼请求：
1. 请求判决不得执行【执行标的名称/描述】；
2. 请求确认原告对【执行标的】享有【所有权/物权期待权等】；
3. 请求判令被告承担本案诉讼费用。
```

### 3.2 申请执行人执行异议之诉

```
诉讼请求：
1. 请求判决准许执行【执行标的名称/描述】；
2. 请求判令被告承担本案诉讼费用。
```

### 3.3 追加/变更被执行人异议之诉

**被追加人起诉：**
```
诉讼请求：
1. 请求判决不得追加原告为被执行人；
2. 请求判令被告承担本案诉讼费用。
```

**申请人起诉：**
```
诉讼请求：
1. 请求判决追加被告为被执行人，在【未出资/抽逃出资】范围内承担清偿责任；
2. 请求判令被告承担本案诉讼费用。
```

### 3.4 执行分配方案异议之诉

```
诉讼请求：
1. 请求撤销【执行法院】于【日期】作出的【分配方案名称】；
2. 请求按照【原告主张的分配方案】重新分配；
3. 请求判令被告承担本案诉讼费用。
```

---

## 四、事实与理由（通用要点）

1. **前置程序说明**：说明已提出执行异议，收到裁定，不服裁定提起诉讼
2. **执行标的描述**：准确描述执行标的（房产地址、车辆信息、股权比例、账户信息等）
3. **权利形成时间**：重点说明权利形成时间**早于查封时间**
4. **权利性质**：说明享有的权利类型（物权/物权期待权/租赁权/股权等）
5. **排除执行的依据**：引用法律依据和事实依据，说明权利足以排除执行

---

## 五、尾部

- 具状人：原告
- 日期：起诉日期
- 附注：副本份数（按被告人数 + 1 计算）
