# 03 执行异议之诉：实体权利的诉讼救济

> 🎯 **阶段定位**：执行审查程序中，对异议裁定不服时，通过诉讼程序解决实体权利争议。
>
> 执行异议之诉是执行程序中产生的**独立诉讼类型**，虽诉讼结构与一审类似，但在管辖、前置程序、起诉期限、当事人地位、诉讼请求、争议焦点等方面均有显著差异。各子类型配备独立完整的四阶段工作流，不再简单引用一审框架。

## 四种执行异议之诉速查

| 类型 | 原告 | 被告 | 核心争议 | 前置程序 |
|------|------|------|---------|---------|
| **案外人执行异议之诉** | 案外人 | 申请执行人 | 案外人对执行标的是否享有足以排除执行的实体权利 | 案外人异议被驳回 |
| **申请执行人执行异议之诉** | 申请执行人 | 案外人 | 案外人对执行标的是否享有足以排除执行的实体权利 | 案外人异议成立 |
| **追加/变更被执行人异议之诉** | 被追加人/申请人 | 对方 | 是否应当追加/变更被执行人 | 追加/变更裁定 |
| **执行分配方案异议之诉** | 异议债权人/被执行人 | 反对人 | 分配方案是否公平 | 对分配方案提出异议后被反对 |

## 使用方法

1. **确定类型**：根据案件情况确定属于四种类型中的哪一种
2. **进入对应子文件夹**：打开该类型的 README.md 和 SKILL.md
3. **执行独立工作流**：各子类型 SKILL.md 包含完整的四阶段工作流，直接按步骤执行
4. **调整关键参数**：按子类型 SKILL 调整当事人地位、管辖、诉讼请求、争议焦点

## 关键期限提醒

| 事项 | 期限 | 法律依据 |
|------|------|---------|
| 起诉 | 收到执行异议裁定之日起 **15日** | 《民事诉讼法解释》第 304 条 |
| 上诉 | 判决15日 / 裁定10日 | 民诉法第 171 条 |
| 执行异议审查 | 15日 | 《民事诉讼法解释》第 303 条 |

## 与执行程序的衔接

| 衔接点 | 说明 |
|--------|------|
| **前置程序** | 必须先经过执行异议程序，收到裁定后方可起诉 |
| **执行不停止** | 异议之诉审理期间，原则上不停止执行 |
| **判决效力** | 判决对执行程序有既判力，直接影响执行标的处置 |
| **程序回归** | 异议之诉判决后，根据结果回到执行程序继续或解除执行 |

## ❓ 常见问题

**Q1: 执行异议之诉和一审诉讼有什么区别？**
> 结构类似（均为一审诉讼），但关键差异：① 前置程序不同（必须先经执行异议）；② 管辖不同（执行法院专属管辖）；③ 起诉期限不同（15日不变期间）；④ 争议焦点不同（实体权利是否足以排除执行）。各子类型 SKILL.md 已据此设计独立工作流。

**Q2: 执行异议之诉的判决还能上诉吗？**
> 可以。执行异议之诉是一审程序，对判决不服可在15日内上诉，对裁定不服可在10日内上诉。例外：执行分配方案异议之诉一审即为终审。

**Q3: 执行异议之诉审理期间，执行会停止吗？**
> 原则上不停止执行。但案外人提供充分有效担保请求解除查封的，法院可以准许。

**Q4: 四种执行异议之诉能不能合并审理？**
> 不能合并。四种类型针对不同的实体争议，诉讼标的不同，应分别审理。

## 重要互斥规则

根据《民事诉讼法解释》第 301 条：

> **案外人执行异议之诉** 与 **第三人撤销之诉** 基于同一事实和理由的，**只能择一启动**。先启动的程序优先，后启动的不予受理。

AI在分析案件时，如检测到两种路径都可能适用，必须提醒用户注意此互斥规则，协助选择最优路径。
