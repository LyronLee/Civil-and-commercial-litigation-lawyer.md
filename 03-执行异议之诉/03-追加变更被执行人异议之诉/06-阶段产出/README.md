# 06-阶段产出

本文件夹用于存放追加/变更被执行人异议之诉阶段 AI 生成的工作成果文件。

## 产出文件清单

| 序号 | 文件名 | 说明 |
|------|--------|------|
| 01 | 策略方案书.md | 追加/变更异议之诉策略分析 |
| 02 | 起诉状.md | 追加/变更被执行人异议之诉起诉状 |
| 03 | 证据目录.md | 支持追加/变更异议的证据目录 |
| 04 | 开庭提纲.md | 庭审发言提纲（内部使用） |
| 05 | 庭审纪要.md | 庭审过程记录（内部使用） |
| 06 | 代理词.md | 庭后代理词 |
| 07 | 类案检索报告.md | 支持追加/变更事由的类案检索 |

## 使用方式

AI 根据 `SKILL.md` 工作流生成的文件自动保存于此。用户可直接使用或按需修改。

## 注意事项

- 所有文件生成后标注"初稿，需律师审核后使用"
- 起诉状必须在收到追加/变更裁定之日起 **15 日内** 提交
- 追加/变更被执行人异议之诉由**执行法院专属管辖**
- 核心审查：追加事由是否符合《变更追加当事人规定》第14-24条
- 开庭提纲、庭审纪要为内部使用文件，不提交法院
