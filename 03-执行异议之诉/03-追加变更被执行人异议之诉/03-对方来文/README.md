# 对方来文

本文件夹用于存放追加/变更被执行人异议之诉中，对方当事人提交的材料。

## 应放入的材料

| 材料名称 | 说明 |
|---------|------|
| **答辩状** | 对方提交的答辩状 |
| **反驳证据** | 对方提交的反驳证据（如出资凭证、财务审计报告等） |
| **代理词** | 对方的代理词（如有） |
| **补充意见** | 庭后补充的书面意见 |

## 使用方式

1. 收到对方材料后，及时扫描/复印并存入本文件夹
2. AI 在准备质证意见和庭审材料时会读取本文件夹内容
3. 如为被追加人，重点分析对方主张的追加事由和证据；如为申请执行人，重点分析对方的抗辩理由

## 注意事项

- 被追加人可能主张：已实际出资、出资期限未届满、财务独立、不应加速到期等
- 申请执行人可能主张：存在抽逃出资、人格混同、恶意转移财产等
