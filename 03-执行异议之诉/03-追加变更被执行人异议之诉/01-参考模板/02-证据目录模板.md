# 追加/变更被执行人异议之诉 · 证据目录

> 核心争议：追加/变更被执行人是否符合法定情形。

---

## 证据目录（被追加人角度）

| 序号 | 证据名称 | 证明目的 | 页码 |
|------|---------|---------|------|
| 1 | 追加/变更被执行人裁定书 | 前置程序已完成 | |
| 2 | 出资凭证/验资报告 | 已实际缴纳出资 | |
| 3 | 公司章程 | 出资期限尚未届满 | |
| 4 | 财务账册/审计报告 | 一人公司财产独立 | |
| 5 | 【其他抗辩证据】 | | |

## 证据目录（申请执行人角度）

| 序号 | 证据名称 | 证明目的 | 页码 |
|------|---------|---------|------|
| 1 | 驳回追加裁定书 | 前置程序已完成 | |
| 2 | 工商登记信息 | 股东身份、出资额、出资期限 | |
| 3 | 银行流水 | 抽逃出资的资金流向 | |
| 4 | 财务混同证据 | 人格混同、财产不独立 | |
| 5 | 终本裁定 | 被执行人财产不足以清偿 | |

---

> ⚠️ **关键提示**：
> - 追加事由严格限定于《变更追加当事人规定》第14-24条
> - 出资凭证、财务账册等关键证据必须详细列明
