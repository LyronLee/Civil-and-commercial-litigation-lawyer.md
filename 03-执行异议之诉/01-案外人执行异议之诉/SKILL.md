# 案外人执行异议之诉 SKILL

> ⚠️ 依据2023年修正《民事诉讼法》（2024年1月1日施行）及《最高人民法院关于人民法院办理执行异议和复议案件若干问题的规定》。

---

## 触发条件

用户表达以下意图时触发：
- "案外人异议被驳回了，要起诉"
- "我是案外人，对执行标的有权利"
- "准备案外人执行异议之诉"
- "查封了我的房子/车子，我想阻止执行"

---

## 前置依赖

触发本SKILL前必须确认：
1. 已提出案外人执行异议
2. 已收到执行法院**驳回异议**的书面裁定
3. 起诉期限未届满（收到裁定之日起**15日内**）
4. 用户身份为案外人（对执行标的主张实体权利的人）

> ⚠️ **15日起诉期限是不可变期间**，逾期不起诉，异议裁定即生效，丧失执行异议之诉的起诉权。

---

按以下路径读取案件材料：

| 材料类型 | 读取路径 | 用途 |

## Step 1：战略分析（阶段一）

### 1.1 前置程序确认

读取 `05-前置材料/` 中的执行异议裁定书，确认：
- 裁定结果是否为"驳回案外人异议"
- 裁定日期 → 计算起诉期限（收到之日起15日）
- 如期限≤3日，立即提醒用户优先完成起诉状

### 1.2 权利类型确认

根据案外人提供的材料，判断主张的权利类型：

| 权利类型 | 法律依据 | 核心要件 |
|---------|---------|---------|
| 不动产物权期待权 | 《执行异议复议规定》第28条 | 查封前签合同+合法占有+付大部分款+非因买受人原因未过户 |
| 消费者购房人权利 | 《执行异议复议规定》第29条 | 用于居住+名下无其他住房+已付超50%价款 |
| 担保物权 | 《执行异议复议规定》第30条 | 已办理抵押登记 |
| 租赁权 | 《民法典》第725条 | 查封前已签订合法租赁合同+已占有使用 |
| 股权/其他财产权 | 具体法律+司法解释 | 权利真实+合法取得+足以排除执行 |

### 1.3 时间轴构建

构建以下时间轴（必须清晰标注先后顺序）：

```
权利形成时间 → 查封时间 → 执行异议时间 → 起诉时间
```

> ⚠️ **关键判断**：权利形成时间必须**早于查封时间**，否则难以排除执行。

### 1.4 策略分析

1. **实体权利分析**：案外人的权利是否"足以排除强制执行"
2. **证据充分性**：现有证据能否证明权利形成时间早于查封时间
3. **法律适用检索**：检索对应的司法解释条款和类案
4. **风险评估**：胜诉概率评估（高/中/低）
5. **改判方案**：如胜诉，执行标的解除执行；如败诉，评估是否上诉

### 1.5 生成《案外人执行异议之诉策略方案书》

输出到 `06-阶段产出/01-策略方案书.md`，包含：
- 权利类型和法律依据
- 时间轴及关键节点
- 证据充分性分析
- 争议焦点预判
- 胜诉概率评估
- 后续路径建议

## Step 2：材料制备（阶段二）

### 2.1 起诉状

参考 `01-参考模板/00-起诉状模板.md`，起草起诉状：

- **原告**：案外人
- **被告**：申请执行人
- **共同被告/第三人**：被执行人（反对异议的→共同被告；不反对的→第三人）
- **管辖**：执行法院（专属管辖，不可协议变更）
- **诉讼请求**：
  1. 不得执行【执行标的描述】
  2. 确认案外人对【执行标的】享有【权利类型】

> ⚠️ **必须包含前置程序说明**：已提出案外人异议，执行法院于【日期】作出【案号】裁定，驳回案外人异议。

### 2.2 证据目录

参考 `01-参考模板/02-证据目录模板.md`（如有），整理证据：

| 证据类别 | 证明目的 | 优先级 |
|---------|---------|--------|
| 权属证明 | 案外人对执行标的享有实体权利 | ★★★ |
| 时间证据 | 权利形成时间早于查封时间 | ★★★ |
| 占有/使用证据 | 实际占有/使用执行标的 | ★★☆ |
| 付款凭证 | 已支付价款（如物权期待权） | ★★☆ |
| 执行文书 | 执行标的描述、查封时间 | ★★☆ |
| 异议裁定 | 前置程序已完成 | ★☆☆ |

### 2.3 类案检索（如需）

如以"适用法律错误"或需要类案支撑时，检索类似案例：
- 指导性案例：注明"应当参照"
- 其他类案：注明"可以参考"

## Step 3：庭审交锋（阶段三）

### 3.1 开庭提纲

参考 `01-参考模板/03-开庭提纲模板.md`（如有），生成开庭提纲：

1. **程序核对**：管辖（执行法院专属管辖）、当事人地位
2. **前置程序确认**：已提出异议并收到驳回裁定
3. **权利展示计划**：权利形成证据的出示顺序
4. **争议焦点预判**：
   - 权利是否真实
   - 权利形成时间是否早于查封
   - 权利是否"足以排除执行"
5. **法官发问预判**：
   - "权利来源是什么？"
   - "取得时间是什么时候？"
   - "为什么没过户？"
   - "与被执行人是什么关系？"
6. **对方攻击点应对**：
   - 恶意串通 → 提供独立第三方证据
   - 权利不足以排除 → 论证权利性质和司法解释适用

### 3.2 质证意见

针对对方可能提交的证据，准备质证意见：
- 攻击对方证据真实性（如主张权属证明虚假）
- 攻击对方证据关联性（如主张权利与执行标的无关）
- 质疑权利合法性（如主张权利取得不合法）

### 3.3 庭审纪要

开庭后24小时内生成庭审纪要，记录：
- 程序事项（回避、身份核对）
- 法官关注重点和倾向性表态
- 对方新观点和证据
- 法官口头要求（期限、补充材料等）

### 3.4 代理词

参考 `01-参考模板/04-代理词模板.md`（如有），提交书面代理词：
- 围绕"足以排除强制执行的民事权益"展开
- 引用原审判决原文+批驳（如相关）
- 新证据突出展示
- 类案引用

---

## Step 4：收官（阶段四）

### 4.1 判决分析

收到判决书后分析：
- **胜诉**：执行标的解除执行，回到执行程序处理其他财产
- **败诉**：执行继续进行，评估是否上诉

### 4.2 后续路径

| 结果 | 后续路径 | 期限 |
|------|---------|------|
| 胜诉 | 执行标的解除执行 | — |
| 败诉 | 上诉 | 判决15日/裁定10日 |
| 败诉+不上诉 | 评估其他救济路径 | — |

> ⚠️ 审理期间**原则上不停止执行**。但案外人提供充分有效担保请求解除查封的，法院可以准许。

## 质量检查清单

生成全部文件后，逐项核对：

- [ ] 前置程序是否已完成（已提出异议并收到驳回裁定）
- [ ] 起诉期限是否在收到裁定之日起15日内
- [ ] 管辖法院是否为执行法院（专属管辖）
- [ ] 当事人地位是否正确（原告：案外人，被告：申请执行人）
- [ ] 诉讼请求是否围绕"不得执行+确认权利"展开
- [ ] 起诉状是否包含前置程序说明
- [ ] 权利形成时间是否早于查封时间（时间轴清晰）
- [ ] 证据是否围绕实体权利展开（权属+时间+占有）
- [ ] 法条引用是否准确（《执行异议复议规定》第28-31条）
- [ ] 是否排除了互斥情形（未同时启动第三人撤销之诉，《民事诉讼法解释》第301条）
- [ ] 全部文件编码为UTF-8，中文显示正常
- [ ] 所有文件生成后标注"初稿，需律师审核后使用"

## 输出规范

- 所有文书按案外人执行异议之诉的独立结构生成
- 时间轴必须清晰标注权利形成时间与查封时间的先后关系
- 起诉状中必须说明前置程序（已提出异议并收到驳回裁定）
- 管辖法院必须是执行法院（专属管辖）
- 所有期限提醒以法院指定为准，法院未指定的适用法定期限
- 所有文件生成后标注"初稿，需律师审核"
