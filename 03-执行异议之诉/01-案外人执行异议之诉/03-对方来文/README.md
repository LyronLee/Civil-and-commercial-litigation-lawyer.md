# 对方来文

本文件夹用于存放案外人执行异议之诉中，对方当事人（申请执行人）提交的材料。

## 应放入的材料

| 材料名称 | 说明 |
|---------|------|
| **答辩状** | 申请执行人提交的答辩状 |
| **反驳证据** | 申请执行人提交的否定案外人权利的证据 |
| **代理词** | 对方的代理词（如有） |
| **补充意见** | 庭后补充的书面意见 |

## 使用方式

1. 收到对方材料后，及时扫描/复印并存入本文件夹
2. AI 在准备质证意见和庭审材料时会读取本文件夹内容
3. 重点分析对方攻击案外人权利真实性的论点

## 注意事项

- 对方可能主张：案外人与被执行人恶意串通、权利不足以排除执行、权利取得时间不真实等
- 收到对方新证据后，应及时评估是否需要补充我方证据
