# 对方来文

本文件夹用于存放申请执行人执行异议之诉中，对方当事人（案外人）提交的材料。

## 应放入的材料

| 材料名称 | 说明 |
|---------|------|
| **答辩状** | 案外人提交的答辩状 |
| **权属证据** | 案外人提交的证明其享有实体权利的证据 |
| **代理词** | 对方的代理词（如有） |
| **补充意见** | 庭后补充的书面意见 |

## 使用方式

1. 收到对方材料后，及时扫描/复印并存入本文件夹
2. AI 在准备质证意见和庭审材料时会读取本文件夹内容
3. 重点分析案外人主张的权利类型、权利形成时间和证据充分性

## 注意事项

- 案外人可能主张：物权期待权、消费者购房人权利、担保物权、租赁权等
- 收到案外人新证据后，应及时评估是否需要补充反驳证据
