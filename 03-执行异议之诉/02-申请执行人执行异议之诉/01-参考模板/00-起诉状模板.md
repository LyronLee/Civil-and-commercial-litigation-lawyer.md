# 申请执行人执行异议之诉起诉状

```
原告（申请执行人）：【姓名/名称】
住所地：

被告（案外人）：【姓名/名称】
住所地：

第三人（被执行人）：【姓名/名称】
住所地：

诉讼请求：
1. 请求判决准予执行【执行标的的具体描述】；
2. 本案诉讼费用由被告承担。

事实与理由：
【执行法院】在执行（【执行案号】）案件中，被告提出案外人执行异议。【执行法院】于【日期】作出（【案号】）执行裁定书，裁定【中止执行/支持异议等】。

原告认为，被告对执行标的不享有足以排除强制执行的民事权益，具体理由如下：

一、被告对执行标的不享有合法权利
【阐述被告主张的权利不成立】

二、即使被告享有某种权利，也不足以排除强制执行
【论证被告权利的效力不足以对抗执行】

三、执行标的属于被执行人责任财产
【论证执行标的应继续执行】

综上所述，被告对执行标的不享有足以排除强制执行的民事权益，恳请贵院依法支持原告的诉讼请求，准予继续执行。

此致
【管辖法院名称】

原告：【签章】
日期：
```

---

## 撰写要点

1. **起诉前提**：案外人执行异议成立后，申请执行人对裁定不服的，可提起申请执行人执行异议之诉
2. **核心论证**：案外人对执行标的不享有足以排除强制执行的民事权益
3. **注意与案外人异议之诉的对抗关系**：两诉互为原被告
