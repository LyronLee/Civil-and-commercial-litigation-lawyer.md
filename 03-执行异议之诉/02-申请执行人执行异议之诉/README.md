# 02 申请执行人执行异议之诉

> **案由定位**：申请执行人不服案外人异议成立的裁定，认为案外人对执行标的不享有足以排除执行的实体权利时提起的诉讼。
>
> **本质**：独立诉讼类型，配备独立完整的四阶段工作流。

---

## 当事人地位

| 角色 | 诉讼地位 | 说明 |
|------|---------|------|
| 申请执行人 | **原告** | 申请执行的债权人 |
| 案外人 | **被告** | 异议成立的案外人 |
| 被执行人 | **共同被告或第三人** | 反对申请执行人主张的 → 共同被告；不反对的 → 第三人 |

---

## 前置程序

必须先经过**案外人异议**程序：
1. 案外人向执行法院提出书面异议
2. 执行法院审查后裁定**中止执行**（案外人异议成立）
3. 申请执行人不服 → 收到裁定之日起 **15日** 内提起本诉

> ⚠️ **注意**：本诉的启动前提是案外人异议**成立**（法院裁定中止执行），而非被驳回。

---

## 管辖

**执行法院专属管辖**。

---

## 诉讼请求标准表述

```
诉讼请求：
1. 请求判决准许执行【执行标的名称/描述】；
2. 请求判令被告承担本案诉讼费用。
```

---

## 核心争议焦点

案外人对执行标的是否享有**足以排除强制执行的民事权益**。

申请执行人的核心任务：**否定案外人的实体权利**或**证明该权利不足以排除执行**。

常见攻击方向：

| 案外人主张的权利 | 申请执行人的攻击方向 |
|-----------------|---------------------|
| 所有权 | 权属虚假、恶意转移、未实际取得 |
| 物权期待权 | 未满足四要件（合同、价款、占有、未过户原因） |
| 租赁权 | 租赁虚假、租金未实际支付、占有不真实 |
| 股权 | 未登记、未实际出资、股权转让未生效 |

---

## 证据重点

1. **案外人权利瑕疵**：证明案外人主张的权利不真实或存在瑕疵
2. **查封时间**：证明查封时间早于案外人权利形成时间
3. **恶意串通证据**：如有证据证明案外人与被执行人恶意串通
4. **执行标的与债权的关联**：证明执行标的与被执行人责任的关联

---

## 与执行程序的衔接

- **审理期间**：原则上不停止执行（但案外人异议成立时已裁定中止）
- **胜诉后果**：恢复对执行标的的执行
- **败诉后果**：执行标的解除执行，需寻找其他财产线索
