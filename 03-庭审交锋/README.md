# 03-庭审交锋

> **定位**：收到对方诉讼材料后完成质证，开庭前完成庭审准备，庭审后整理纪要。
> **工作时机**：对方提交答辩状、证据目录及证据材料后；开庭前；庭审结束后。

## 阶段产出物说明

| 编号 | 文件名 | 用途 | 提交对象 |
|---|---|---|---|
| 01 | 质证意见.md | 对对方全部证据逐一进行真实性、合法性、关联性、证明力分析；同时强化我方观点，利用对方有利证据反击 | 提交法院 + 内部存档 |
| 02 | 开庭提纲.md | 开庭前准备的核心工作文件：法官可能的发问预判、我方发问清单、争议焦点梳理、辩论要点、突发应对预案 | 内部使用，**不提交法院** |
| 03 | 庭审纪要.md | 开庭后即时整理：程序事项记录、法庭调查要点、双方辩论意见摘要、法官要求及其他安排 | 内部使用，**不提交法院** |

## 关键设计原则

1. **卷宗索引，不重复搬运**：本阶段通过路径引用阶段二的完整卷宗，保持单份文件、避免冗余。
2. **质证即攻防**：质证意见不是单向的"挑对方毛病"，而是双向的"攻防转换"——对方证据有利时主动拿来强化我方观点。
3. **开庭提纲是内部作战地图**：包含敏感策略预判（如对方可能的抗辩路径、我方底线），仅限内部使用。
4. **庭审纪要及时闭环**：开庭后 24 小时内完成，记录法官口头要求（可能不在书面裁定中体现），防止遗漏。
