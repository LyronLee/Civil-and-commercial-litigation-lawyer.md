# 程序转换索引

> 核心原则：
> 1. **法院指定 > 法定期限**：如法院来文（举证通知书、传票、裁定书）中载明特定期限，以法院指定为准
> 2. **法定期限为基准**：如法院未指定或指定不明，适用本索引载明的法定期限
> 3. **关键期限到达前 3 日提醒用户**，超期可能导致失权
> 4. 每次转换时，优先读取 `角色判定规则.md` 确认新程序中的诉讼地位

## 一、一审结束后 → 下一步程序选择

```
收到一审判决/裁定书
    │
    ├─ 调解/和解成功（任何阶段）────────────┐
    │                                      │
    │  ├─ 调解书生效 → 具有强制执行力
    │  │    ├─ 对方履行 → 结案
    │  │    └─ 对方不履行 → 进入 04-执行程序/01-执行实施主线
    │  │         ⚠️ 申请执行期限：2 年
    │  │
    │  └─ 诉前/庭外和解协议（无强制执行力）
    │       ├─ 对方履行 → 结案
    │       └─ 对方不履行 → 就和解协议起诉 或 申请恢复执行（如已立案）
    │
    ├─ 胜诉 ─────────────────────────────┐
    │                                      │
    │  ├─ 对方在 15 日内（判决）/10 日内（裁定）上诉
    │  │    → 进入 02-二审程序/03-庭审准备（被上诉人模式）
    │  │    ⚠️ 期限：自判决书送达之日起 15 日；裁定书 10 日
    │  │
    │  └─ 对方未上诉，上诉期届满
    │       → 判决生效
    │       ├─ 对方在履行期内履行 → 结案
    │       └─ 对方不履行 → 进入 04-执行程序/01-执行实施主线
    │            ⚠️ 申请执行期限：2 年，自履行期最后一日起算
    │
    └─ 败诉/部分败诉 ──────────────────────┐
         │                                  │
         ├─ 当事人决定上诉
         │    → 进入 02-二审程序/02-材料制备（上诉人模式）
         │    ⚠️ 期限：判决书送达之日起 15 日；裁定书 10 日
         │    ⚠️ 关键动作：提交上诉状 + 预交上诉费
         │
         └─ 当事人不上诉
              → 上诉期届满，判决生效
              ├─ 本方在履行期内履行 → 结案
              └─ 本方不履行 → 对方可能申请执行
```

### 期限跟踪表模板（一审后）

| 期限事项 | 法定/指定日期 | 届满日期 | 起算点 | 状态 | 备注 |
|---------|-------------|---------|--------|------|------|
| 上诉期 | 15 日（判决）/10 日（裁定） | [计算] | 送达之日 | □未届满 □已届满 | 以法院送达回证为准 |
| 履行期 | [判决确定] | [计算] | 判决生效之日 | □未届满 □已届满 | 以判决主文为准 |
| 申请执行期 | 2 年 | [计算] | 履行期最后一日起 | □未起算 □已起算 | 超期丧失执行权 |

## 二、二审/再审结束后 → 下一步程序选择

### 2.1 二审结束后路径

```
收到二审判决/裁定（终审生效）
    │
    ├─ 调解/和解成功
    │    → 调解书生效（终审，不可上诉）
    │    ├─ 对方履行 → 结案
    │    └─ 对方不履行 → 进入 04-执行程序/01-执行实施主线
    │
    ├─ 二审改判/发回重审
    │    → 根据改判内容执行或重审
    │
    └─ 二审维持原判或改判后仍不服
         → 判决生效 → 执行或履行
         → 如仍不服，进入再审申请阶段（见2.2）
```

### 2.2 再审申请阶段路径（★ 新增）

```
收到生效判决/裁定/调解书（一审未上诉或二审终审）
    │
    ├─ 判决/裁定生效之日起 6 个月内
    │    │
    │    ├─ 符合《民事诉讼法》第211条13种情形之一
    │    │    → 向上一级人民法院申请再审
    │    │    │    ⚠️ 例外：当事人一方人数众多或双方为公民的，也可向原审法院申请
    │    │    │    ⚠️ 再审申请不停止判决、裁定的执行
    │    │    │
    │    │    ├─ 法院裁定再审（提审/指令再审/指定再审）
    │    │    │    → 进入 03-再审程序/02-再审审理
    │    │    │
    │    │    └─ 法院裁定驳回再审申请
    │    │         → 进入检察监督路径（见2.3）
    │    │         ⚠️ 期限：收到驳回裁定之日起 6 个月内
    │    │
    │    └─ 不符合再审事由
    │         → 客观告知成功率极低
    │         → 如坚持申请，可引导准备充分的证据和论证
    │
    ├─ 判决/裁定生效已超过 6 个月
    │    │
    │    ├─ 依据第211条第1、3、12、13项（新证据/伪造证据/法律文书被撤销/枉法裁判）
    │    │    → 自知道或应当知道之日起 6 个月内仍可申请
    │    │
    │    └─ 不符合上述4项特殊情形
    │         → 已过再审申请期限（除斥期间，不可延长）
    │         → 进入检察监督路径（见2.3）
    │         ⚠️ 期限：无固定期限，但越早越好
    │
    └─ 调解书申请再审
         → 提出证据证明调解违反自愿原则或调解协议内容违反法律
         → 期限：调解书生效后 6 个月内
```

### 2.3 检察监督路径（再审申请被驳回或超期后）

```
再审申请被驳回 或 超过再审申请期限
    │
    → 向作出生效裁判的法院的同级人民检察院申请监督
    │    ⚠️ 期限：收到驳回再审申请裁定之日起 6 个月
    │    ⚠️ 检察监督申请不停止判决、裁定的执行
    │
    ├─ 检察院审查（3个月内，可延长）
    │    │
    │    ├─ 提请抗诉（上级检察院向同级法院提出）
    │    │    → 法院应当再审
    │    │
    │    ├─ 发出再审检察建议（向同级法院提出）
    │    │    → 法院收到后3个月内审查决定是否再审
    │    │
    │    └─ 不支持监督申请
    │         → 作出《不支持监督申请决定书》
    │         → 可向上一级检察院申请复查（1年内）
    │         ⚠️ 复查期间不停止原决定的执行
    │
    └─ 当事人也可直接向上一级检察院申诉
```

### 2.4 再审审理结束后路径

```
收到再审裁判
    │
    ├─ 再审中达成调解
    │    → 调解书生效（终审，不可上诉）
    │
    ├─ 再审改判
    │    → 执行回转或重新执行
    │    ⚠️ 执行回转期限：无明确期限，但应及时申请
    │
    ├─ 再审发回重审
    │    → 原审法院重新按一审或二审程序审理
    │
    └─ 再审维持原判
         → 判决生效（如按二审程序再审，为终审裁判；如按一审程序再审，可上诉）
         → 检察监督（抗诉/检察建议）
         ⚠️ 期限：收到驳回再审申请裁定之日起 6 个月（如系再审被驳回）
```

## 三、执行程序中的救济路径选择（★ 最关键）

### 3.1 总览图

```
执行程序启动
    │
    ├─ 对执行行为不服（拍卖违法、查封超标的等）
    │    → 执行行为异议 → 裁定
    │         │
    │         └─ 不服裁定 → 10 日内向上一级法院申请复议
    │              ⚠️ 复议期间不停止执行
    │
    ├─ 案外人对执行标的主张权利
    │    → 案外人执行标的异议 → 裁定
    │         │
    │         ├─ 认为原判决/裁定错误（如判决确认房产归被执行人）
    │         │    → 案外人申请再审（审判监督程序）
    │         │    ⚠️ 期限：6 个月内，向原审法院申请
    │         │    ⚠️ 关键限制：启动执行异议后，不得再提第三人撤销之诉
    │         │
    │         └─ 与原判决/裁定无关（如法院误查封案外人财产）
    │              → 15 日内提起案外人执行异议之诉
    │              ⚠️ 期限：收到裁定之日起 15 日
    │              ⚠️ 专属管辖：执行法院
    │
    ├─ 申请追加/变更被执行人
    │    → 向执行法院提出申请 → 裁定
    │         │
    │         ├─ 一般情形（债权转让变更申请执行人等）
    │         │    → 10 日内向上一级法院复议
    │         │
    │         └─ 6 种特定情形（股东出资瑕疵、一人公司财产混同、清算注销等）
    │              → 15 日内提起追加/变更被执行人异议之诉
    │              ⚠️ 期限：收到裁定之日起 15 日
    │              ⚠️ 专属管辖：执行法院
    │
    ├─ 执行和解协议不履行
    │    → 申请执行人二选一（不可并行，一旦选定不可变更）：
    │         │
    │         ├─ 选项 A：申请恢复执行原生效法律文书
    │         │    ⚠️ 申请期限：自和解协议约定履行期最后一日起 2 年
    │         │    ⚠️ 恢复执行后，不得再就和解协议起诉
    │         │
    │         └─ 选项 B：就履行执行和解协议向执行法院提起诉讼
    │              ⚠️ 诉讼期间，执行中的查冻扣措施自动转为诉讼保全
    │              ⚠️ 如和解协议被确认无效/撤销，可再申请恢复执行
    │
    ├─ 多个债权人参与分配
    │    → 对分配方案提出异议
    │         │
    │         ├─ 程序异议（送达、资格确认等）
    │         │    → 执行行为异议 → 复议
    │         │
    │         └─ 实体异议（债权数额、分配顺位）
    │              → 15 日内提起执行分配方案异议之诉
    │              ⚠️ 前置程序：提出异议 + 收到反对意见
    │
    └─ 发现被执行人资不抵债
         → 执行转破产：向执行法院申请移送破产审查
         ⚠️ 需被执行人或申请执行人之一同意移送
         ⚠️ 破产审查期间，执行程序中止
```

### 3.2 关键期限表

| 程序节点 | 法定期限 | 起算点 | 超期后果 | 备注 |
|---------|---------|--------|---------|------|
| 执行行为异议后复议 | 10 日 | 裁定送达之日 | 丧失复议权 | 向上一级法院 |
| 案外人执行异议后起诉 | 15 日 | 裁定送达之日 | 丧失起诉权 | 与原判无关时 |
| 案外人申请再审 | 6 个月 | 裁定送达之日/知道权益受损之日 | 丧失再审申请权 | 认为原判错误时 |
| 追加/变更被执行人不服起诉 | 15 日 | 裁定送达之日 | 丧失起诉权 | 仅限 6 种情形 |
| 追加/变更被执行人不服复议 | 10 日 | 裁定送达之日 | 丧失复议权 | 一般情形 |
| 执行和解后恢复执行 | 2 年 | 和解协议约定履行期最后一日 | 丧失申请执行权 | 适用申请执行时效 |
| 分配方案异议后起诉 | 15 日 | 收到反对意见之日 | 丧失起诉权 | 需先提出异议 |

### 3.3 案外人救济路径互斥规则（★ 极易出错）

> 核心规则：**按启动程序先后，择一而终，不可变更**

| 启动顺序 | 后续路径 | 禁止路径 |
|---------|---------|---------|
| 先启动**执行异议**（对裁定不服认为原判错误） | → 只能**申请再审** | ❌ 不得提起第三人撤销之诉 |
| 先启动**第三人撤销之诉** | → 只能继续**三撤** | ❌ 即使执行中提异议，也不得申请再审 |
| 先启动**执行异议**（对裁定不服，与原判无关） | → 只能提起**执行异议之诉** | — |

**实务提醒**：如案外人同时符合申请再审和第三人撤销之诉条件，建议优先评估哪种程序胜诉概率更高、效率更快，再决定启动顺序。一旦启动执行异议，即被锁定在"再审"路径，丧失三撤选择权。

### 3.4 案外人救济路径详解

#### 路径A：案外人申请再审

```
案外人对执行标的主张权利，且认为原判决/裁定错误
    │
    → 向执行法院提出书面异议（执行标的异议）
         │
         ├─ 法院裁定中止执行
         │    → 案外人无需继续救济
         │
         └─ 法院裁定驳回异议
              │
              ├─ 案外人认为原判决/裁定错误
              │    → 自裁定送达之日起6个月内，向作出原判决/裁定的法院申请再审
              │    ⚠️ 启动后不得再提起第三人撤销之诉
              │
              └─ 案外人认为与原判决/裁定无关
                   → 自裁定送达之日起15日内，向执行法院提起执行异议之诉
```

**适用条件**：
- 案外人对原判决/裁定确定的执行标的物主张权利
- 无法提起新的诉讼解决争议
- 执行异议被驳回

**期限**：执行异议裁定送达之日起 **6个月**

**管辖法院**：作出原判决/裁定的法院

---

#### 路径B：第三人撤销之诉

```
案外人未参加原审，认为原判决/裁定/调解书错误损害其权益
    │
    → 自知道或应当知道权益受损之日起6个月内，向作出生效裁判的法院提起
         │
         ├─ 法院受理并审理
         │    → 判决撤销/变更原裁判，或驳回诉讼请求
         │
         └─ 执行中案外人提出执行异议
              ⚠️ 已被锁定在"三撤"路径，不得申请再审
```

**适用条件**：
- 案外人因不能归责于本人的事由未参加原审
- 有证据证明原判决/裁定/调解书错误
- 错误内容损害案外人民事权益

**期限**：知道或应当知道权益受损之日起 **6个月**

**管辖法院**：作出生效裁判的法院

---

#### 路径C：案外人执行异议之诉

```
案外人对执行标的主张权利，但认为与原判决/裁定无关
    │
    → 向执行法院提出书面异议（执行标的异议）
         │
         └─ 法院裁定驳回异议
              → 自裁定送达之日起15日内，向执行法院提起执行异议之诉
                   │
                   ├─ 法院判决不得执行该标的
                   │    → 执行程序排除该标的
                   │
                   └─ 法院判决驳回诉讼请求
                        → 执行继续
```

**适用条件**：
- 案外人对执行标的主张实体权利
- 执行标的与原判决/裁定无关（如法院误查封案外人财产）

**期限**：执行异议裁定送达之日起 **15日**

**管辖法院**：执行法院

---

### 3.5 案外人救济路径选择决策树

```
案外人权益受损
    │
    ├─ 是否已参加原审？
    │    ├─ 是 → 一般救济路径（上诉/再审）
    │    └─ 否 → 继续判断
    │
    ├─ 是否执行中？
    │    ├─ 否 → 第三人撤销之诉（6个月）
    │    └─ 是 → 继续判断
    │
    ├─ 执行标的是否与原判决/裁定有关？
    │    │
    │    ├─ 是（如判决确认房产归被执行人）
    │    │    → 执行异议 → 被驳回 → 案外人申请再审（6个月）
    │    │    ⚠️ 注意：一旦启动执行异议，不得再提三撤
    │    │
    │    └─ 否（如法院误查封案外人财产）
    │         → 执行异议 → 被驳回 → 执行异议之诉（15日）
    │
    └─ 案外人是否已提三撤？
         ├─ 是 → 继续三撤，执行异议仅用于中止执行，不得申请再审
         └─ 否 → 如上判断
```

### 3.6 案外人救济期限对照表

| 救济路径 | 法定期限 | 起算点 | 超期后果 | 管辖法院 |
|---------|---------|--------|---------|---------|
| 案外人申请再审 | 6个月 | 执行异议裁定送达之日 | 丧失再审申请权 | 原审法院 |
| 第三人撤销之诉 | 6个月 | 知道或应当知道权益受损之日 | 丧失起诉权 | 生效裁判法院 |
| 案外人执行异议之诉 | 15日 | 执行异议裁定送达之日 | 丧失起诉权 | 执行法院 |

> ⚠️ **重要**：案外人申请再审和第三人撤销之诉的期限均为6个月，但起算点不同。如案外人先得知权益受损（可提三撤），后执行异议被驳回（可申请再审），应仔细计算两个期限，避免超期。

## 四、执行审查程序 → 执行异议之诉的衔接

```
执行审查裁定（执行行为异议/标的异议/追加异议）
    │
    ├─ 复议程序（执行行为异议、一般追加异议）
    │    → 上一级法院复议裁定
    │         │
    │         ├─ 维持 → 复议结束，执行继续
    │         └─ 撤销/变更 → 执行法院重新审查或改正
    │
    └─ 诉讼程序（标的异议、特定追加异议、分配方案异议）
         → 执行法院一审
         │
         ├─ 不服一审判决 → 上诉（15 日）
         │    → 二审判决为终审
         │
         └─ 判决生效 → 按判决结果执行或排除执行
```

## 五、执行衍生程序与特殊救济

### 5.1 执行和解协议之诉

| 触发条件 | 路径选择 | 期限 | 后果 |
|---------|---------|------|------|
| 被执行人不履行和解协议 | A：恢复执行原生效文书 | 和解协议履行期最后日起 2 年 | 恢复后不得就和解协议起诉 |
| | B：就和解协议起诉 | 诉讼时效 3 年 | 诉讼中查冻扣自动转保全 |
| 和解协议履行完毕，但迟延/瑕疵 | → 另行起诉索赔 | 诉讼时效 3 年 | 不得恢复执行 |
| 和解协议无效/可撤销 | → 起诉确认无效/撤销 | 诉讼时效/除斥期间 | 无效/撤销后可恢复执行 |

### 5.2 执行转破产

| 触发条件 | 路径 | 关键节点 |
|---------|------|---------|
| 被执行人资不抵债，具备破产原因 | → 向执行法院申请移送破产审查 | 需被执行人或申请执行人之一同意 |
| 破产审查受理 | → 裁定受理破产 → 执行程序中止 | 全部财产纳入破产财产 |
| 破产审查驳回 | → 恢复执行 | 继续原执行程序 |

### 5.3 执行交叉救济（第三人撤销之诉与案外人再审）

| 场景 | 适用程序 | 前置条件 | 期限 |
|------|---------|---------|------|
| 案外人未参加原审，原判错误损害其权益 | 第三人撤销之诉 | 无需前置 | 知道或应当知道之日起 6 个月 |
| 案外人已提执行异议，对裁定不服，认为原判错误 | 案外人申请再审 | 执行异议被驳回 | 裁定送达之日起 6 个月 |
| 案外人未参加原审，执行标的与原判无关 | 案外人执行异议之诉 | 执行异议被驳回 | 裁定送达之日起 15 日 |

## 六、期限提取与提醒机制

### 6.1 提取优先级

```
第一顺位：法院来文明确指定的期限
  → 举证通知书载明的举证期限
  → 传票载明的开庭日期
  → 应诉通知书载明的答辩期限
  → 判决书/裁定书载明的上诉期（如法院指定）

第二顺位：法定期限
  → 本索引载明的法定期限
  → 适用于法院未指定或指定不明的情形

冲突处理：法院指定与法定不一致的，以法院指定为准，但应在文书中注明依据
```

### 6.2 必须提取的期限节点清单

| 阶段 | 期限来源 | 期限事项 | 法定基准 |
|------|---------|---------|---------|
| 一审 | 应诉通知书 | 答辩期 | 15 日 |
| 一审 | 举证通知书 | 举证期限 | 法院指定（通常 ≥30 日） |
| 一审 | 传票 | 开庭日期 | 法院指定 |
| 一审 | 判决书 | 上诉期 | 判决 15 日 / 裁定 10 日 |
| 二审 | 上诉状提交 | 上诉费预交 | 上诉期内或法院指定期限 |
| 执行 | 执行通知书 | 履行期限 | 法院指定 |
| 执行 | 执行异议裁定 | 复议/起诉期 | 复议 10 日 / 起诉 15 日 |
| 执行 | 执行和解协议 | 履行期限 | 协议约定 |
| 再审申请 | 裁判生效 | 再审申请期限 | 6 个月 | 除斥期间，不可延长 |
| 再审申请（特殊4项） | 知道或应当知道 | 再审申请期限 | 6 个月 | 第211条第1、3、12、13项 |
| 检察监督 | 驳回再审裁定送达 | 监督申请期限 | 6 个月 | 向同级检察院申请 |
| 检察监督复查 | 不支持决定送达 | 复查申请期限 | 1 年 | 向上一级检察院申请 |
| 再审审理 | 立案 | 审结期限 | 3 个月 | 特殊情况可延长 |

### 6.3 提醒规则

- **关键期限到达前 3 日**：向用户发出提醒
- **关键期限到达前 1 日**：再次紧急提醒
- **超期当日**：评估补救可能性（如申请顺延、恢复等），并告知用户法律后果
- **所有期限**：写入《关键期限跟踪表》，随阶段产出文件同步更新
