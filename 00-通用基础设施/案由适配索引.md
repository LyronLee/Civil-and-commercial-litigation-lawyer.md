# 案由适配索引

> 使用规则：
> 1. 阶段一（战略把脉）识别案由后，如为特殊案由，调用本索引对应条目
> 2. 按索引要求调整材料范围、诉讼地位、举证重点、诉讼请求
> 3. 阶段二（诉讼材料）使用对应案由的专用模板（位于 `01-一审程序/02-诉讼材料及来文管理/03-参考模板/特殊案由适配/`）
> 4. 每个案由条目末尾附【专属质量检查项】，完成通用清单后继续勾选

## 一、债权人代位权纠纷

### 法律依据
- 《民法典》第 535-537 条
- 《民法典合同编通则解释》第 33-41 条

### 阶段一材料差异
除原债权债务关系材料外，必须增加：
1. **债务人与次债务人的债权债务关系证明**（合同、发票、结算单、对账函等）
2. **债务人怠于行使债权的证据**：
   - ⚠️ 关键取证：起诉前**书面催告**债务人限期向次债务人主张权利，保留催告凭证（EMS 回执、微信记录等）
   - 债务人未在合理期限内起诉或申请仲裁的证据
3. **次债务人偿付能力初步信息**（工商登记、涉诉信息、财产线索）
4. **排除专属债权**：确认债务人对次债务人的债权非人身专属性（非抚养费、劳动报酬、人身损害赔偿等）

### 阶段二关键差异

| 项目 | 特殊要求 |
|------|---------|
| **被告** | 次债务人（债务人的债务人） |
| **第三人** | 债务人（无独立请求权第三人，法院可追加） |
| **管辖** | 被告住所地（次债务人所在地），不受债务人与次债务人之间管辖条款/仲裁条款约束 |
| **诉讼请求** | 必须明确"代位范围以债权人对债务人的到期债权为限"（就低不就高） |
| **诉讼费** | 由次债务人负担（如债权人胜诉），但律师费通常不支持由次债务人承担 |

### 阶段三特殊点
- **次债务人抗辩预判**：次债务人可同时主张"对债务人的抗辩"+"自己对债务人的抗辩"，需双重准备
- **债务人破产风险**：如债务人已进入破产程序，不得提起代位权诉讼，应转破产债权申报
- **次债务人破产风险**：如次债务人进入破产程序，代位权诉讼应中止，转破产债权申报

### 专属质量检查项
- [ ] 已书面催告债务人限期向次债务人主张权利，并保留催告凭证
- [ ] 次债务人的民事主体资格已确认（排除政府派出机构等特殊主体）
- [ ] 代位行使的债权范围未超过债权人对债务人的到期债权额
- [ ] 已评估次债务人的偿付能力及可能提出的抗辩（履行抗辩、抵销权等）
- [ ] 债务人未进入破产程序（破产程序中不得提起代位权诉讼）
- [ ] 债务人对次债务人的债权非人身专属性债权

## 二、债权人撤销权纠纷

### 法律依据
- 《民法典》第 538-542 条
- 《民法典合同编通则解释》第 42-46 条

### 阶段一材料差异
除原债权债务关系材料外，必须增加：
1. **债务人处分财产行为材料**：
   - 转让合同、赠与协议、股权转让协议、工商变更登记等
   - 放弃债权声明、债权债务抵销协议、担保设立文件等
2. **相对人/受益人身份信息**（用于恶意认定）
3. **债务人与相对人关联关系证据**（亲属关系、股东关系、实际控制人关系等）
4. **债务人的行为对债权造成损害的证据**：
   - 债务人行为前后的财产状况对比
   - 债务人已无其他财产清偿债务的证据
5. **除斥期间核查**：
   - 1 年：自债权人知道或应当知道撤销事由之日起
   - 5 年：自债务人行为发生之日起（最长保护期）

### 阶段二关键差异

| 项目 | 特殊要求 |
|------|---------|
| **被告** | 债务人和相对人为**共同被告**（民法典合同编通则解释第 44 条明确要求） |
| **管辖** | 债务人或相对人住所地（专属管辖除外），排除协议管辖和仲裁管辖 |
| **诉讼请求** | **必须同时写**：①撤销债务人的行为；②恢复原状/返还财产/折价补偿（第 46 条）。遗漏返还请求将导致另诉 |
| **律师费** | 合理的律师代理费、差旅费可认定为"必要费用"由债务人负担（第 45 条第 2 款） |
| **标的可分性** | 标的可分→在受影响债权范围内撤销；标的不可分→全部撤销（第 45 条） |

### 阶段三特殊点
- **恶意举证策略**：
  - 无偿行为：只需证明客观要件（减少责任财产、行为发生在债权成立后、有害债权）
  - 有偿行为：还需证明相对人**知道或应当知道**（恶意）。债务人与相对人存在亲属/关联关系的，法院更容易认定恶意
- **连环转让**：次相对人非善意的，可请求撤销连环转让链条上的全部处分行为（第 46 条）
- **与其他程序的衔接**：如债务人与相对人的行为已被生效司法文书确认，撤销权诉讼可能败诉，此时应考虑第三人撤销之诉

### 专属质量检查项
- [ ] 除斥期间已核查（1 年/5 年），确认未超过
- [ ] 诉讼请求同时包含"撤销"和"恢复原状/返还财产/折价补偿"
- [ ] 如为有偿行为，已收集相对人恶意的证据（关联关系、知情证据等）
- [ ] 债务人的行为已被生效文书确认的，已评估第三人撤销之诉的可行性
- [ ] 合理的律师费、差旅费已列入必要费用请求
- [ ] 如请求一并审理原债权债务关系，已确认受诉法院有管辖权

## 三、损害公司债权人利益责任纠纷（人格否认）

### 法律依据
- 《公司法》（2023 修订）第 23 条
- 《九民纪要》第 10-13 条
- 2025 年《民事案件案由规定》：统一三级案由"损害公司债权人利益责任纠纷"，不再细分股东/实际控制人

### 阶段一材料差异
除原债权债务关系材料外，必须增加：
1. **公司人格混同证据**（五维混同）：
   - **人员混同**：法定代表人、财务人员、高管交叉任职，共用劳动合同
   - **业务混同**：经营范围重合、业务资源无偿转移、共用客户/供应商
   - **财务混同**：资金往来无合理对价、账簿不分、公司财产记载于股东名下
   - **组织机构混同**：共用办公地址、联系电话、印章
   - **财产边界消失**：资产随意调拨、无偿使用公司资金/财产
2. **控制权滥用证据**：
   - 股东/实际控制人未经决策程序转移重大资产
   - 关联交易无书面合同或价格不公允
   - 公司沦为股东逃避债务的工具
3. **因果关系证据**：公司偿债能力因股东行为而严重降低
4. **一人公司特殊证据**：如为一人公司，举证责任倒置（股东自证财产独立）

### 阶段二关键差异

| 项目 | 特殊要求 |
|------|---------|
| **被告范围** | 股东、实际控制人、董监高（根据实际责任主体确定） |
| **诉讼地位（三种情形）** | ①债权已确认→股东被告/公司第三人；②一并提起→公司和股东共同被告；③未经确认直接诉股东→法院释明追加公司为共同被告，拒绝则驳回 |
| **管辖** | 侵权行为地（含原告住所地/侵权结果发生地）或被告住所地。**不适用**公司诉讼特殊管辖 |
| **举证责任** | 一般：债权人举证人格混同；一人公司：股东自证财产独立 |
| **既判力限制** | 人格否认仅个案适用，不当然及于其他诉讼 |

### 阶段三特殊点
- **横向人格否认（关联公司）**：如控制股东滥用多个关联公司，可主张各关联公司承担连带责任。需证明：①关联关系（债务形成时开始）；②人格混同；③严重损害债权人利益
- **逆向人格否认**：2024 年最高法法答网第九批精选答问已认可，实务中逐步成为主流
- **与执行程序的衔接**：执行程序中追加股东为被执行人的，仅部分情形可直接追加，其余需另行提起人格否认之诉

### 专属质量检查项
- [ ] 公司人格混同证据已覆盖五维（人员/业务/财务/组织机构/财产）
- [ ] 诉讼地位已按三种情形正确列置（特别注意：未经确认直接诉股东的，公司已列为共同被告）
- [ ] 如为一人公司，已准备/应对举证责任倒置
- [ ] 关联公司横向否认的，已证明关联关系自债务形成时即存在
- [ ] 已评估穷尽常规救济（代位权、撤销权、强制执行）后仍不足清偿，再启用人格否认
- [ ] 已向客户释明：人格否认判决仅个案有效，不当然及于其他债权人

## 四、股东出资纠纷（加速到期/追缴）

### 法律依据
- 《公司法》（2023 修订）第 47-54 条
- 《公司法解释三》第 13 条、第 18 条
- 《九民纪要》第 6 条（加速到期）

### 阶段一材料差异
除原债权债务关系材料外，必须增加：
1. **公司章程/股东会决议**：认缴出资额、出资方式、出资期限
2. **股东实际出资证据**：验资报告、银行流水、出资凭证、工商登记实缴信息
3. **公司资不抵债证据**：资产负债表、审计报告、终本裁定、破产受理裁定
4. **加速到期触发证据**（择一）：
   - 公司作为被执行人，人民法院穷尽执行措施无财产可供执行（终本裁定）
   - 公司债务产生后，股东会决议或以其他方式延长股东出资期限
   - 公司具备破产原因但不申请破产
5. **抽逃出资路径证据**（如主张抽逃）：资金流向、关联交易、虚假合同

### 阶段二关键差异

| 项目 | 特殊要求 |
|------|---------|
| **被告** | 股东（未出资/抽逃出资/转让瑕疵股权的原股东） |
| **第三人** | 建议将公司列为第三人或共同被告 |
| **管辖** | **存在争议**：公司住所地 vs 被告住所地。需提前确认受诉法院立场 |
| **案由选择** | 注意与"损害公司债权人利益责任纠纷"区分：出资纠纷重"出资义务违反"，人格否认重"滥用控制权" |
| **加速到期条件** | 必须证明公司"不能清偿到期债务"（通常以终本裁定为初步证据） |

### 阶段三特殊点
- **受让股东责任**：未出资即转让股权的，原股东和受让股东可能承担连带责任（公司法第 88 条）
- **出资期限利益**：一般情形下股东享有期限利益，仅在法定例外情形下可加速到期
- **与执行程序的衔接**：执行程序中可直接追加未出资股东为被执行人（变更追加当事人规定第 17-20 条），对追加裁定不服的，部分情形复议，部分情形异议之诉

### 专属质量检查项
- [ ] 公司章程记载的认缴出资情况已核实
- [ ] 股东实际出资情况已查明（银行流水、验资报告等）
- [ ] 加速到期的法定触发条件已满足（终本/延长出资期限/具备破产原因）
- [ ] 管辖法院立场已确认（公司住所地 vs 被告住所地）
- [ ] 如主张抽逃出资，资金流向和抽逃路径证据链完整
- [ ] 如股权转让，原股东和受让股东的责任划分已分析
- [ ] 已与"人格否认"案由完成区分，选择更匹配的案由

## 五、执行异议之诉（四类）

### 法律依据
- 《民事诉讼法》第 238 条
- 《执行异议复议规定》
- 法释〔2025〕10 号《最高人民法院关于审理执行异议之诉案件适用法律问题的解释》（2025 年 7 月 24 日施行）

### 通用规则（适用于全部四类）

#### 阶段一材料差异
除一般一审材料外，必须增加：
1. **前置执行异议裁定书**（必备：执行异议之诉以执行异议为前置程序）
2. **执行依据**：生效判决书/裁定书/调解书/仲裁裁决/公证债权文书
3. **执行标的的权属材料**：房产证、买卖合同、付款凭证、占有使用证据等

#### 阶段二关键差异

| 项目 | 特殊要求 |
|------|---------|
| **工作流特点** | 独立完整的四阶段工作流，管辖、前置程序、起诉期限等与一审诉讼有显著差异 |
| **专属管辖** | 执行法院管辖，不得约定其他法院 |
| **角色判定** | 复杂，调用 `角色判定规则.md` 第四章 |
| **起诉期限** | 收到执行异议裁定之日起 **15 日内** |

#### 阶段三特殊点
- 质证重点：权利形成时间、权利性质（物权 vs 债权）、权利顺位
- 注意与执行审查程序区分：执行异议之诉是**审判程序**，由审判庭审理；执行异议审查是**审查程序**，由执行裁决部门办理

#### 专属质量检查项（通用）
- [ ] 前置执行异议裁定已收到并已读取
- [ ] 起诉期限（15 日）已起算并跟踪
- [ ] 执行标的的权属证据材料完整
- [ ] 已确认管辖法院为执行法院

---

### 5.1 案外人执行异议之诉

**核心争议**：案外人对执行标的是否享有足以排除强制执行的实体权利

**诉讼地位**：
- 原告：案外人
- 被告：申请执行人
- 被执行人反对案外人异议的 → 共同被告；不反对的 → 第三人

**举证重点**：
- 案外人享有所有权/物权期待权/租赁权等实体权利
- 权利形成时间在查封之前
- 权利足以排除执行（如符合《执行异议复议规定》第 28-30 条）

---

### 5.2 申请执行人执行异议之诉

**核心争议**：申请执行人主张继续执行

**诉讼地位**：
- 原告：申请执行人
- 被告：案外人
- 被执行人反对申请执行人主张的 → 共同被告；不反对的 → 第三人

**举证重点**：
- 执行标的属于被执行人责任财产
- 案外人不享有足以排除执行的权利
- 或案外人的权利不能对抗申请执行人的执行权利

---

### 5.3 追加/变更被执行人异议之诉

**核心争议**：被追加主体是否应承担清偿责任

**适用情形**（仅限 6 种法定情形）：
1. 未缴纳或未足额缴纳出资的股东
2. 抽逃出资的股东
3. 原股东转让未出资股权
4. 一人公司股东财产混同
5. 未经清算即注销的股东/董事
6. 其他法定情形

**诉讼地位**：
- 被追加人不服裁定 → 原告：被追加人 / 被告：申请执行人 / 第三人：被执行人
- 申请人不服驳回裁定 → 原告：申请执行人 / 被告：被追加人 / 第三人：被执行人

**举证重点**：
- 股东出资情况（章程、验资报告、银行流水）
- 公司财产与股东财产是否独立（一人公司）
- 清算程序是否合法（清算责任）

---

### 5.4 执行分配方案异议之诉

**核心争议**：分配顺位、比例、数额

**诉讼地位**：
- 原告：对分配方案提出异议后被反对的债权人/被执行人
- 被告：反对该异议的其他债权人/被执行人

**举证重点**：
- 各债权的真实性、数额、优先权顺位
- 担保物权、建设工程价款优先权、税收优先权等
- 分配比例的计算依据
