# 02-法院来文

本文件夹用于存放二审收官阶段收到的法院文书。

## 应放入的材料

| 材料名称 | 说明 |
|---------|------|
| 二审判决书 | 二审法院作出的终审判决 |
| 二审裁定书 | 二审法院作出的裁定（维持/改判/发回/调解） |
| 调解书 | 二审中达成的调解协议经法院确认后的文书 |
| 送达回证 | 裁判文书送达证明 |

## 使用方式

1. 收到法院文书后，及时存入本文件夹
2. 确认裁判文书的生效时间
3. 唤起 AI 时，AI 会读取本文件夹内容并分析后续路径

## 注意事项

- 二审判决原则上为**终审判决**，送达即生效
- 收到判决书后，立即评估后续路径：执行/再审/结案
- 如申请再审，期限为判决生效之日起 **6 个月**
- 如申请执行，期限为判决生效后 **2 年**
