# 参考模板

本文件夹存放二审程序阶段四（收官）的参考模板，供 AI 生成阶段产出时参照使用。

## 模板清单

| 序号 | 模板名称 | 用途 |
|------|---------|------|
| 00-二审代理词模板.md | 二审开庭后、判决前提交的代理意见 | 基于庭审纪要，强化法律论证，回应争议焦点 |
| 01-再审申请书模板.md | 二审败诉后申请再审的文书 | 依据民诉法第211条，阐述法定再审事由 |
| 02-强制执行申请书模板.md | 二审胜诉后对方不履行时的执行申请 | 申请法院强制执行生效二审判决 |
| 03-结案复盘模板.md | 案件终局后的复盘总结 | 一审策略回顾、上诉决策回顾、二审结果分析、经验沉淀 |

## 使用方式

1. 本文件夹的模板为**参照模板**，AI 在执行 SKILL.md 工作流时会自动调用
2. 用户无需手动填写模板，将二审判决书放入 `02-法院来文/` 后，唤起 AI 即可
3. 如需了解模板格式，可直接打开查看

## 工作流分支

| 触发时机 | 生成分支 |
|---------|---------|
| 二审开庭后、判决前 | 分支A：代理词 |
| 收到二审判决后 | 分支B：裁判应对方案（执行/再审/发回重审/结案） |
| 决定申请再审 | 分支C：再审申请书 |
| 二审胜诉且对方不履行 | 分支D：强制执行申请书 |
| 案件终局后 | 分支F：结案报告 + 复盘总结 |

## 后续程序衔接

| 二审结果 | 后续程序 |
|---------|---------|
| 二审改判我方胜诉，对方不履行 | → **04-执行程序/01-执行实施主线** |
| 二审维持原判，不服 | → **03-再审程序/01-再审申请**（6个月内） |
| 二审发回重审 | → **01-一审程序** 重新审理 |
| 调解结案 | → 调解书生效 → 对方不履行 → **04-执行程序** |

## 产出位置

AI 根据本阶段 SKILL.md 生成的文件保存在 `03-阶段产出/` 中。
