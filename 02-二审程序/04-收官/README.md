# 04 收官：二审判决后的处理

> 🎯 **阶段定位**：收到二审判决书/裁定书后，根据裁判结果选择后续路径。二审是终审，判决生效后进入执行或申请再审程序。

> 本阶段的核心决策：**执行 / 申请再审 / 申诉** 三选一（视结果而定）。

## 🚀 使用流程

### 时点一：二审开庭后（如开庭）

庭审结束后，唤起AI：
> **"二审开完庭了，请帮我写代理词。"**

AI基于庭审情况生成代理词，提交法院。

### 时点二：收到二审判决后

将二审判决书放入 `02-法院来文/`，唤起AI：
> **"收到二审判决了，请帮我分析结果并推荐后续路径。"**

AI会：
1. 解析二审判决结果（维持/改判/发回重审/撤销）
2. 对比一审判决，分析变化
3. 根据结果推荐路径：
   - **维持原判**：评估是否申请再审
   - **改判**：确认改判内容，准备执行或应对对方再审
   - **发回重审**：准备重审（回到一审程序）

### 时点三：二审中达成调解

二审中法院组织调解的成功率较高。如达成调解：

**流程**：
1. 法院制作调解笔录
2. 双方签收调解书
3. 调解书生效后原判决视为撤销
4. 直接进入时点五（结案复盘）

> ⚠️ **注意**：二审调解书生效后即为终审，**不能上诉**。如对方不履行，直接进入 `04-执行程序/` 申请执行。

### 时点四：决定后续路径后

| 判决结果 | 推荐路径 | 对应产出 |
|---------|---------|---------|
| 我方胜诉，对方不履行 | 申请强制执行 | `03-强制执行申请书.md` → 进入 `04-执行程序/` |
| 我方败诉，有再审理由 | 申请再审 | `03-再审申请书.md` → 进入 `03-再审程序/` |
| 发回重审 | 准备重审 | 回到 `01-一审程序/` |
| 调解结案 | 监督履行 | `04-结案报告.md` |

### 时点四：案件终局后

案件彻底结束后，唤起AI：
> **"案件结束了，请帮我写结案报告和复盘总结。"**

## 再审 vs. 申诉

| 对比项 | 申请再审 | 申诉 |
|--------|---------|------|
| 受理机关 | 作出生效裁判的法院的上一级法院 | 检察院或人大信访 |
| 期限 | 判决生效后 **6 个月内** | 无明确期限 |
| 法定事由 | 民诉法第 211 条规定的 13 种情形 | 无严格限制 |
| 效果 | 法院裁定再审则中止原判执行 | 检察院抗诉才中止执行 |
| 次数限制 | 一般一次 | 无限制 |

> ⚠️ **再审是有限救济**：再审启动门槛高，需有明确的法定事由。AI会评估是否符合再审条件，给出客观建议。
