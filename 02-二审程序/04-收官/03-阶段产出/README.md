# 03-阶段产出

本文件夹用于存放二审收官阶段 AI 生成的工作成果文件。

## 产出文件清单

| 序号 | 文件名 | 说明 |
|------|--------|------|
| 01 | 代理词.md | 二审庭后代理词（如开庭后判决前） |
| 02 | 裁判应对方案.md | 根据二审结果选择后续路径 |
| 03 | 结案报告.md | 案件总结与复盘 |

## 使用方式

AI 根据 `SKILL.md` 工作流生成的文件自动保存于此。用户可直接使用或按需修改。

## 注意事项

- 所有文件生成后标注"初稿，需律师审核后使用"
- 如二审发回重审，裁判应对方案应引导至 `01-一审程序/` 重新审理
- 如二审维持原判且不服，裁判应对方案应评估再审可能性
- 结案报告应在案件终局后完成，沉淀经验教训
