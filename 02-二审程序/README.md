# 02 二审程序：不服一审判决的上诉救济

> 🎯 **程序定位**：当事人对一审判决/裁定不服时，向上一级法院提起上诉的救济程序。二审是终审程序（除特殊情况外），判决生效后具有既判力。
>
> **核心原则**：二审不是一审的简单重复，而是聚焦于"原审哪里错了"。

---

## 什么时候进入二审程序

| 情形 | 说明 |
|------|------|
| 收到一审判决，不服 | ✅ 进入二审程序（15日内上诉） |
| 收到一审裁定，不服 | ✅ 进入二审程序（10日内上诉） |
| 一审判决已生效且未上诉 | ❌ 上诉期已过，如需救济→03-再审 |
| 作为被上诉人（一审胜诉方） | ✅ 收到对方上诉状后进入，准备二审答辩状 |

## 与相邻程序的衔接

### 从一审进入

```
01-一审程序/04-收官与执行/
    │ 收到败诉/部分败诉判决
    ▼
02-二审程序/01-战略分析/
    │ 复制一审全部卷宗
    ▼
完成二审四阶段
```

### 二审结束后的路径选择

| 二审结果 | 后续程序 |
|---------|---------|
| 二审改判我方胜诉，对方不履行 | → **04-执行程序** |
| 二审维持原判，不服 | → **03-再审程序**（判决生效后6个月内） |
| 二审调解结案 | → 调解书生效 → 对方不履行 → **04-执行程序** |
| 二审发回重审 | → 回到 **01-一审程序** 重新审理 |
| 作为被上诉人胜诉 | → 结案复盘 |

## 关键期限（硬性 deadline）

| 事项 | 期限 | 起算日 |
|------|------|--------|
| **上诉期** | **判决15日 / 裁定10日** | 送达之日起 |
| 二审答辩期 | 15日 | 收到上诉状副本起 |
| 再审申请期 | 6个月 | 判决生效之日起 |

> ⏰ **上诉期是硬约束**：逾期不上诉，一审判决生效，丧失上诉权。收到判决书后必须**立即启动**二审程序。

## 快速开始

### 第一步：复制一审卷宗

将 `01-一审程序/` 四个阶段的全部产出复制到 `02-二审程序/01-战略分析/01-一审卷宗材料/`。

### 第二步：放入判决书

将一审判决书放入 `01-一审卷宗材料/`。

### 第三步：唤起 AI

> **"我在做02二审程序的01战略分析阶段，请帮我分析一审判决，评估上诉策略。一审卷宗在01-一审卷宗材料里。"**

## 二审 vs. 再审 vs. 一审

| 对比项 | 一审 | 二审 | 再审 |
|--------|------|------|------|
| 启动条件 | 起诉 | 上诉 | 符合法定再审事由 |
| 期限 | 诉讼时效（通常3年） | **15日/10日** | **6个月** |
| 审理范围 | 全部事实和法律 | 限于上诉请求 | 限于再审请求 |
| 改判原则 | — | 维持原判为原则 | 维持原判为原则 |
| 是否终审 | 否 | **是**（原则上） | 视审理程序而定 |
