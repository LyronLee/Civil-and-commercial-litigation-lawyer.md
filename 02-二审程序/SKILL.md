# 02 二审程序 SKILL

> 核心理念：二审不是一审的简单重复，而是聚焦于"原审哪里错了"。本程序配备独立完整的四阶段工作流，直接按本文件步骤执行。

---

## 触发条件

用户表达以下意图时触发：
- "我要上诉"
- "对一审判决不服"
- "收到对方上诉状了"
- "帮我写上诉状"
- "二审答辩状怎么写"
- "评估一下改判可能性"

---

## 前置依赖

触发本SKILL前必须确认：
1. 已有**一审判决书/裁定书**且已送达
2. **上诉期未届满**：判决15日/裁定10日（送达之日起算）
3. 明确我方角色：上诉人（不服一审判决）或被上诉人（对方上诉）

> ⚠️ **上诉期是硬约束**：逾期不上诉，一审判决生效，丧失上诉权。收到判决书后必须**立即启动**二审程序。

---

## Step 0：条件确认与角色判定

### 0.1 向用户确认

- 收到的是判决书还是裁定书？
- 送达日期？（计算上诉期）
- 我方是上诉人还是被上诉人？
- 不服一审的具体理由？（事实认定错误/法律适用错误/程序违法）

### 0.2 关键期限计算

```
判决书送达日期: ____年____月____日
+ 15日上诉期
= 上诉截止日期: ____年____月____日

或

裁定书送达日期: ____年____月____日
+ 10日上诉期
= 上诉截止日期: ____年____月____日
```

> 🔴 **如上诉期即将届满（≤3日）**：立即提醒用户尽快提交上诉状，可先提交后补理由。

### 0.3 一审卷宗检测

扫描项目目录中是否存在一审材料：
- 如存在 `01-一审程序/` 卷宗 → 读取并分析
- 如不存在 → 提示用户将一审判决书、证据材料、庭审笔录等放入 `01-战略分析/01-一审卷宗材料/`

---

## Step 1：阶段路由

```
收到一审判决/裁定
    │
    ├─ 上诉期已届满 ──────────────────→ 告知丧失上诉权，推荐 03-再审程序/
    │
    ├─ 决定不上诉 ────────────────────→ 结案复盘或 04-执行程序/
    │
    ├─ 决定上诉（上诉人）─────────────→ 01-战略分析/
    │    （解析判决、错误诊断、改判评估、上诉策略）
    │
    ├─ 收到上诉状（被上诉人）─────────→ 01-战略分析/
    │    （分析对方上诉理由、制定答辩策略）
    │
    ├─ 文书制备 ─────────────────────→ 02-材料制备/
    │    （上诉状/二审答辩状、新证据目录）
    │
    ├─ 庭审准备 ─────────────────────→ 03-庭审准备/
    │    （二审质证、开庭提纲、庭审纪要）
    │
    └─ 判决后处理 ───────────────────→ 04-收官/
         （裁判应对方案：再审/执行/调解/结案）
```

---

## Step 2：与相邻程序的衔接

### 2.1 从一审进入

```
01-一审程序/04-收官与执行/
    │ 收到败诉/部分败诉判决
    ▼
02-二审程序/01-战略分析/
    │ 复制一审全部卷宗
    ▼
完成二审四阶段
```

### 2.2 二审结束后的路径选择

| 二审结果 | 后续程序 |
|---------|---------|
| 二审改判我方胜诉，对方不履行 | → **04-执行程序** |
| 二审维持原判，不服 | → **03-再审程序**（判决生效后6个月内） |
| 二审调解结案 | → 调解书生效 → 对方不履行 → **04-执行程序** |
| 二审发回重审 | → 回到 **01-一审程序** 重新审理 |
| 作为被上诉人胜诉 | → 结案复盘 |

---

## 质量检查清单（二审程序通用版）

| 序号 | 检查项 | 合格标准 |
|------|--------|---------|
| 2.1 | 上诉期 | 判决15日/裁定10日，未届满 |
| 2.2 | 上诉请求 | 明确、具体，围绕一审错误展开 |
| 2.3 | 上诉理由 | 聚焦"原审哪里错了"（事实/法律/程序） |
| 2.4 | 新证据 | 如提出新证据，符合《民事诉讼法》第211条第（一）项 |
| 2.5 | 管辖法院 | 上一级法院 |
| 2.6 | 当事人地位 | 正确标注上诉人/被上诉人/原审地位 |
| 2.7 | 二审答辩 | 针对对方上诉理由逐条反驳 |
| 2.8 | 审理范围 | 限于上诉请求（《民事诉讼法》第175条） |
| 2.9 | 所有文件编码 | UTF-8，中文显示正常 |
| 2.10 | AI产出标注 | 所有提交法院的文书标注"初稿，需律师审核后使用" |

---

## 关键期限提醒

> 🔴 **上诉期**：判决 **15 日** / 裁定 **10 日**（送达之日起算）
>
> 🔴 **二审答辩期**：收到上诉状副本之日起 **15 日**
>
> 🔴 **再审申请期**：二审判决生效之日起 **6 个月**
>
> ⚠️ **上诉期不可延长**：逾期不上诉，一审判决生效，丧失上诉权。
