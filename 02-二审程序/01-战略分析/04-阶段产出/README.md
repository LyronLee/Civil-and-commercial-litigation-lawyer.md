# 阶段产出

本文件夹存放二审程序阶段一（战略分析）的 AI 生成工作成果。

## 预期产出文件

| 序号 | 文件名 | 内容说明 |
|------|--------|---------|
| 01-判决书分析意见.md | 判决书结构化解析 + 错误诊断清单 | 逐段拆解判决书，标注事实认定错误、法律适用错误、程序违法、裁判结果不当 |
| 02-上诉策略方案书.md | 上诉策略方案 | 上诉目标、核心理由（1-3个）、主攻方向、风险评估、时间规划、费用预算 |
| 03-新证据评估报告.md | 新证据可采性和证明力评估 | 如有新证据，评估其是否属于法定"新证据"、证明力、对上诉请求的支持作用 |
| 04-待补充材料清单.md | 上诉阶段缺失材料及获取方法 | 优先级、负责方、截止日期 |
| 05-关键期限跟踪表.md | 上诉期、缴费期、二审审理期限等 | 关键时间节点提醒 |

## 使用方式

1. 本文件夹文件由 AI 根据 `SKILL.md` 工作流自动生成
2. 生成后需由**执业律师审核**，确认错误诊断的准确性、上诉策略的可行性
3. 审核通过后，`01-判决书分析意见.md` 和 `02-上诉策略方案书.md` 可进入下一阶段（材料制备）使用

## 衔接下一阶段

阶段一完成后，将本文件夹全部文件复制到 `02-材料制备/01-上一阶段产出/`，继续阶段二工作。

## 注意事项

- 所有产出文件均标注"初稿，需律师审核后使用"
- 上诉期限很短（判决15日/裁定10日），如期限紧迫，可简化策略分析，优先完成上诉状
- 如未检测到已有上诉状且上诉期≤3日，进入紧急模式
