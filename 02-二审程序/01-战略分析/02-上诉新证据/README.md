# 上诉新证据

本文件夹用于存放拟在二审程序中提交的新证据材料。

## 什么是"新证据"

根据《民事诉讼法》司法解释，二审中的"新证据"包括：

1. **一审庭审结束后新发现的证据**
2. **一审中申请法院调查取证未获准许的证据**
3. **一审中因客观原因无法提供的证据**

## 应放入的材料

| 类型 | 说明 | 示例 |
|------|------|------|
| 新发现的书证 | 一审庭审结束后才发现或取得 | 新找到的合同、函件、账单 |
| 新调取的证据 | 一审中申请调取未获准许 | 银行流水、工商档案、不动产登记 |
| 新形成的证据 | 一审结束后新形成 | 后续履约记录、新的鉴定意见 |
| 新证人证言 | 一审中未能出庭的证人 | 证人书面证言或出庭申请 |

## 使用方式

1. 将新证据原件/复印件放入本文件夹
2. 在文件名中标注证据类型和形成时间
3. AI 将评估每份证据的**可采性**、**关联性**、**证明力**和**策略价值**

## 注意事项

- ⚠️ **非"新证据"的材料，二审法院原则上不予采纳**
- 每份新证据需准备"一审未提交的合理原因说明"
- 新证据应在举证期限内提交（法院指定，通常 ≥30日）
- 如没有新证据，本文件夹可留空
