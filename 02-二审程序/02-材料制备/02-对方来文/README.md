# 02-对方来文

本文件夹用于存放二审阶段对方当事人提交的各类文书。

## 应放入的材料

| 材料名称 | 说明 |
|---------|------|
| 对方上诉状 | 对方当事人提交的上诉状副本（如我方为被上诉人） |
| 对方答辩状 | 对方当事人提交的二审答辩状 |
| 对方新证据 | 对方在二审中提交的新证据材料 |
| 对方代理词 | 对方在二审庭审中提交的代理词 |

## 使用方式

1. 收到对方文书后，及时存入本文件夹
2. 在文件名中标注收到日期（如 `2026-04-25_对方上诉状.pdf`）
3. 唤起 AI 时，AI 会读取本文件夹内容并分析对方观点

## 注意事项

- 收到对方上诉状后，注意二审答辩期为 **15 日**
- 对方提交新证据的，应评估其三性（真实性、合法性、关联性）
- 所有来文按时间顺序排列，便于构建案件时间线
