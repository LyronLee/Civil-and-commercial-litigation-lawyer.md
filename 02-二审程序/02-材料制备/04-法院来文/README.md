# 04-法院来文

本文件夹用于存放二审法院送达的各类文书。

## 应放入的材料

| 材料名称 | 说明 |
|---------|------|
| 上诉受理通知书 | 二审法院受理上诉的通知 |
| 交纳诉讼费通知书 | 上诉费用缴纳通知 |
| 传票 | 二审开庭传票 |
| 合议庭组成通知书 | 告知合议庭组成人员 |
| 二审判决书/裁定书 | 二审裁判文书 |

## 使用方式

1. 收到法院文书后，及时存入本文件夹
2. 在文件名中标注收到日期
3. 唤起 AI 时，AI 会读取本文件夹内容并提取关键信息

## 注意事项

- 收到**传票**后，立即确认开庭时间、地点，安排出庭
- 收到**交纳诉讼费通知书**后，按期缴费，逾期按撤诉处理
- 收到**二审判决书/裁定书**后，评估是否申请再审（6个月内）
- 法院指定的期限优先于法定期限
