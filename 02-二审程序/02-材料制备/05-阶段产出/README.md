# 阶段产出

本文件夹存放二审程序阶段二（材料制备）的 AI 生成工作成果。

## 预期产出文件

### 上诉人模式

| 序号 | 文件名 | 内容说明 |
|------|--------|---------|
| 01-上诉状.md | 正式上诉文书 | 针对一审判决错误，请求二审改判或发回重审 |
| 02-新证据目录.md | 二审新证据编目 | 含新证据类型标注和一审未提交原因说明 |
| 03-上诉费计算表.md | 上诉案件受理费计算 | 按不服部分标的额分段计算 |
| 04-待补充材料清单.md | 阶段二视角的缺失信息 | 上诉状引用证据核对、原件准备、缴费安排 |
| 05-关键期限跟踪表.md | 期限汇总 | 答辩期、举证期、开庭日、缴费期等 |

### 被上诉人模式

| 序号 | 文件名 | 内容说明 |
|------|--------|---------|
| 01-二审答辩状.md | 正式答辩文书 | 逐条反驳对方上诉理由，维护一审判决 |
| 02-新证据目录.md | 反驳新证据编目（如有） | 针对对方新证据的反驳证据 |
| 03-待补充材料清单.md | 阶段二视角的缺失信息 | 答辩状引用证据核对、原件准备 |
| 04-关键期限跟踪表.md | 期限汇总 | 答辩期、举证期、开庭日等 |

## 使用方式

1. 本文件夹文件由 AI 根据 `SKILL.md` 工作流自动生成
2. 生成后需由**执业律师审核**，确认上诉请求具体明确、理由聚焦、格式规范
3. 审核通过后，上诉状/答辩状应及时提交二审法院

## 注意事项

- 所有产出文件均标注"初稿，需律师审核后使用"
- **上诉状默认不自动生成**：仅在未检测到已有上诉状且用户确认需要时生成
- 如检测到用户已在一审阶段完成上诉状，AI 会询问是否沿用/修改/重新起草
- 上诉请求必须具体（写明"撤销后怎么改"），不能只写"请求撤销原判"
- 部分上诉时，首部需明确仅对哪些判项不服
