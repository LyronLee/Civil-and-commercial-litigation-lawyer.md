# 02 材料制备：上诉状与二审诉讼材料

> 🎯 **阶段定位**：上诉策略已明确，本阶段核心任务是撰写**上诉状**和整理**二审诉讼材料**。
>
> 上诉状是二审程序的入口文书，直接决定二审法院的审查方向。一份好的上诉状不是"翻旧账"，而是**精准指出原审错误并提出明确的上诉请求**。

## 🚀 使用流程

### 第一步：复制上一阶段产出 → `01-上一阶段产出/`

将阶段一（`01-战略分析/04-阶段产出/`）的全部文件复制到本文件夹。

### 第二步：确认角色和材料

| 角色 | 需要放入的材料 |
|------|---------------|
| **上诉人** | 判决书分析意见 + 上诉策略方案书 |
| **被上诉人** | 对方上诉状（放入 `02-对方来文/`）+ 一审胜诉策略材料 |

### 第三步：唤AI → "帮我写上诉状/二审答辩状"

**上诉人：**
> **"我在做02二审程序的02材料制备阶段，请帮我起草上诉状。上一阶段产出在01-上一阶段产出文件夹里。"**

**被上诉人：**
> **"我在做02二审程序的02材料制备阶段，对方提交了上诉状在02-对方来文里，请帮我起草二审答辩状。"**

AI会：
1. 读取判决书分析意见和策略方案书（上诉人）/ 对方上诉状（被上诉人）
2. 起草上诉状/二审答辩状
3. 整理新证据目录（如有新证据）
4. 生成辅助文书（按需）

## ❓ 常见问题

**Q1: 上诉状和起诉状有什么区别？**
> - **起诉状**：面向一审法院，陈述事实、提出请求
> - **上诉状**：面向二审法院，**针对一审判决的错误**提出请求。上诉状不是重新讲一遍案情，而是指出原审哪里错了。

**Q2: 上诉请求怎么写？**
> 常见写法：
> - "请求撤销一审判决第X项，改判……"
> - "请求撤销一审判决，发回重审"
> - "请求变更一审判决第X项，改判……"
> AI会根据策略方案书中的目标自动生成。

**Q3: 被上诉人需要提交答辩状吗？**
> 建议提交。虽然二审被上诉人可以选择口头答辩，但书面答辩状能更系统地反驳对方上诉理由，影响二审法官心证。

**Q4: 二审还能提交新证据吗？**
> 可以，但受限。必须是"新证据"（一审后新发现、一审申请调取未获准许等）。一审能提交而未提交的，二审通常不采纳。
