# 上诉状

```
上诉人（原审原告/被告）：【姓名/名称】
住所地：
法定代表人/负责人：

被上诉人（原审被告/原告）：【姓名/名称】
住所地：
法定代表人/负责人：

（原审第三人：如有）

上诉人因与被上诉人【案由】纠纷一案，不服【一审法院名称】（【案号】）民事判决，现依法提起上诉。

上诉请求：
1. 请求撤销一审判决第【X】项，改判【具体内容】；
2. 请求撤销一审判决，发回【一审法院名称】重审；
3. 请求判令被上诉人承担本案一、二审诉讼费用。

上诉理由：

一、原审判决认定事实错误（或事实不清）
【具体指出原审认定错误的某个事实，引用证据反驳】

二、原审判决适用法律错误
【具体指出原审引用的法条错误或不完整，引用正确的法律依据】

三、原审违反法定程序
【如存在程序问题，具体说明】

综上所述，原审判决【错误之处】，恳请二审法院依法改判。

此致
【二审法院名称】

上诉人：【签章】
日期：

附：
1. 本上诉状副本【】份
2. 新证据材料【】份（如有）
```

---

## 撰写要点

1. **上诉请求必须具体**：不能只写"请求撤销原判"，必须写明撤销后怎么改
2. **聚焦重点**：选择1-3个最有力的上诉理由，不宜过多
3. **不重复一审论述**：重点在于"原审哪里错了"，不是"事实本来怎样"
4. **引用判决原文**：指出原审判决具体哪一页、哪一段有误
5. **篇幅控制**：3000-5000字为宜
