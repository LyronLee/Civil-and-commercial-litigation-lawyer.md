# 二审答辩状

```
答辩人（原审原告/被告）：【姓名/名称】
住所地：

上诉人（原审被告/原告）：【姓名/名称】
住所地：

答辩人因上诉人【姓名/名称】不服【一审法院】（【案号】）民事判决提起上诉一案，现答辩如下：

一、针对上诉请求一的答辩
【逐条反驳】

二、针对上诉理由的答辩
【分点反驳对方的上诉理由】

三、一审判决认定事实清楚、适用法律正确
【正面阐述一审判决的正确性】

综上所述，请求二审法院驳回上诉，维持原判。

此致
【二审法院名称】

答辩人：【签章】
日期：
```

---

## 撰写要点

1. **逐条对应反驳**：对方上诉状有几条理由，答辩状就逐条反驳
2. **维护一审判决**：强调一审判决认定事实清楚、适用法律正确
3. **可适当主动出击**：如对方上诉理由暴露了新漏洞，可主动指出
4. **不引入新争议**：以防守为主，避免节外生枝
