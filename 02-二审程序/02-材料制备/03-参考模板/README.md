# 参考模板

本文件夹存放二审程序阶段二（材料制备）的参考模板，供 AI 生成阶段产出时参照使用。

## 模板清单

| 序号 | 模板名称 | 用途 |
|------|---------|------|
| 00-上诉状模板.md | 上诉人向二审法院提交的上诉文书 | 针对一审判决错误，请求二审改判或发回重审 |
| 01-二审答辩状模板.md | 被上诉人针对对方上诉状的答辩文书 | 反驳对方上诉理由，维护一审判决 |
| 02-新证据目录模板.md | 二审新证据的编目表格 | 序号/名称/证明目的/页码/新证据类型/一审未提交原因 |
| 03-调查取证申请书模板.md | 申请二审法院调取证据 | 一审中未调取或新发现的证据 |

## 使用方式

1. 本文件夹的模板为**参照模板**，AI 在执行 SKILL.md 工作流时会自动调用
2. 用户无需手动填写模板，将阶段一产出放入 `01-上一阶段产出/` 后，唤起 AI 即可
3. 如需了解模板格式，可直接打开查看

## 角色分流

| 角色 | 主要产出 |
|------|---------|
| 上诉人 | 上诉状、新证据目录、上诉费计算表 |
| 被上诉人 | 二审答辩状、反驳新证据目录 |

## 产出位置

AI 根据本阶段 SKILL.md 生成的正式文件保存在 `05-阶段产出/` 中。
