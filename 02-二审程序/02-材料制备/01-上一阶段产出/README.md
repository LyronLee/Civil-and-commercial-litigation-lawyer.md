# 01-上一阶段产出

本文件夹用于存放前一阶段（01-战略分析）的工作成果文件，供本阶段读取和复用。

## 应放入的文件

将 `02-二审程序/01-战略分析/04-阶段产出/` 中的以下文件复制到本文件夹：

| 文件名 | 说明 |
|--------|------|
| 01-判决书分析报告.md | 一审判决错误诊断 |
| 02-上诉策略方案书.md | 上诉方案与风险评估 |
| 03-改判概率评估.md | 改判可能性分析 |

## 使用方式

1. 阶段一完成后，将产出文件复制到本文件夹
2. 唤起 AI 时，AI 会读取本文件夹中的前一阶段成果
3. 本阶段（材料制备）将基于阶段一的结论起草上诉状/二审答辩状

## 注意事项

- 保持文件完整，确保 AI 能读取全部内容
- 如阶段一有更新，及时同步到本文件夹
- 本文件夹为**只读引用**，不在此生成新文件
