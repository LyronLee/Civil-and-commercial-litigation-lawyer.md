# 03 庭审准备：二审庭审交锋

> 🎯 **阶段定位**：上诉状/答辩状已提交，等待二审开庭。本阶段核心任务是准备庭审质证和辩论。
>
> 二审庭审与一审不同：审理范围限于上诉请求，法官更关注"原审是否错了"，而不是"事实到底怎样"。

## 🚀 使用流程

### 收到对方二审材料后

将对方在二审中提交的新材料放入 `02-材料制备/02-对方来文/` 或单独文件夹（视具体材料类型）。

然后唤起AI：
> **"对方在二审中提交了新证据，帮我写质证意见。"**

### 收到二审开庭传票后

唤起AI：
> **"二审即将开庭，请帮我准备开庭提纲。"**

AI会：
1. 读取上诉状/答辩状和一审判决
2. 梳理二审争议焦点（不同于一审）
3. 预判二审法官可能的发问
4. 准备我方发问清单
5. 准备辩论要点

### 二审开庭后

庭审结束后 24 小时内，唤起AI：
> **"二审开完庭了，请帮我整理庭审纪要。"**

## 二审庭审的特殊性

| 对比项 | 一审 | 二审 |
|--------|------|------|
| 审理范围 | 全部事实和法律问题 | **限于上诉请求** |
| 开庭率 | 较高 | 可能不开庭（书面审理） |
| 法官关注 | 事实查明 | **原审是否错误** |
| 举证 | 全面举证 | 主要围绕新证据和争议焦点 |
| 改判难度 | — | 较高（维持原判是原则） |

> 💡 **核心策略**：二审不是"再讲一遍"，而是"精准攻击原审错误"。所有庭审发言都围绕"原审哪里错了"展开。

## ❓ 常见问题

**Q1: 二审质证意见和一审有什么不同？**
> 二审质证主要围绕：① 对方新提交的证据；② 一审已提交但双方仍有争议的证据。对已确认的证据通常不再重复质证。

**Q2: 二审法官会问什么？**
> 二审法官通常关注：① 上诉理由是否成立；② 一审认定的事实是否有误；③ 新证据是否可采；④ 法律适用是否正确。对一审已查明且无争议的事实，通常不再详细询问。

**Q3: 如果二审不开庭怎么办？**
> 如法院决定书面审理，律师可提交书面代理意见，系统阐述上诉理由。开庭提纲转为"书面代理意见提纲"使用。
