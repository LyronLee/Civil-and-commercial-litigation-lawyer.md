# 02-阶段产出

本文件夹用于存放二审庭审准备阶段 AI 生成的工作成果文件。

## 产出文件清单

| 序号 | 文件名 | 说明 |
|------|--------|------|
| 01 | 质证意见.md | 对对方证据的真实性、合法性、关联性意见 |
| 02 | 开庭提纲.md | 二审庭审发言提纲（围绕上诉焦点） |
| 03 | 庭审纪要.md | 庭审过程记录（庭后整理） |

## 使用方式

AI 根据 `SKILL.md` 工作流生成的文件自动保存于此。用户可直接使用或按需修改。

## 注意事项

- 所有文件生成后标注"初稿，需律师审核后使用"
- 质证意见和开庭提纲为**内部使用**，不直接提交法院
- 庭审纪要应在庭审结束后尽快整理，确保细节准确
- 二审审理范围限于上诉请求，所有准备围绕上诉焦点展开
