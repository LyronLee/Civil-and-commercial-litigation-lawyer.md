# 参考模板

本文件夹存放二审程序阶段三（庭审准备）的参考模板，供 AI 生成阶段产出时参照使用。

## 模板清单

| 序号 | 模板名称 | 用途 |
|------|---------|------|
| 00-二审质证意见模板.md | 对对方二审新证据的质证文书 | 四性分析（真实性、合法性、关联性、证明力）+ 逾期举证攻击 + 策略性评价 |
| 01-二审开庭提纲模板.md | 庭审前内部准备文件 | 程序核对、法官发问预判、回答禁区、我方发问、争议焦点、辩论要点、突发应对 |
| 02-二审庭审纪要模板.md | 庭审记录模板 | 基本信息、程序事项、法庭调查、法庭辩论、最后陈述、其他安排 |

## 使用方式

1. 本文件夹的模板为**参照模板**，AI 在执行 SKILL.md 工作流时会自动调用
2. 用户无需手动填写模板，将案件材料按 SKILL.md 要求准备后，唤起 AI 即可
3. 如需了解模板格式，可直接打开查看

## 工作流分支

| 触发条件 | 生成分支 |
|---------|---------|
| 收到对方二审新材料 | 分支A：质证意见 |
| 收到二审开庭传票 | 分支B：开庭提纲 |
| 二审庭审结束后 | 分支C：庭审纪要 |
