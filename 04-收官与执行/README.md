# 04-收官与执行

> **定位**：庭审结束后至案件彻底闭环的全部工作。分两个时点：① 开完庭后立即补充代理词；② 收到判决书后处理裁判应对、退费、结案与复盘。
> 
> **时点一（开完庭后）**：根据庭审情况撰写代理词，强化法官心证。  
> **时点二（收到判决书后）**：根据判决结果选择上诉或执行，完成退费、结案归档和复盘总结。

## 关键设计原则

1. **两个时点，分步触发**：阶段四不是一次性完成的。开完庭后先写代理词；收到判决后再处理后续事项。避免在尚未收到判决时提前生成不适用的文书。
2. **代理词基于庭审事实**：代理词必须基于庭审中已经查明的事实和双方确认的证据，不引入新的事实主张，重在系统梳理和强化论证。
3. **卷宗索引，不重复搬运**：本阶段通过路径引用前三阶段的完整卷宗，保持单份文件。
4. **裁判应对二选一**：收到判决书后，根据结果选择"上诉"或"执行"，二者互斥。
5. **复盘是阶段四的核心增值**：不仅是结案手续，更是对全流程的系统回顾。AI基于完整卷宗提供客观分析，律师/当事人补充主观经验。
6. **退费不遗漏**：诉讼费退费是容易被忽略的程序性事项，单独设立文件夹和产出，确保闭环。
7. **法院来文闭环管理**：判决书、裁定书、生效证明等放入`02-法院来文/`，与阶段二的`05-法院来文/`形成时间线延续。
