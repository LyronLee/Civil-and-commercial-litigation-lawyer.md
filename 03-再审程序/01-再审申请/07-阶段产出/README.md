# 07-阶段产出

本文件夹用于存放再审申请阶段 AI 生成的工作成果文件。

## 产出文件清单

| 序号 | 文件名 | 说明 |
|------|--------|------|
| 01 | 再审申请书.md | 提交法院的再审申请书 |
| 02 | 证据目录.md | 再审申请证据材料目录 |
| 03 | 类案检索报告.md | 支持再审事由的类案检索 |
| 04 | 原审判决分析清单.md | 原审判决问题诊断 |
| 05 | 关键期限跟踪表.md | 再审相关期限跟踪 |
| 06 | 再审审查询问准备.md | 法院询问时的应答准备 |

## 使用方式

AI 根据 `SKILL.md` 工作流生成的文件自动保存于此。用户可直接使用或按需修改。

## 注意事项

- 所有文件生成后标注"初稿，需律师审核后使用"
- 再审申请书为核心产出，必须准确引用2023年版《民事诉讼法》条号
- 再审申请期限为判决生效后 **6 个月**（除斥期间，不可延长）
- 关键期限跟踪表应持续更新，确保不遗漏任何期限
