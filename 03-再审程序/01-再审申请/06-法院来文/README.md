# 法院来文

> 存放再审申请阶段法院发送的各类法律文书，包括受理通知书、应诉通知书、传票、裁定书等。

---

## 常见法院来文类型

| 文书类型 | 触发时机 | 注意事项 |
|---------|---------|---------|
| 受理通知书 | 法院决定受理再审申请 | 确认案号、承办法官、联系方式 |
| 应诉通知书 | 发送给对方当事人时 | 附带再审申请书副本 |
| 传票 | 组织询问/听证时 | 注意时间地点，提前准备 |
| 裁定书（再审） | 法院裁定再审时 | 确认再审审理法院 |
| 裁定书（驳回） | 法院驳回再审申请时 | 起算检察监督2年期限 |
| 补充材料通知书 | 法院要求补充材料时 | 注意指定期限 |

---

## 期限提取

收到法院来文后，应立即提取其中指定的所有期限，并更新《关键期限跟踪表》。
