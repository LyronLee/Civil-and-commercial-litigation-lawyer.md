# 01 再审申请 SKILL

> ⚠️ 依据2023年修正《民事诉讼法》（2024年1月1日施行）。旧版条号已全面变更，详见根目录SKILL.md法条对照表。

---

## 工作流总览

```
Step 1: 原审裁判结构化分析
Step 2: 再审事由逐一匹配（13种+调解书2种+案外人规则）
Step 3: 成功率预测与策略决策
Step 4: 新证据评估（三步递进审查法）
Step 5: 管辖确定与期限核查
Step 6: 再审申请书起草
Step 7: 证据目录与类案检索
Step 8: 被申请人视角（书面意见）
Step 9: 再审审查询问/听证准备
Step 10: 材料提交与期限跟踪
```

## Step 1：原审裁判结构化分析

> 借鉴二审程序的"判决书分析"方法，但聚焦于"原审裁判哪里错了"。

### 1.1 分析清单

| 分析维度 | 分析要点 | 与再审事由的关联 |
|---------|---------|-----------------|
| **裁判类型** | 判决/裁定/调解书 | 调解书仅适用第212条2种事由 |
| **生效时间** | 判决送达日期、生效日期 | 计算6个月除斥期间 |
| **诉讼请求 vs 判项** | 逐一比对，识别漏判/超判 | 对应第211条第（十一）项 |
| **事实认定** | 哪些基本事实有证据支撑？哪些缺乏证据？ | 对应第211条第（二）项 |
| **主要证据** | 是否存在伪造/未经质证/申请调取未获准？ | 对应第211条第（三）（四）（五）项 |
| **法律适用** | 引用了哪些法条？是否正确？ | 对应第211条第（六）项 |
| **程序事项** | 审判组织、回避、辩论权、传唤等 | 对应第211条第（七）至（十）项 |
| **裁判依据** | 据以裁判的其他法律文书是否被撤销？ | 对应第211条第（十二）项 |
| **审判人员** | 是否存在枉法裁判行为？ | 对应第211条第（十三）项 |

### 1.2 产出文件

生成 `01-原审裁判结构化分析.md`，格式参考 `02-二审程序/01-战略分析/03-参考模板/00-判决书分析清单模板.md`。

## Step 2：再审事由逐一匹配

### 2.1 判决/裁定再审事由（《民事诉讼法》第二百一十一条）

> 法院仅对当事人提出的再审事由进行审查，必须精准对应。

| 项 | 再审事由 | 常见适用场景 | 成功率评估 | 证据支持要求 |
|---|---------|------------|-----------|------------|
| （一） | 有新的证据，足以推翻原判决、裁定的 | 原审庭审结束后新发现；原审申请调取未获准；客观上无法取得 | ⭐⭐⭐ 高（如有真新证据） | 新证据+三步审查+不可归责说明 |
| （二） | 原判决、裁定认定的基本事实缺乏证据证明的 | 关键事实无任何证据支撑 | ⭐⭐ 中 | 指出具体事实+证明其缺乏证据 |
| （三） | 原判决、裁定认定事实的主要证据是伪造的 | 对方提交的证据系伪造 | ⭐⭐⭐ 高（如已证实） | 伪造证据的鉴定/刑事判决 |
| （四） | 原判决、裁定认定事实的主要证据未经质证的 | 关键证据未在庭审中质证 | ⭐⭐⭐ 高 | 庭审笔录+证据清单比对 |
| （五） | 对审理案件需要的主要证据，当事人因客观原因不能自行收集，书面申请人民法院调查收集，人民法院未调查收集的 | 申请法院调取证据被无理由拒绝 | ⭐⭐ 中 | 书面调取申请+法院拒绝记录 |
| （六） | 原判决、裁定适用法律确有错误的 | 引用法条错误、遗漏法律适用、违背司法解释 | ⭐⭐ 中 | 正确法条+类案检索报告 |
| （七） | 审判组织的组成不合法或者依法应当回避的审判人员没有回避的 | 合议庭组成违法、应回避未回避 | ⭐⭐⭐ 高（如已证实） | 审判组织信息+回避事由 |
| （八） | 无诉讼行为能力人未经法定代理人代为诉讼或者应当参加诉讼的当事人，因不能归责于本人或者其诉讼代理人的事由，未参加诉讼的 | 遗漏必要共同诉讼当事人 | ⭐⭐⭐ 高（如已证实） | 当事人身份+未参加原因 |
| （九） | 违反法律规定，剥夺当事人辩论权利的 | 不允许发言、打断陈述、未告知权利 | ⭐⭐ 中 | 庭审笔录/录像 |
| （十） | 未经传票传唤，缺席判决的 | 未依法送达开庭传票即缺席判决 | ⭐⭐⭐ 高 | 送达回证+传票记录 |
| （十一） | 原判决、裁定遗漏或者超出诉讼请求的 | 漏判某项诉请或判决超出诉请范围 | ⭐⭐⭐ 高 | 诉请清单+判项比对 |
| （十二） | 据以作出原判决、裁定的法律文书被撤销或者变更的 | 作为裁判依据的其他裁判/行政行为被撤销 | ⭐⭐⭐ 高（如已发生） | 撤销/变更文书 |
| （十三） | 审判人员审理该案件时有贪污受贿，徇私舞弊，枉法裁判行为的 | 已生效刑事判决或纪律处分确认 | ⭐⭐⭐ 高（如已证实） | 刑事判决/纪律处分决定 |

> 💡 **实务提醒**：成功率评估仅供参考，实务中因案而异。第（一）（三）（四）（七）（八）（十）（十一）（十二）（十三）项在有充分证据支持时成功率较高；第（二）（五）（六）（九）项需结合具体案情判断。

### 2.2 调解书再审事由（《民事诉讼法》第二百一十二条）

> ⚠️ **独立适用**：调解书不适用上述13种事由，仅适用以下2种：

| 项 | 再审事由 | 证明要求 |
|---|---------|---------|
| 1 | 调解违反自愿原则 | 证明调解过程中存在胁迫、欺诈、重大误解等情形 |
| 2 | 调解协议内容违反法律 | 证明调解协议违反法律强制性规定或公序良俗 |

### 2.3 案外人申请再审规则

> 依据《民事诉讼法》第二百三十四条（原第227条）及《民诉法解释》第420-423条

**适用情形**：案外人对原判决、裁定、调解书确定的执行标的物主张权利，且无法提起新的诉讼解决争议的。

**期限**：自执行异议裁定送达之日起 **6个月内**。

**路径互斥**：
- 先启动执行异议 → 只能申请再审（不得提起第三人撤销之诉）
- 先提起第三人撤销之诉 → 只能继续三撤（不得申请再审）

## Step 3：成功率预测与策略决策

### 3.1 五维度评分模型

| 维度 | 权重 | 评估标准 | 得分 |
|------|------|---------|------|
| 再审事由强度 | 30% | 是否属于高成功率事由 | /10 |
| 证据充分性 | 25% | 证据是否足以支撑再审事由 | /10 |
| 错误清晰度 | 20% | 原审错误是否明确、具体 | /10 |
| 新证据冲击力 | 15% | 新证据对原审认定的冲击程度 | /10 |
| 类案/先例支持 | 10% | 是否有支持我方观点的类案 | /10 |

**评级**：
- A级（≥8.0）：建议积极申请再审
- B级（6.0-7.9）：建议申请，但需充分准备
- C级（<6.0）：客观告知成功率较低，由当事人决定是否申请

### 3.2 策略决策

```
评分结果
    │
    ├─ A级 → 进入Step 4-10，全面准备
    │
    ├─ B级 → 进入Step 4-10，重点补强薄弱环节
    │
    └─ C级 → 客观告知风险，提供替代路径（检察监督/和解/履行）
         → 如当事人坚持申请，进入Step 4-10，但明确提示风险
```

## Step 4：新证据评估（三步递进审查法）

> 再审中"新证据"的认定标准高于二审，需经过严格审查。

### 4.1 第一步：审实质——是否足以推翻原审要件事实

| 评估项 | 分析要点 | 合格标准 |
|--------|---------|---------|
| 关联性 | 是否直接指向原审裁判的"要件事实" | 决定法律关系成立、变更、消灭的核心事实 |
| 证明力 | 是否达到"高度盖然性" | 能使原审要件事实陷入"真伪不明"或被直接推翻 |
| 显著性 | 是否直接冲击原裁判据以定案的基本事实 | 非一般瑕疵 |

### 4.2 第二步：审形式——是否具有崭新性和不可归责性

| 评估项 | 分析要点 | 三层过滤机制 |
|--------|---------|-------------|
| 崭新性 | 原审庭审结束后新发现 / 原审中申请调取未获准 / 原审已存在但客观上无法取得 | — |
| 不可归责性 | 逾期举证不可归责于当事人 | ①证据由第三方掌控 ②不可抗力 ③法院程序瑕疵 |

### 4.3 第三步：审主观——逾期举证是否可归责于当事人

| 情形 | 结论 |
|------|------|
| 可归责（故意或重大过失） | 证据失权，不予采纳 |
| 不可归责 | 可认定为新证据 |
| 涉及国家利益/公共利益/他人合法权益 | 可例外采纳，但可训诫/罚款 |

### 4.4 新证据说明模板

每份新证据需单独说明：
- 证据名称及形式
- 证据来源
- 取得时间（必须在原审庭审结束后，或在原审中因客观原因无法取得）
- 原审未提交的具体原因
- 对推翻原判的作用分析
- 三步审查自评结论

## Step 5：管辖确定与期限核查

### 5.1 管辖规则（《民事诉讼法》第二百一十条）

| 情形 | 管辖法院 | 备注 |
|------|---------|------|
| 一般情形 | 原审法院的上一级人民法院 | 原则 |
| 当事人一方人数众多 | 可以向原审人民法院申请再审 | 例外 |
| 当事人双方为公民 | 可以向原审人民法院申请再审 | 例外 |

### 5.2 诉讼费用（再审特殊规则）

| 情形 | 费用规则 |
|------|---------|
| 一般情形 | 原则上**免交**案件受理费 |
| 当事人有新证据，足以推翻原判决、裁定，向人民法院申请再审，人民法院经审查决定再审的案件 | 应当**交纳**案件受理费 |
| 其他情形 | 参照《诉讼费用交纳办法》 |

### 5.3 期限核查（最终确认）

```
判决/裁定生效日期: ____年____月____日
+ 6个月除斥期间
= 再审申请截止日期: ____年____月____日

今日日期: ____年____月____日
距离截止: ____天

⚠️ 如截止日期 ≤ 30日，标记为紧急
⚠️ 如截止日期 ≤ 7日，标记为极度紧急，建议立即起草申请书
```

## Step 6：再审申请书起草

### 6.1 必须使用标准结构

1. **当事人基本信息**：各方当事人的一、二审诉讼地位完整标注
2. **原审裁判基本情况**：原审法院、案号、裁判日期、裁判类型
3. **再审请求**：三项结构——对原审裁判意见 + 改判请求 + 诉讼费用
4. **事实与理由**：
   - （一）基本事实（时间轴梳理）
   - （二）申请再审的法定情形（按第211条逐项论证，观点先行）
5. **证据和证据来源**：如有新证据，写明证据来源、证人姓名和住所
6. **尾部**：此致法院、再审申请人签章（不得由律师代签）、日期

### 6.2 撰写要点

1. 文书名称统一为"再审申请书"
2. 精准对应第211条具体款、项
3. 新证据应在一级标题中醒目标注（加粗、醒目颜色）
4. 引用原审判决原文后加以批驳
5. 再审请求不得超出原审诉讼请求范围
6. 单面打印、左上角装订

### 6.3 产出文件

- `01-再审申请书.md`

## Step 7：证据目录与类案检索

### 7.1 证据目录

按4组排列：
1. 主体资格及原审裁判文书
2. 原审诉讼中已提交的主要证据
3. 支持再审事由及再审请求的证据
4. 新证据（如有）

> 新证据应按被申请人及原审其他当事人人数提交相应份数副本。

### 7.2 类案检索报告

如以"适用法律错误"申请再审，建议提交类案检索报告：
- 检索主体、时间、平台、方法、关键词
- 类案核验（生效、未撤销、无风险）
- 按争议焦点分层次编排
- 结果运用分析

### 7.3 产出文件

- `02-证据目录.md`
- `03-类案检索报告.md`（如适用）

## Step 8：被申请人视角（书面意见）

> 依据《民事诉讼法》第二百一十四条：法院应在收到再审申请书之日起5日内将副本发送对方当事人，对方当事人应在15日内提交书面意见。

### 8.1 如我方是被申请人（原审胜诉方）

**收到再审申请书副本后15日内**，需提交书面意见：

1. **逐条反驳再审事由**：再审申请人有几项事由，就逐条反驳
2. **维护原审判决**：强调原审认定事实清楚、适用法律正确
3. **新证据阻击**：重点论证对方提交的证据不属于新证据或不足以推翻原判
4. **程序抗辩**：如对方超期申请、已有再审裁判等

**产出文件**：`04-再审答辩状.md`（如对方为申请人）或独立书面意见

### 8.2 如我方是再审申请人

**预判对方书面意见的反驳方向**，在代理词中提前应对。

## Step 9：再审审查询问/听证准备

> 高级人民法院审查再审申请时，常组织询问或听证。询问/听证是再审审查的重要环节。

### 9.1 询问/听证的特点

- **非正式开庭**：通常由承办法官主持，程序较灵活
- **重点审查**：围绕再审事由展开，不全面审理案件
- **新证据质证**：如有新证据，询问/听证是质证的关键场合
- **时间较短**：通常1-2小时

### 9.2 准备要点

1. **简明陈述**：5-10分钟内清晰陈述再审事由和核心主张
2. **新证据展示**：如有新证据，准备PPT或可视化材料
3. **法官提问预判**：预判法官可能关注的问题，准备应答
4. **对方反驳应对**：预判对方可能提出的反驳，准备回应
5. **最后陈述**：简洁有力的结论性陈述

### 9.3 产出文件

- `05-再审审查询问准备.md`

|------|---------|------|------|
| 1 | 再审申请书正本 | 1份 | 申请人签章 |

| 3 | 身份证明/营业执照 | 1份 | |
| 4 | 原审裁判文书 | 1份 | 二审终审的附一审文书 |
| 5 | 原审主要证据 | 1份 | |
| 6 | 支持再审事由的证据 | 1份 | |
| 7 | 新证据 | 按对方人数+1 | |
| 8 | 送达地址确认书 | 1份 | |
| 9 | 材料清单 | 2份 | |

### 10.2 提交后的程序节点

```
提交再审申请书
    │
    ├─ 法院5日内发送副本给对方当事人（第214条）
    │
    ├─ 对方15日内提交书面意见（第214条）
    │
    ├─ 法院3个月内审查（第215条）
    │    │
    │    ├─ 裁定再审 → 进入 02-再审审理/
    │    │
    │    └─ 裁定驳回 → 检察监督路径（2年内）
    │
    └─ 审查期间可能组织询问/听证
```
