# 原审卷宗

本文件夹用于存放原审程序（一审+二审）的全部卷宗材料，是再审申请阶段的核心输入。

## 应放入的材料

| 类别 | 具体材料 | 优先级 |
|------|---------|--------|
| 生效裁判文书 | 一审判决书/裁定书、二审判决书/裁定书、调解书 | 🔴 必备 |
| 起诉状/上诉状 | 原审起诉状、上诉状 | 🔴 必备 |
| 答辩状 | 原审答辩状、二审答辩状 | 🔴 必备 |
| 证据目录及材料 | 原审全部证据目录和证据册 | 🔴 必备 |
| 庭审笔录/纪要 | 一审、二审庭审记录 | 🟡 重要 |
| 代理词 | 一审、二审代理词 | 🟡 重要 |
| 程序性文书 | 受理通知、举证通知、传票、应诉通知等 | 🟡 重要 |
| 对方来文 | 原审中对方提交的全部文书 | 🟡 重要 |
| 法院来文 | 原审中法院送达的全部文书 | 🟡 重要 |

## 使用方式

1. 将原审全部卷宗材料放入本文件夹
2. AI 将基于原审裁判文书进行结构化分析，匹配再审事由
3. 确保生效裁判文书位于本文件夹内，便于 AI 优先读取

## 注意事项

- 原审卷宗是再审分析的**唯一事实基础**，务必确保完整
- 如原审由其他律师代理，需尽可能收集全部材料
- 生效裁判文书的**生效日期**是计算再审申请期限（6个月）的关键
- 缺少关键材料（如庭审笔录）时，应在阶段产出中标注并列入待补充清单
