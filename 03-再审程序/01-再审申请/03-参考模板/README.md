# 参考模板

本文件夹存放再审程序阶段一（再审申请）的参考模板，供 AI 生成阶段产出时参照使用。

## 模板清单

| 序号 | 模板名称 | 用途 |
|------|---------|------|
| 00-再审申请书模板.md | 向法院申请再审的正式文书 | 当事人信息、原审裁判情况、再审请求、事实与理由（精准对应第211条） |
| 01-证据目录模板.md | 再审证据编目 | 4组排列：主体资格及原审裁判、原审已提交证据、支持再审事由证据、新证据 |
| 02-类案检索报告模板.md | 类案检索格式 | 如以"适用法律错误"为由申请再审，建议提交 |
| 03-再审审查询问准备模板.md | 高院询问/听证准备 | 简明陈述、新证据展示、法官提问预判、对方反驳应对 |
| 04-原审裁判结构化分析模板.md | 原审裁判错误诊断框架 | 事实认定、主要证据、法律适用、程序事项等维度分析 |
| 05-关键期限跟踪表模板.md | 期限汇总 | 再审申请期、法院发送副本期、对方提交意见期、法院审查期、检察监督期 |

## 特殊案由适配

如案件属于特殊再审案由（新证据类/适用法律错误类/程序违法类/合同纠纷格式条款类/物权确认类），AI 会自动调用 `特殊案由适配/` 子文件夹中的专属材料收集和论证要点。

## 使用方式

1. 本文件夹的模板为**参照模板**，AI 在执行 `SKILL.md` 工作流时会自动调用
2. 用户无需手动填写模板，将原审卷宗放入 `01-原审卷宗/` 后，唤起 AI 即可
3. 如需了解模板格式，可直接打开查看

## 产出位置

AI 根据本阶段 SKILL.md 生成的文件保存在 `07-阶段产出/` 中。
