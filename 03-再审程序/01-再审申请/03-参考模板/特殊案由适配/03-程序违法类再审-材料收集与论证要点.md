# 程序违法类再审（第211条第7-11项）材料收集与论证要点

> 适用情形：原审存在审判组织不合法、回避缺失、当事人遗漏、辩论权剥夺、缺席判决、漏判/超判等程序违法情形。

---

## 一、核心证据材料（按违法类型分类）

### 1. 审判组织不合法或回避缺失（第7项）

- [ ] **审判组织组成信息**：合议庭成员名单、人民陪审员信息
- [ ] **回避申请记录**：原审中是否提出回避申请及处理结果
- [ ] **应回避未回避的证据**：如审判人员与当事人有亲属关系、利益关系的证明
- [ ] **审判组织组成不合法的证据**：如应当组成合议庭却独任审理、陪审员比例不符合规定等

### 2. 当事人遗漏（第8项）

- [ ] **遗漏当事人的身份信息**：姓名/名称、与原审当事人的法律关系
- [ ] **遗漏当事人的必要性说明**：说明该当事人为何是必要共同诉讼当事人
- [ ] **遗漏原因证明**：证明该当事人未参加诉讼非因其自身原因（如未收到通知）
- [ ] **原审送达记录**：查看是否依法向遗漏当事人送达诉讼文书

### 3. 辩论权剥夺（第9项）

- [ ] **原审庭审笔录/录像**：查看庭审中是否存在不允许发言、打断陈述、限制辩论时间等情形
- [ ] **当事人书面陈述**：当事人对辩论权被剥夺的书面说明
- [ ] **代理人陈述**：代理人对辩论权受限的书面说明
- [ ] **证人证言**：旁听人员对庭审情况的证言

### 4. 未经传票传唤缺席判决（第10项）

- [ ] **送达回证**：查看是否依法送达开庭传票
- [ ] **送达方式记录**：邮寄送达的签收记录、公告送达的报纸版面
- [ ] **当事人未收到传票的说明**：当事人书面说明未收到开庭传票
- [ ] **送达地址确认书**：查看送达地址是否正确

### 5. 遗漏或超出诉讼请求（第11项）

- [ ] **原审起诉状/上诉状**：完整列出全部诉讼请求
- [ ] **原审判决/裁定主文**：逐一比对诉讼请求与判项
- [ ] **变更诉讼请求记录**：原审中是否变更过诉讼请求
- [ ] **反诉请求**：如有反诉，比对反诉请求与判项

---

## 二、辅助证据材料

- [ ] **原审庭审笔录**：全面审查庭审程序是否合法
- [ ] **原审送达回证**：审查所有诉讼文书的送达情况
- [ ] **原审合议庭评议记录**（如可调取）：审查合议庭组成和评议程序
- [ ] **原审判决书**：逐页审查程序事项的记载

---

## 三、论证要点

### 3.1 程序违法与实体结果的因果关系

> 重要提示：并非所有程序违法都能启动再审。程序违法必须"可能影响案件正确判决/裁定"。

| 程序违法类型 | 因果关系论证要点 |
|-------------|-----------------|
| 审判组织不合法 | 如合议庭组成违法，可能导致案件未经充分讨论即作出裁判 |
| 回避缺失 | 如审判人员与当事人有利害关系，可能影响裁判中立性 |
| 当事人遗漏 | 遗漏必要共同诉讼当事人，导致案件事实未查清 |
| 辩论权剥夺 | 当事人未能充分陈述意见，导致法院未能全面查明事实 |
| 未经传票传唤缺席判决 | 当事人未能到庭陈述和质证，导致事实认定错误 |
| 遗漏诉讼请求 | 当事人的合法权益未得到裁判确认 |
| 超出诉讼请求 | 法院对当事人未主张的事项作出裁判，违反处分原则 |

### 3.2 常见抗辩及应对

| 对方抗辩 | 应对策略 |
|---------|---------|
| "程序瑕疵不影响实体结果" | 论证程序违法与实体结果之间的因果关系 |
| "当事人已放弃相关权利" | 证明当事人未放弃权利，或放弃行为非真实意思表示 |
| "原审已补正程序" | 指出原审未补正，或补正不符合法律规定 |

---

## 四、注意事项

1. **证据取得难度**：程序违法的证据（如庭审录像、合议庭评议记录）可能需要向法院档案室申请调取
2. **因果关系是关键**：单纯证明程序存在瑕疵不足以启动再审，必须论证该程序违法可能影响实体结果
3. **及时调取证据**：庭审录像等证据可能有一定的保存期限，需及时申请调取
4. **注意与第7-11项的精准对应**：不同的程序违法对应不同的再审事由项，必须精准对应
