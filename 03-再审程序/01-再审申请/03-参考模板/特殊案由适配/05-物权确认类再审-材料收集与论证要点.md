# 物权确认类再审材料收集与论证要点

> 适用情形：物权确认纠纷（如车位、车库、地下停车场、共有部分权属确认等）中，原审判决对权属归属的认定存在适用法律错误（第211条第6项）或认定事实缺乏证据证明（第211条第2项）。

---

## 一、核心证据材料

### 1. 权属来源证据

- [ ] **土地出让合同/划拨决定书**：查看关于配套设施权属的原始约定
- [ ] **建设工程规划许可证**：确认争议部位的规划用途
- [ ] **房屋面积测绘报告**：确认争议部位是否计入公摊面积
- [ ] **竣工验收资料**：确认争议部位的竣工验收情况
- [ ] **不动产权证书/房产证**：查看权属登记情况

### 2. 合同证据

- [ ] **商品房买卖合同（预售/现售）**：查看关于争议部位权属的条款
- [ ] **补充协议/特别约定**：查看是否有关于争议部位的特别约定
- [ ] **前期物业服务合同**：查看物业管理范围的约定
- [ ] **业主公约/管理规约**：查看业主共同约定的权属规则

### 3. 费用分摊证据

- [ ] **房价构成说明**：查看房价中是否包含争议部位的建设成本
- [ ] **公摊面积计算表**：查看争议部位是否计入公摊
- [ ] **物业费/维修金缴纳记录**：查看争议部位是否由业主共同承担费用
- [ ] **建设成本审计报告**：如有，查看争议部位的建设成本分摊情况

### 4. 实际使用与管理证据

- [ ] **实际使用照片/视频**：证明争议部位的实际使用情况
- [ ] **管理记录**：如停车场的管理、收费、维护记录
- [ ] **业主大会决议**：关于争议部位管理的业主大会决议
- [ ] **物业公司交接记录**：物业公司更换时的交接记录

---

## 二、辅助证据材料

- [ ] **同类小区裁判案例**：检索同类小区物权确认纠纷的裁判规则
- [ ] **地方性法规/规章**：如地方关于车位、车库权属的规定
- [ ] **行政部门意见**：如住建部门、规划部门关于争议部位的行政答复
- [ ] **专家意见**：房地产法专家关于争议部位权属的法律意见

---

## 三、论证要点

### 3.1 《民法典》物权编的适用框架

```
一、争议部位的性质认定
   1. 规划用途：争议部位是否属于建筑区划内规划用于停放汽车的车位、车库
   2. 建设成本：建设成本是否已分摊至全体业主的房价中
   3. 土地使用权：争议部位占用的土地使用权归属

二、《民法典》第二百七十五条的适用
   1. 如属于"规划用于停放汽车的车位、车库"：适用"约定优先"原则
   2. 但"约定"应指所有权转移的约定，而非格式合同中的单方声明
   3. 如约定不明或存在格式条款：应审查提示说明义务

三、《民法典》第二百七十四条的适用
   1. 如争议部位属于"其他公共场所、公用设施"：属于业主共有
   2. 应结合规划用途、建设成本分摊、实际使用情况综合判断

四、对原审判决的批驳
   1. 原审判决对争议部位性质的认定错误
   2. 原审判决对《民法典》第二百七十五条/第二百七十四条的适用错误
   3. 原审判决忽视了建设成本分摊、土地使用权归属等关键因素
```

### 3.2 常见争议焦点

| 争议焦点 | 对我方有利的论证方向 | 关键证据 |
|---------|-------------------|---------|
| 是否属于"规划车位" | 争议部位实际用途与规划用途不一致 | 规划许可证、实际使用照片 |
| "约定"是否有效 | 约定属于格式条款，未履行提示说明义务 | 合同文本、提示说明记录 |
| 建设成本是否分摊 | 房价构成中包含争议部位的建设成本 | 房价构成说明、审计报告 |
| 是否属于业主共有 | 争议部位属于公用设施，应归业主共有 | 测绘报告、管理记录 |

### 3.3 常见抗辩及应对

| 对方抗辩 | 应对策略 |
|---------|---------|
| "合同约定归开发商所有" | 论证约定属于格式条款，未履行提示说明义务；或论证约定违反了《民法典》关于业主共有权益的强制性规定 |
| "开发商享有原始所有权" | 论证争议部位的建设成本已分摊至房价，开发商已通过对价取得对价，不享有原始所有权 |
| "业主20余年未提出异议" | 论证业主未提出异议不等于知情并认可；或论证业主大会决议已表明业主的真实意思 |

---

## 四、注意事项

1. **规划用途是关键**：规划许可证上关于争议部位的规划用途，是判断权属归属的重要依据
2. **建设成本分摊是核心**：如建设成本已分摊至房价，应认定属于业主共有（或至少不能单独归开发商所有）
3. **注意地方性规定**：部分地区有关于车位、车库权属的地方性法规或规章，应一并检索
4. **类案检索重点**：重点检索最高法院和本省高院的物权确认类案，尤其是同一城市的类案
5. **新证据方向**：如原审中未能取得规划许可证、测绘报告等关键证据，可作为新证据申请再审
