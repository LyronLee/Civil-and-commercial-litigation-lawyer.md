# 适用法律错误类再审（第211条第6项）材料收集与论证要点

> 适用情形：原判决、裁定适用法律确有错误的。

---

## 一、核心证据材料

### 1. 法律条文材料

- [ ] **原审引用的法条全文**：准确摘录原审判决/裁定中引用的所有法律条文
- [ ] **正确法条全文**：准备应当适用的法律条文及其立法解释、司法解释
- [ ] **法条变迁说明**：如法律有修订，说明原审适用的是旧法还是新法，是否存在适用错误
- [ ] **上位法与下位法关系**：如原审引用了下位法但遗漏了上位法，或下位法与上位法冲突

### 2. 类案检索材料

- [ ] **最高人民法院指导性案例**：与本案法律适用问题相关的指导性案例
- [ ] **最高人民法院公报案例**：与本案法律适用问题相关的公报案例
- [ ] **上级法院类案**：与本案同一法院或上级法院的类似裁判
- [ ] **类案检索报告**：详细记录检索方法、检索结果、对比分析

### 3. 法学理论材料

- [ ] **权威学者观点**：知名学者关于争议法律问题的学术论文或著作观点
- [ ] **立法机关解释**：全国人大法工委、最高法院的立法解释或理解与适用
- [ ] **司法观点**：最高法院各业务庭的司法观点、会议纪要

---

## 二、辅助证据材料

- [ ] **原审判决/裁定全文**：用于逐条批驳法律适用错误
- [ ] **原审庭审笔录**：查看原审中当事人是否已提出法律适用异议
- [ ] **原审代理词**：查看原审代理词中是否已指出法律适用问题

---

## 三、论证要点

### 3.1 "适用法律错误"的常见类型

| 错误类型 | 示例 | 论证方法 |
|---------|------|---------|
| 引用法条错误 | 原审应引用A法条却引用了B法条 | 对比A法条与B法条的适用范围和构成要件 |
| 遗漏法律适用 | 原审遗漏了应当适用的司法解释 | 说明该司法解释的适用条件和本案的符合性 |
| 法律解释错误 | 原审对法律条文的理解违背了立法本意 | 引用立法解释、权威学者观点、类案裁判规则 |
| 新旧法适用错误 | 原审在应当适用新法时适用了旧法 | 说明新旧法的生效时间和本案事实发生时间 |
| 上位法与下位法冲突 | 原审引用的下位法与上位法相抵触 | 引用上位法优先原则 |

### 3.2 论证结构

```
一、原审判决第X页引用了《XX法》第X条，该引用存在以下错误：
   1. 该法条的适用范围是XX，而本案的事实是XX，不属于该法条的调整范围
   2. 正确的法条应当是《XX法》第X条，其构成要件为XX，本案事实完全符合

二、原审判决遗漏了应当适用的《XX司法解释》第X条：
   1. 该司法解释的适用条件是XX
   2. 本案完全符合该适用条件
   3. 如适用该司法解释，裁判结果应当是XX

三、类案检索显示，最高人民法院/上级法院在类似案件中的裁判规则为XX，与原审判决不一致
```

### 3.3 常见抗辩及应对

| 对方抗辩 | 应对策略 |
|---------|---------|
| "原审法律适用正确" | 逐条对比法条原文，指出引用错误或遗漏 |
| "类案与本案事实不同" | 强调法律适用问题的相似性，而非事实的完全相同 |
| "法律解释存在多种理解" | 引用立法解释、最高法院观点、多数说，说明原审采用的是少数说或错误理解 |

---

## 四、注意事项

1. **精准引用法条原文**：论证时必须引用法条的完整原文，不能仅凭记忆转述
2. **类案检索要全面**：不能仅检索对我方有利的类案，也要检索对对方有利的类案并说明不适用原因
3. **避免学术化过度**：论证应以法条和司法解释为核心，学者观点仅作辅助
4. **注意法律生效时间**：确认原审裁判作出时，所引用的法律是否已经生效
