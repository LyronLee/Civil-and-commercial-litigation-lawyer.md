# 合同纠纷格式条款类再审材料收集与论证要点

> 适用情形：合同纠纷中，原审判决依据格式条款作出对当事人不利的认定，存在适用法律错误（第211条第6项）或认定事实缺乏证据证明（第211条第2项）。

---

## 一、核心证据材料

### 1. 合同文本材料

- [ ] **完整合同文本**：包括主合同、补充协议、附件、特别约定等全部文件
- [ ] **格式条款标识**：标注哪些条款属于格式条款（通常由一方预先拟定、未与对方协商的条款）
- [ ] **合同签署过程记录**：如签约现场照片、视频、见证人证言等
- [ ] **合同版本历史**：如合同有多个版本，收集全部版本并标注变更内容

### 2. 提示说明义务相关材料

- [ ] **提示说明的书面记录**：如签约时向对方发送的条款说明邮件、短信、微信记录
- [ ] **条款加粗/下划线/特殊字体标识**：证明已采取合理方式提示对方注意
- [ ] **对方确认记录**：对方签字确认已阅读并理解条款内容的文件
- [ ] **签约时的录音/录像**：证明签约时已口头提示说明（如有）
- [ ] **无提示说明的证据**：如格式条款无任何特殊标识，未要求对方签字确认等

### 3. 行业惯例与交易习惯材料

- [ ] **同行业合同范本**：证明涉案条款是否为行业通用格式条款
- [ ] **交易习惯证明**：如双方长期合作中形成的交易习惯，与格式条款不一致
- [ ] **行业协会意见**：行业协会对争议条款的意见或指引

### 4. 显失公平相关材料

- [ ] **权利义务对比表**：对比格式条款提供方与对方在合同中的权利义务分配
- [ ] **市场公允价格/条件**：证明格式条款中的价格、条件偏离市场公允水平
- [ ] **弱势地位证明**：如消费者、小微企业等弱势地位的证明

---

## 二、辅助证据材料

- [ ] **原审庭审笔录**：查看原审中是否对格式条款效力进行充分辩论
- [ ] **原审代理词**：查看原审中是否已提出格式条款异议
- [ ] **类案检索报告**：检索类似格式条款纠纷的裁判规则
- [ ] **消费者投诉记录**：如涉及消费者合同，相关投诉处理记录

---

## 三、论证要点

### 3.1 格式条款效力的论证框架

```
一、争议条款属于格式条款
   1. 该条款由XX（提供方）预先拟定
   2. 该条款未与XX（接受方）协商
   3. 该条款在全部/多数合同中重复使用

二、提供方未履行提示说明义务（如主张第496条）
   1. 争议条款未以加粗、下划线、特殊字体等合理方式提示
   2. 提供方未向对方说明条款的法律含义和后果
   3. 对方在签约时未注意到该条款或未能理解其含义

三、争议条款无效（如主张第497条）
   1. 该条款不合理地免除/减轻提供方责任
   2. 该条款不合理地加重对方责任
   3. 该条款不合理地限制/排除对方主要权利

四、对原审判决的批驳
   1. 原审判决第X页认定"该条款有效"，但未审查提示说明义务
   2. 原审判决将举证责任错误地分配给接受方
   3. 正确适用法律应当是《民法典》第496条/第497条
```

### 3.2 举证责任分配

| 争议焦点 | 举证责任 | 常见错误 |
|---------|---------|---------|
| 是否属于格式条款 | 接受方初步证明 → 提供方反驳 | 原审错误认定不属于格式条款 |
| 是否履行提示说明义务 | **提供方**承担举证责任 | 原审错误将举证责任分配给接受方 |
| 是否显失公平 | 接受方承担举证责任 | — |

> ⚠️ **关键**：根据《民法典》第496条，**提供格式条款的一方**应当履行提示和说明义务，并承担相应的举证责任。原审判决如将举证责任分配给接受方，属于法律适用错误。

### 3.3 常见抗辩及应对

| 对方抗辩 | 应对策略 |
|---------|---------|
| "该条款不属于格式条款" | 证明条款由对方预先拟定、未协商、重复使用 |
| "已履行提示说明义务" | 要求对方举证证明（书面记录、特殊标识、签字确认） |
| "对方已签字即表示认可" | 签字不等于知情并认可，需结合提示说明义务综合判断 |
| "该条款是行业通用条款" | 行业通用不等于合法，仍需审查是否显失公平 |

---

## 四、注意事项

1. **区分"格式条款"与"非格式条款"**：并非所有预先拟定的条款都是格式条款，需证明"未与对方协商"这一核心要件
2. **注意《民法典》第496条与第497条的适用顺序**：先审查提示说明义务（第496条），再审查条款效力（第497条）
3. **类案检索重点**：重点检索最高法院和本省高院的格式条款类案，尤其是同一行业的类案
4. **消费者权益保护法**：如涉及消费者合同，可同时引用《消费者权益保护法》第26条关于格式条款的规定
