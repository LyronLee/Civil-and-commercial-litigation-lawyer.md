## 二、原审裁判基本情况

再审申请人________因与被申请人________【案由】纠纷一案，不服________人民法院于________年____月____日作出的（________）________号民事________（判决/裁定/调解书），现依据《中华人民共和国民事诉讼法》第________条之规定，向贵院申请再审。

> 💡 提示：
> - 如对一审、二审裁判均不服，应分别列明一审、二审裁判法院及案号
> - 如仅对二审裁判不服，列明二审裁判法院及案号即可

## 三、再审请求

1. **对原审裁判的意见**：请求撤销________人民法院（________）________号民事________（判决/裁定/调解书）【如二审维持一审结果，应写明一并撤销一审裁判】；
2. **改判请求**：请求提审本案并改判________（具体改判内容，不得超出原审诉请范围）；
3. **诉讼费用承担**：请求判令被申请人承担本案一、二审及再审诉讼费用。

> ⚠️ **重要**：再审请求不能超出原审诉讼请求范围。再审中对于当事人没有争议的事项不予审查和审理，有必要以明确的改判请求确定再审审查范围。

## 撰写要点

1. **文书名称统一**：必须使用"再审申请书"，不得使用"申诉书""申请书"等其他名称
2. **当事人信息完整**：原审地位应包括各当事人在原一审、二审的地位，不应遗漏
3. **精准对应第211条**：必须在申请书中明确指出符合哪一项或哪几项再审事由，法院仅对提出的再审事由进行审查
4. **再审请求三项结构**：对原审裁判的意见 → 改判请求 → 诉讼费用承担
5. **观点先行**：在一级大标题提出观点，再展开论述；新证据应醒目标注
6. **引用判决原文**：指出原审判决具体哪一页、哪一段有误，再加以批驳
7. **可视化辅助**：复杂法律关系或核心证据可附图表（彩色打印、单面打印、左上角装订）
8. **期限严格**：判决/裁定生效后6个月内提出；依据第211条第1、3、12、13项申请的，自知道或应当知道之日起6个月内
9. **禁止重复申请**：被驳回后，不得再次申请再审
10. **不停止执行**：再审申请不停止原判决、裁定的执行

## 附录：《民事诉讼法》第211条再审事由对照表

| 项 | 再审事由 | 常见适用场景 |
|---|---------|------------|
| 第1项 | 有新的证据，足以推翻原判决、裁定的 | 原审庭审结束后新发现的证据；原审申请调取未获准的证据 |
| 第2项 | 原判决、裁定认定的基本事实缺乏证据证明的 | 关键事实无任何证据支撑 |
| 第3项 | 原判决、裁定认定事实的主要证据是伪造的 | 对方提交的证据系伪造 |
| 第4项 | 原判决、裁定认定事实的主要证据未经质证的 | 关键证据未在庭审中质证 |
| 第5项 | 对审理案件需要的主要证据，当事人因客观原因不能自行收集，书面申请人民法院调查收集，人民法院未调查收集的 | 申请法院调取证据被无理由拒绝 |
| 第6项 | 原判决、裁定适用法律确有错误的 | 引用法条错误、遗漏法律适用、违背司法解释 |
| 第7项 | 审判组织的组成不合法或者依法应当回避的审判人员没有回避的 | 合议庭组成违法、应回避未回避 |
| 第8项 | 无诉讼行为能力人未经法定代理人代为诉讼或者应当参加诉讼的当事人，因不能归责于本人或者其诉讼代理人的事由，未参加诉讼的 | 遗漏必要共同诉讼当事人 |
| 第9项 | 违反法律规定，剥夺当事人辩论权利的 | 不允许发言、打断陈述、未告知权利 |
| 第10项 | 未经传票传唤，缺席判决的 | 未依法送达开庭传票即缺席判决 |
| 第11项 | 原判决、裁定遗漏或者超出诉讼请求的 | 漏判某项诉请或判决超出诉请范围 |
| 第12项 | 据以作出原判决、裁定的法律文书被撤销或者变更的 | 作为裁判依据的其他裁判/行政行为被撤销 |
| 第13项 | 审判人员审理该案件时有贪污受贿，徇私舞弊，枉法裁判行为的 | 已生效刑事判决或纪律处分确认 |
