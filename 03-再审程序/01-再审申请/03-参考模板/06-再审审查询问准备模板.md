## 二、询问/听证特点

1. **非正式开庭**：由承办法官主持，程序较灵活，通常无合议庭全体成员到庭
2. **重点审查**：围绕再审事由展开，不全面审理案件实体争议
3. **新证据质证**：如有新证据，询问/听证是质证的关键场合
4. **时间较短**：通常1-2小时
5. **可携带可视化材料**：PPT、图表、时间轴等

---

## 三、我方陈述准备（5-10分钟）

### 3.1 陈述结构

1. **开场**（30秒）：简明介绍当事人地位和再审请求
2. **再审事由陈述**（3-5分钟）：逐项说明符合第211条第几项，核心观点先行
3. **新证据说明**（2-3分钟）：如有新证据，重点说明其崭新性、不可归责性和证明力
4. **结论**（30秒）：恳请法院裁定再审

## 五、法官提问预判与应答准备

| 预判问题 | 应答要点 | 禁区/注意事项 |
|---------|---------|------------|
| 为什么原审中没有提交这份证据？ | | 避免承认"故意不提交"或"疏忽大意" |
| 这份证据的来源是什么？ | | 如实说明，不得编造 |
| 你认为原审哪一点认定最关键？ | | 紧扣再审事由，不扩大范围 |
| 如果裁定再审，你希望怎么判？ | | 明确具体，不超出原审诉请 |
| 这份新证据是否足以推翻原判？ | | 论证关联性和证明力，避免夸大 |

## 六、对方可能反驳及应对

| 对方反驳 | 应对策略 |
|---------|---------|
| 该证据不属于"新证据" | 依据《民事诉讼法解释》第386条，论证崭新性和不可归责性 |
| 该证据不足以推翻原判 | 论证证据与要件事实的关联性和证明力 |
| 原审判决适用法律正确 | 引用类案裁判规则，说明原审法律适用错误 |
