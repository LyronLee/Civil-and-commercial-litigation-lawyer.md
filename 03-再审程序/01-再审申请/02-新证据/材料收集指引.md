# 新证据材料收集指引

> 以"有新的证据，足以推翻原判决、裁定"（第211条第1项）申请再审的，需严格按照"三步递进审查法"收集和整理新证据。

---

## 一、新证据的来源与类型

### 1.1 常见新证据来源

| 来源类型 | 示例 | 取得方式 |
|---------|------|---------|
| **第三方机构 newly 出具** | 原审结束后作出的鉴定意见、审计报告、评估报告 | 委托有资质的机构重新鉴定/审计/评估 |
| **行政机关 newly 作出** | 原审结束后行政机关作出的行政决定、批复、登记 | 向行政机关申请或等待行政机关依职权作出 |
| **司法机关 newly 作出** | 原审结束后其他法院作出的生效裁判、刑事判决 | 查询裁判文书网或等待相关案件审结 |
| **新发现的原始证据** | 原审中未发现的合同原件、账册、银行流水 | 在档案、仓库、旧文件中找到 |
| **申请调取未获准的证据** | 原审中申请法院调取但未获准许的证据 | 重新申请调取或通过其他合法途径取得 |
| **客观上无法在原审中取得的证据** | 因疫情、自然灾害、第三方拒绝等原因无法在原审中取得的证据 | 障碍消除后取得 |

### 1.2 新证据的常见形式

| 证据形式 | 适用场景 | 注意事项 |
|---------|---------|---------|
| 书证 | 合同、票据、账册、银行流水等 | 提供原件或经核对的复印件 |
| 物证 | 产品质量问题中的实物、现场痕迹等 | 保管链条完整，提供照片和视频 |
| 视听资料/电子数据 | 录音、录像、微信记录、电子邮件等 | 原始载体保存完好，提供提取过程说明 |
| 证人证言 | 新发现的证人 | 证人出庭作证或提供书面证言并签字 |
| 鉴定意见 | 新作出的鉴定 | 鉴定机构有资质，鉴定程序合法 |
| 勘验笔录 | 新的现场勘验 | 由法院或公证机关进行 |

## 二、新证据收集步骤

### Step 1：确定需要收集的新证据

根据原审裁判结构化分析，确定哪些"要件事实"缺乏证据支撑或被错误认定，进而确定需要收集的新证据方向。

### Step 2：评估取得可能性

| 评估项 | 内容 | 评估结果 |
|--------|------|---------|
| 证据是否存在 | 该证据是否客观存在？ | □ 存在 / □ 不确定 / □ 不存在 |
| 取得难度 | 取得的难度如何？ | □ 容易 / □ 中等 / □ 困难 |
| 取得时间 | 预计需要多长时间？ | ____天 |
| 费用预估 | 预计需要多少费用？ | ____元 |

### Step 3：实施收集

- **委托专业机构**：如鉴定、审计、评估，委托有资质的机构进行
- **申请行政机关出具**：如需要行政机关的决定或证明，提交书面申请
- **自行收集**：如书证、视听资料等，自行收集并妥善保管
- **申请法院调取**：如证据由第三方掌控且无法自行取得，向法院申请调取

### Step 4：固定和保全

- **书证**：复印并加盖"与原件核对无异"章，保留原件
- **视听资料/电子数据**：刻录光盘并封存，保留原始载体
- **物证**：拍照、录像固定，妥善保管实物
- **证人证言**：制作书面证言，证人签字/捺印，附身份证明

## 三、新证据的整理与提交

### 3.1 每份新证据的必备说明

每份新证据需附以下说明：

| 说明项 | 内容 |
|-------|------|
| 证据名称 | |
| 证据形式 | 书证/物证/视听资料/电子数据/证人证言/鉴定意见/勘验笔录 |
| 证据来源 | 证据的原始出处 |
| 取得时间 | 具体的取得日期 |
| 原审未提交原因 | 详细说明为何原审中未能提交 |
| 证明目的 | 该证据要证明什么事实 |
| 对推翻原判的作用 | 该证据如何推翻原审裁判的认定 |

### 3.2 新证据的三步审查自评

| 审查步骤 | 评估结论 | 说明 |
|---------|---------|------|
| 第一步：审实质 | □ 足以推翻 / □ 辅助证明 / □ 不足以推翻 | |
| 第二步：审形式 | □ 具有崭新性和不可归责性 / □ 崭新性较弱 / □ 不具有崭新性 | |
| 第三步：审主观 | □ 不可归责 / □ 部分可归责 / □ 可归责 | |

---

## 四、注意事项

1. **时间紧迫**：特殊4项（新证据属于第1项）的期限是自知道或应当知道之日起6个月，需尽快收集和提交
2. **不可归责性是关键**：如法院认定逾期举证可归责于当事人，即使证据真实也可能被排除
3. **避免"事后制造"**：证据必须是客观存在的，不能为了诉讼而伪造或变造
4. **原始载体保存**：视听资料、电子数据等必须保留原始载体，复印件/刻录件仅作提交用
5. **份数要求**：新证据应按照被申请人及原审其他当事人人数提交相应份数副本
