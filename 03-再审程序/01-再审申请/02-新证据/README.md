# 新证据

本文件夹用于存放拟在再审程序中提交的"新证据"材料。

## 再审"新证据"的认定标准

再审中"新证据"的认定标准**高于二审**，需通过**三步递进审查法**：

### 第一步：审实质——是否足以推翻原审要件事实
- 是否直接指向原审裁判的"要件事实"
- 是否能使原审要件事实陷入"真伪不明"或被直接推翻

### 第二步：审形式——是否具有崭新性和不可归责性
- **崭新性**：原审庭审结束后新发现 / 原审申请调取未获准 / 原审已存在但客观上无法取得
- **不可归责性**：逾期举证不可归责于当事人（第三方掌控、不可抗力、法院程序瑕疵）

### 第三步：审主观——逾期举证是否可归责于当事人
- 可归责（故意或重大过失）→ 证据失权
- 不可归责 → 可认定为新证据

## 应放入的材料

| 类型 | 说明 | 示例 |
|------|------|------|
| 新发现的书证 | 原审庭审结束后才发现或取得 | 新找到的合同、账册、银行流水 |
| 新调取的证据 | 原审中申请调取未获准许 | 工商档案、不动产登记、刑事判决 |
| 新形成的证据 | 原审结束后新形成 | 新的鉴定意见、评估报告 |
| 刑事判决/纪律处分 | 证明枉法裁判或证据伪造 | 审判人员刑事判决书、纪律处分决定 |
| 被撤销的法律文书 | 据以裁判的其他文书被撤销 | 原裁判依据的行政决定被撤销 |

## 使用方式

1. 将新证据原件/复印件放入本文件夹
2. 每份新证据需准备单独的"新证据说明"（名称、来源、取得时间、原审未提交原因、推翻作用分析）
3. AI 将评估每份证据的三步审查结论

## 注意事项

- ⚠️ **非"新证据"的材料，再审法院原则上不予采纳**
- 新证据应按被申请人及原审其他当事人人数提交相应份数副本
- 新证据在再审文书中应**醒目标注**（加粗、醒目颜色）
- 新证据类再审案件，被申请人**可能需要交纳**案件受理费
