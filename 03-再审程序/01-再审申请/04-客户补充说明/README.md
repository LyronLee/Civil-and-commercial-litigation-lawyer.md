# 客户补充说明

本文件夹用于存放客户在再审申请阶段提供的补充说明材料，帮助 AI 更全面地了解案件背景和客户的再审诉求。

## 可放入的材料

| 材料类型 | 说明 |
|---------|------|
| **客户情况说明** | 客户对案件事实的补充描述、对原审判决的具体异议点 |
| **类案检索报告** | 律师或客户自行检索的类似案例（如以"适用法律错误"为由申请再审） |
| **律师初步分析意见** | 律师对再审可行性的初步评估意见 |
| **专家意见/鉴定材料** | 支持再审事由的专家意见、补充鉴定报告等 |
| **其他补充材料** | 任何有助于说明再审事由的补充文件 |

## 使用方式

1. 在唤起 AI 前，将补充材料放入本文件夹
2. AI 会读取本文件夹中的全部材料，结合 `01-原审卷宗/` 和 `02-新证据/` 进行综合分析
3. 如材料与卷宗内容冲突，AI 会向用户确认以何者为准

## 注意事项

- 本文件夹为**可选输入**，如无可不创建
- 类案检索报告如已放入，AI 会在再审申请书中引用相关案例
- 涉及隐私的信息（身份证号、银行账户等）建议脱敏处理
