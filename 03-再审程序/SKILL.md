# 03 再审程序 SKILL

## 触发条件

用户表达以下意图时触发：
- "我要申请再审"
- "对生效判决不服，想再审"
- "看看能不能再审"
- "再审申请书怎么写"
- "收到驳回再审申请裁定，下一步怎么办"

## 前置检查（触发本SKILL前必须完成）

| 检查项 | 合格标准 | 不满足时的替代路径 |
|--------|---------|-------------------|
| 判决/裁定/调解书是否已生效 | 已生效 | 未生效 → 推荐 `02-二审程序/` |
| 是否在再审申请期限内 | 生效后 **6个月内**（除斥期间） | 超期 → 推荐检察监督路径 |
| 是否已有再审裁判 | 未经再审审理 | 已再审 → 推荐检察监督路径 |
| 是否有明确的再审事由 | 判决/裁定：符合《民事诉讼法》**第二百一十一条**13种情形之一；调解书：符合**第二百一十二条**（违反自愿原则或内容违法） | 无理由 → 客观告知成功率极低 |

> ⚠️ **期限不可延长**：再审申请期限是**除斥期间**，不存在中止、中断、延长。但依据第二百一十一条第（一）（三）（十二）（十三）项申请的，自知道或应当知道之日起6个月内提出。

> ⚠️ **调解书特殊检查**：调解书仅适用两种再审事由——（1）违反自愿原则；（2）调解协议内容违反法律。不适用第二百一十一条的13种情形。

## Step 0：条件确认与现有材料检测

### 0.1 向用户确认

- 原审是几审？（一审/二审）
- 判决/裁定/调解书的生效日期？
- 是否已有再审经历？
- 主张的再审事由是什么？（对应第二百一十一条第几项/第二百一十二条哪种情形）
- 是否有新证据？

### 0.2 关键期限计算

```
判决/裁定生效日期: ____年____月____日
+ 6个月除斥期间
= 再审申请截止日期: ____年____月____日

知道或应当知道日期: ____年____月____日
+ 6个月
= 特殊截止日期: ____年____月____日
```

## Step 1：阶段路由

```
条件确认完成
    │
    ├─ 判决/裁定未生效 ───────────────→ 推荐 02-二审程序/
    │
    ├─ 超过6个月期限，且不符合特殊4项 ─→ 推荐检察监督路径
    │                                    ⚠️ 期限：驳回裁定/再审裁判生效之日起2年
    │                                    依据：《人民检察院民事诉讼监督规则》第20条
    │
    ├─ 已有再审裁判 ──────────────────→ 推荐检察监督路径
    │
    ├─ 尚未提交再审申请 ──────────────→ 进入 01-再审申请/
    │
    ├─ 法院已裁定再审 ────────────────→ 进入 02-再审审理/
    │
    ├─ 再审审理已结束 ────────────────→ 进入 03-收官/
    │                                    （分析再审裁判效力，选择后续路径）
    │
    └─ 法院已驳回再审申请 ────────────→ 推荐检察监督路径
         ⚠️ 前提：符合《民事诉讼法》第二百二十条规定的3种情形之一
              （1）法院驳回再审申请
              （2）法院逾期未对再审申请作出裁定
              （3）再审判决、裁定有明显错误
```

---

## Step 2：替代路径详解

### 路径A：超期 → 检察监督

如再审申请期限已过（且不符合第211条第1/3/12/13项的特殊期限）：
- **期限**：驳回再审申请裁定或再审判决、裁定发生法律效力之日起 **2年内**
- **性质**：不变期间，不适用中止、中断、延长
- **受理机关**：作出生效裁判的法院的同级人民检察院
- **3种前提**（须符合其一）：
  1. 法院驳回再审申请
  2. 法院逾期未对再审申请作出裁定
  3. 再审判决、裁定有明显错误
- **检察院审查期限**：3个月内审查终结
- **结果**：提请抗诉 / 发出再审检察建议 / 不支持监督申请

### 路径B：未生效 → 上诉

如判决/裁定尚未生效：
- 告知用户应在判决书送达之日起15日/裁定书10日内上诉
- 引导至 `02-二审程序/`

### 路径C：已再审 → 检察监督

如案件已经过再审审理：
- 不得再次申请再审
- 引导至检察监督路径

---

## Step 3：阶段衔接

### 3.1 从 01-再审申请 到 02-再审审理

如法院裁定再审：
- 引导用户进入 `02-再审审理/`
- 说明：再审审理配备独立完整的四阶段工作流，直接按步骤执行
- 法律上按原审程序审理（第218条）：原一审→按一审程序审理；原二审→按二审程序审理
- 特别提示：再审按一审程序审理的，裁判可以上诉；按二审程序审理的，裁判为终审裁判（第218条）

### 3.2 从 02-再审审理 到 03-收官

如再审审理结束（收到再审裁判文书）：
- 引导用户进入 `03-收官/`
- 说明核心任务：分析再审裁判效力 → 选择后续路径 → 退费 → 结案复盘
- 提示后续路径选择：
  - 按一审程序再审 + 不服 → 15 日内上诉 → `02-二审程序/`
  - 按二审程序再审 + 维持原判 → 2 年内检察监督
  - 再审改判 + 原判已执行 → 执行回转
  - 发回重审 → `01-一审程序/`

### 3.3 从 01-再审申请 到 检察监督

如法院驳回再审申请：
- 告知用户可在驳回裁定发生法律效力之日起2年内向同级检察院申请监督
- 提供检察监督申请的基本指引（参见路径A）

## 质量检查清单（再审程序通用版）

| 序号 | 检查项 | 合格标准 |
|------|--------|---------|
| 1.1 | 法条引用准确性 | 引用2023年版《民事诉讼法》条号，与官方文本一致 |
| 1.2 | 期限计算正确 | 6个月除斥期间/2年检察监督期限计算无误，起算点正确 |
| 1.3 | 再审事由匹配 | 判决/裁定适用第211条13种情形；调解书适用第212条2种情形 |
| 1.4 | 当事人地位标注 | 各方当事人的原审地位（一审+二审）已完整标注 |
| 1.5 | 再审请求三项结构 | 对原审裁判意见 + 改判请求 + 诉讼费用承担 |
| 1.6 | 新证据三步审查 | 已通过"审实质→审形式→审主观"三步递进审查 |
| 1.7 | 管辖法院正确 | 原则上上一级法院；人多/公民案件可原审法院（第210条） |
| 1.8 | 材料完整性 | 再审申请书、身份证明、原审裁判文书、证据材料、送达地址确认书齐全 |
| 1.9 | 内部文件隔离 | 开庭提纲、庭审纪要、策略分析等内部文件已标注"内部使用，不提交法院" |
| 1.10 | 禁止重复申请 | 已确认本案未经再审审理，不存在重复申请 |

## 关键期限提醒

> 🔴 **再审申请期限**：判决/裁定生效之日起 **6个月**（除斥期间，不可延长）
> 
> 🔴 **特殊4项期限**：依据第211条第（一）（三）（十二）（十三）项申请的，自知道或应当知道之日起 **6个月**
> 
> 🔴 **检察监督期限**：驳回再审申请裁定或再审判决、裁定发生法律效力之日起 **2年**（不变期间）
> 
> ⚠️ **关键期限到达前 3 日提醒用户**，超期可能导致失权
