# 参考模板

本文件夹存放再审程序阶段三（收官）的参考模板，供 AI 生成阶段产出时参照使用。

## 模板清单

| 序号 | 模板名称 | 用途 |
|------|---------|------|
| 01-执行回转申请书模板.md | 再审改判推翻已执行判项时 | 申请法院责令取得财产的人返还 |
| 02-退费申请书模板.md | 再审裁判生效后退诉讼费 | 再审案件原则免交，按改判结果调整 |
| 03-结案复盘模板.md | 案件终局后的复盘总结 | 原审问题诊断、再审经验沉淀、AI分析+人工补充 |

## 使用方式

1. 本文件夹的模板为**参照模板**，AI 在执行 `SKILL.md` 工作流时按三个时点独立触发
2. 用户无需手动填写模板，将再审裁判文书放入 `02-法院来文/` 后，唤起 AI 即可
3. 如需了解模板格式，可直接打开查看

## 工作流分支

| 触发时机 | 生成分支 | 输出文件 |
|---------|---------|---------|
| 收到再审裁判后 | 再审裁判应对 | 裁判分析 + 后续文书（视结果而定） |
| 再审裁判生效后 | 诉讼费退费 | `05-阶段产出/02-退费申请书.md` |
| 案件终局后 | 结案与复盘 | `05-阶段产出/03-结案报告.md` + `04-复盘总结.md` |

## 后续路径选择

| 再审结果 | 后续路径 |
|---------|---------|
| 按一审程序再审 + 不服 | → **02-二审程序/**（15日内上诉） |
| 按二审程序再审 + 维持原判 | → 检察监督路径（2年内） |
| 再审改判 + 原判已执行 | → 执行回转 |
| 发回重审 | → **01-一审程序/** 重新审理 |
| 调解结案 | → 调解书生效 → 对方不履行 → **04-执行程序/** |

## 产出位置

AI 根据本阶段 SKILL.md 生成的文件保存在 `05-阶段产出/` 中。

## 注意事项

- 再审裁判效力判定是核心：按一审程序再审的可上诉；按二审程序再审的为终审
- 检察监督是最后的司法救济，期限 **2年**（不变期间）
- 再审复盘的核心特色是"原审问题诊断"，从纠错角度审视整个诉讼过程
