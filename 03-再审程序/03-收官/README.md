# 03-再审程序 / 03-收官

再审程序的庭后收尾与终局处理阶段。再审的收官与一审、二审有本质区别——再审裁判的效力和后续路径取决于再审审理适用的程序（按一审程序审理的可上诉，按二审程序审理的为终审），且再审改判后可能涉及执行回转等复杂问题。

## 目录结构

| 子目录 | 用途 |
|--------|------|
| `01-参考模板/` | 再审收官阶段可用的参考模板 |
| `02-法院来文/` | 再审裁判文书、生效证明等 |
| `05-阶段产出/` | AI 生成的终局文书 |

## 核心任务

再审收官阶段包含三个核心任务：

1. **再审裁判应对**：根据再审裁判结果确定后续路径（上诉 / 执行回转 / 检察监督 / 结案）
2. **诉讼费退费**：办理再审案件的诉讼费退费手续
3. **结案复盘**：案件终局后完成结案归档和复盘总结

## 与其他阶段的关系

```
02-再审审理 → [再审裁判] → 03-收官
                    │
                    ├─ 按一审程序再审 → 裁判可上诉 → 02-二审程序/
                    ├─ 按二审程序再审 + 维持原判 → 检察监督路径
                    ├─ 按二审程序再审 + 改判 → 执行程序/ 或 对方申请再审
                    ├─ 发回重审 → 01-一审程序/
                    └─ 调解结案 → 执行程序/
```

## 特别注意事项

- 再审裁判是否可上诉，取决于再审适用的审理程序，而非原审程序
- 再审改判后，已执行完毕的判项需要执行回转
- 再审案件原则上免交受理费，但当事人有新证据的除外
- 按二审程序再审维持原判的，当事人仍可申请检察监督
