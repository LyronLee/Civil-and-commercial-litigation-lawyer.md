# 退费材料

本文件夹用于存放再审案件诉讼费退费相关的凭证和申请材料。

## 可放入的材料

| 材料类型 | 说明 |
|---------|------|
| **原审缴费凭证** | 一审/二审的诉讼费交纳通知书、缴费凭证复印件 |
| **再审缴费凭证** | 如当事人实际预交了再审受理费，提供缴费凭证 |
| **收款账户信息** | 退费收款账户（开户名、开户行、账号） |
| **退费申请回执** | 向法院提交退费申请后的回执或受理凭证 |
| **法院退费通知** | 法院出具的退费到账通知或相关文书 |

## 使用方式

1. 案件终局后，如需办理退费，将相关凭证放入本文件夹
2. AI 在生成《退费申请书》时会读取本文件夹中的缴费凭证和收款账户信息
3. 退费完成后，将法院退费通知归档至此

## 注意事项

- 再审案件**原则上免交**受理费，但当事人有新证据的除外
- 如再审改判，按改判结果确定诉讼费承担方式
- 各地法院退费流程不同，具体以受理法院要求为准
- 收款账户信息必须准确，建议提供银行开户许可证或银行卡复印件
