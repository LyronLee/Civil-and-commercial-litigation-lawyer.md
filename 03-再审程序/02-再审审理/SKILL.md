# 02 再审审理 SKILL

> ⚠️ 依据2023年修正《民事诉讼法》（2024年1月1日施行）。旧版条号已全面变更，详见根目录SKILL.md法条对照表。
> 
> 本阶段配备**独立完整的四阶段工作流**。法律上再审按原审程序审理（第218条），但本SKILL不再引导AI引用其他程序框架，而是直接提供再审审理所需的全部步骤和规则。

---

## 一、程序确定

### 1.1 审理程序规则（《民事诉讼法》第二百一十八条）

| 原审程序 | 再审审理程序 | 裁判效力 |
|---------|-------------|---------|
| 一审（未上诉生效） | 按照第一审程序审理 | **可以上诉** |
| 二审（终审） | 按照第二审程序审理 | **终审裁判** |
| 上级法院提审 | 按照第二审程序审理 | **终审裁判** |

> ⚠️ **重要**：再审按第一审程序审理的，裁判可以上诉；按第二审程序审理或上级法院提审的，裁判为终审裁判。

### 1.2 启动再审的三种方式

| 启动方式 | 法律依据 | 特点 |
|---------|---------|------|
| **当事人申请再审** | 第210-216条 | 最常见；法院审查后裁定再审 |
| **法院依职权再审** | 第219条 | 各级法院院长提交审判委员会讨论决定；最高人民法院/上级法院提审或指令再审 |
| **检察院抗诉/检察建议** | 第219-223条 | 抗诉必然启动再审；检察建议由法院决定是否采纳 |

> 三种启动方式的程序起点不同，但审理阶段的规则基本一致。

## 二、阶段一：战略分析

### 2.1 材料收集

读取以下材料（位于 `01-原审及再审申请材料/`）：

| 材料类型 | 来源 | 用途 |
|---------|------|------|
| 法院再审裁定书 | 01-原审及再审申请材料/ | 确认再审审理程序（一审/二审） |
| 再审申请书/抗诉书 | 同上 | 明确再审事由和请求范围 |
| 原审裁判文书 | 同上 | 了解原审认定和判决结果 |
| 原审起诉状/答辩状 | 同上 | 了解原审双方主张 |
| 原审证据目录 | 同上 | 了解原审举证情况 |
| 新证据（如有） | 同上 | 再审审查的核心材料 |
| 类案检索报告 | 同上 | 支撑法律适用观点 |

### 2.2 程序核对

1. **确认再审审理程序**：按一审还是二审程序审理
2. **确认当事人地位**：
   | 原审地位 | 再审地位 |
   |---------|---------|
   | 原告/上诉人 | 再审申请人 |
   | 被告/被上诉人 | 被申请人 |
   | 第三人 | 原审第三人 |
3. **确认裁判效力预判**：可上诉 / 终审
4. **确认是否中止执行**（第217条）：一般裁定再审后中止执行；追索赡养费等案件可以不中止

### 2.3 再审事由聚焦分析

> ⚠️ **审理范围限定**：再审审理范围限于再审申请书中载明的再审事由（当事人申请再审时）或抗诉书/检察建议载明的范围（检察院启动时）。

**操作要点**：
1. 从再审申请书/抗诉书中提取全部再审事由
2. 对每一项事由分析：原审哪里错了 → 正确应当如何 → 证据是否充分
3. 对当事人已经认可或没有争议的事实，标记为"不再审查"
4. 不得引入新的、与再审事由无关的争议

### 2.4 策略方案书

生成《再审审理策略方案书》，包含：
- 再审事由清单及优先级排序
- 原审错误点分析（引用原审判决原文+批驳）
- 新证据三性分析（如有）
- 争议焦点预判
- 改判方案及合理性论证
- 裁判效力预判及后续路径准备

## 三、阶段二：材料制备

### 3.1 再审起诉状/答辩状

**如我方为再审申请人**：
- 围绕再审事由重新组织起诉状（不是原审起诉状的简单重复）
- 诉讼请求：对原审裁判意见 + 改判请求 + 诉讼费用承担
- 必须聚焦再审事由，不扩大范围

**如我方为被申请人**：
- 围绕再审申请书逐条反驳
- 强调原审判决认定事实清楚、适用法律正确、程序合法
- 重点阻击新证据（论证不属于新证据或不足以推翻原判）

> 按原审程序是一审的，起诉状格式参照一审起诉状但内容聚焦再审事由；按原审程序是二审的，参照二审上诉状/答辩状格式。

### 3.2 证据目录

| 证据类别 | 证明目的 | 备注 |
|---------|---------|------|
| 新证据 | 支持再审事由 | 醒目标注，最先出示 |
| 原审关键证据 | 证明原审错误 | 重新组织，聚焦再审角度 |
| 法律依据 | 支撑改判方案 | 法条+司法解释+指导性案例 |

> 新证据在法庭调查中**最先出示**，所有文书中新证据应**醒目标注**（加粗、醒目颜色）。

### 3.3 类案检索补充

- 如庭审中发现新的法律适用争议，补充类案检索
- 引用指导性案例的，注明"应当参照"
- 引用其他类案的，注明"可以参考"

## 四、阶段三：庭审交锋

### 4.1 开庭提纲

生成《再审开庭提纲》（内部使用），包含：
1. **程序核对**：另行组成合议庭确认、回避权利告知
2. **审理范围确认**：当庭确认再审审查范围限于再审事由
3. **新证据出示计划**：出示顺序、核心说明要点、预计时长
4. **法庭调查要点**：我方陈述、发问设计、法官发问预判
5. **法庭辩论结构**：再审事由充分性论证 → 原审错误具体说明 → 改判合理性
6. **关键法条准备**：法条内容摘要+适用场景
7. **突发应对预案**：对方新证据、对方变更主张、法官倾向性等

> ⚠️ **另行组成合议庭**（第218条）：原审审判人员不得参与再审合议庭。如交原审法院再审，必须另行组成合议庭。

### 4.2 庭审纪要

开庭后24小时内生成《再审庭审纪要》，包含：
- 程序事项记录（回避、身份核对、材料接收）
- 审理过程摘要（各方陈述、新证据出示、法官询问）
- 法庭归纳的争议焦点
- 法官关注重点与倾向性表态
- 法官口头要求（期限、补充材料等）
- 庭审初步评估（有利/不利因素、裁判结果预判）
- 待办事项清单

### 4.3 补充代理词

庭审结束后3-5日内（或法官指定时间内）提交书面代理词：
- 围绕再审事由展开，引用原审判决原文加以批驳
- 新证据作为首要论证对象（如以新证据申请再审）
- 类案引用与本案对应
- 改判方案明确具体，不超出原审诉讼请求范围

> 抗诉案件调解达成协议的，应及时通知提出抗诉的人民检察院（第219条相关规则）。

## 五、阶段四：收官

### 5.1 裁判效力分析

收到再审裁判文书后，分析：

| 审理程序 | 裁判效力 | 救济途径 |
|---------|---------|---------|
| 按第一审程序再审 | 可以上诉 | 15日内上诉至上一级法院 |
| 按第二审程序再审 | 终审裁判 | 2年内检察监督 |
| 上级法院提审 | 终审裁判 | 2年内检察监督 |

### 5.2 后续路径选择

| 情形 | 后续路径 | 期限 |
|------|---------|------|
| 按一审再审 + 不服裁判 | 上诉 | 判决15日 / 裁定10日 |
| 按二审再审 + 维持原判 | 检察监督 | 2年（不变期间） |
| 再审改判 + 原判已执行 | 执行回转 | 及时申请 |
| 发回重审 | 回到一审程序重新审理 | — |

### 5.3 退费处理

- 再审案件**原则上免交**受理费
- 如当事人实际预交了费用，按改判结果申请退费
- 因新证据等情形由被申请人承担费用的，按判决执行

### 5.4 结案复盘

生成《再审结案复盘》，包含：
- 再审裁判结果及效力判定
- 原审问题诊断（从"纠错"角度总结）
- 再审经验总结
- 后续路径建议及期限提醒
- 案卷归档清单

## 六、再审审理的特殊规则

### 6.1 另行组成合议庭（第218条）

- 人民法院审理再审案件，**应当另行组成合议庭**
- 原审审判人员不得参与再审合议庭
- 当事人可申请原审审判人员回避（虽已另行组成，但仍可主张其他回避事由）

### 6.2 审理范围限定

- 限于再审申请书中载明的再审事由
- 对当事人已经认可或没有争议的事实，不再审查
- 不得引入新的、与再审事由无关的争议

### 6.3 新证据的突出展示

- 新证据在法庭调查中**最先出示**
- 所有文书中新证据应**醒目标注**
- 新证据的证明力需最充分论证

### 6.4 抗诉/检察建议案件的特殊规则

| 特殊规则 | 内容 |
|---------|------|
| 检察院出庭 | 抗诉案件应当派员出庭；检察建议案件可以派员出庭 |
| 30日裁定再审 | 接受抗诉的法院应在30日内作出再审裁定（第222条） |
| 调解通知 | 抗诉案件调解达成协议的，应及时通知提出抗诉的检察院 |
| 调查核实 | 检察院可以向当事人或案外人调查核实（第221条） |

### 6.5 再审调解

- 再审案件**可以调解**（《民事诉讼法解释》第410条）
- 调解书送达后，原判决、裁定视为撤销
- 部分当事人达成调解协议的，不影响对其他当事人的审理

### 6.6 发回重审

- 按照第二审程序再审的案件，如认为原判决认定基本事实不清的，可以发回重审
- 发回后**另行组成合议庭**，按第一审程序审理
- 按第一审程序审理的，裁判可以上诉

### 6.7 是否中止执行（第217条）

| 情形 | 处理方式 |
|------|---------|
| 一般情形 | 裁定再审后，**裁定中止**原判决、裁定、调解书的执行 |
| 追索赡养费、扶养费、抚养费、抚恤金、医疗费用、劳动报酬等案件 | **可以不中止**执行 |

> 被申请人提供担保的，也可以不中止执行。

## 七、质量检查清单（再审审理阶段）

| 序号 | 检查项 | 合格标准 |
|------|--------|---------|
| 2.1 | 另行组成合议庭 | 已确认再审合议庭与原审审判人员无重合 |
| 2.2 | 审理范围限定 | 所有材料聚焦再审事由，未引入无关争议 |
| 2.3 | 新证据突出展示 | 新证据在法庭调查中首先出示，文书中醒目标注 |
| 2.4 | 当事人地位正确 | 各方当事人的再审地位标注准确 |
| 2.5 | 审理程序正确 | 原一审→按一审程序审理；原二审→按二审程序审理 |
| 2.6 | 裁判效力预判 | 已告知当事人再审裁判是否可上诉 |
| 2.7 | 中止执行处理 | 已申请或确认是否中止执行 |
| 2.8 | 抗诉案件特殊规则 | 如系抗诉案件，已确认检察院派员出庭安排 |
| 2.9 | 代理词聚焦再审 | 代理词围绕再审事由展开，引用原审判决原文批驳 |
| 2.10 | 庭审记录完整 | 庭审纪要记录了法官关注重点、口头要求、倾向性表态 |
| 2.11 | 独立工作流执行 | 未引导用户引用其他程序SKILL.md，全程按本SKILL执行 |

## 九、通用输出规范

- 所有文书按再审审理的独立结构生成，不再引用其他程序框架
- 引用原审判决原文时标注页码或段落，增强针对性
- 新证据必须最先出示、最充分论证，文书中醒目标注
- 代理词围绕再审事由展开，不得超出再审审查范围
- 所有期限提醒以法院指定为准，法院未指定的适用法定期限
- 所有文件生成后标注"初稿，需律师审核"
