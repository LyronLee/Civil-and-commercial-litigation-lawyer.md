# 02 再审审理：按原审程序重新审理

> 🎯 **阶段定位**：法院裁定再审后，案件进入再审审理阶段。再审按原审程序重新审理，可能是一审程序，也可能是二审程序。
>
> **核心原则**：本阶段配备**独立完整的四阶段工作流**，直接按步骤执行即可。法律上再审按原审程序审理（民诉法第218条），但工作流本身不再依赖其他程序框架。

## 审理程序确定

根据原审级别确定审理程序：

| 原审级别 | 再审审理程序 | 裁判效力 |
|---------|-------------|---------|
| 原一审案件（未上诉） | 一审程序 | **可以上诉** |
| 原一审案件（已上诉，二审生效） | 二审程序 | **终审裁判** |
| 原二审案件 | 二审程序 | **终审裁判** |

### 审理程序的特殊规则

1. **另行组成合议庭**：原审审判人员不得参与再审合议庭
2. **审理范围限定**：限于再审申请书中载明的再审事由
3. **新证据突出展示**：新证据最先出示、醒目标注、充分论证
4. **中止执行**：一般裁定再审后中止执行；特殊案件（赡养费等）可以不中止

## 🚀 使用流程

### 第一步：确认再审裁定

收到法院的再审裁定书，确认：
- 法院已裁定再审
- 再审按一审程序还是二审程序审理
- 是否追加/变更当事人

### 第二步：复制材料 → `01-原审及再审申请材料/`

将以下材料放入本文件夹：
- 原审全部卷宗
- 再审申请材料（再审申请书、法院裁定书）
- 新证据（如有）

### 第三步：放入补充材料（如有）→ `02-补充材料/`

再审审理阶段新发现的证据或其他补充材料，放入此处。此步骤为可选。

### 第四步：唤起 AI → "准备再审审理"

> **"法院已经裁定再审了，请帮我准备再审审理。材料在01-原审及再审申请材料里。原审是【一审/二审】案件，再审按【一审/二审】程序审理。"**

AI 会：
1. 读取再审裁定书和原审卷宗
2. 确定审理程序（一审/二审）
3. 聚焦再审事由，生成再审审理策略方案书
4. 按独立四阶段工作流生成全部材料
5. 提示再审审理的特殊注意事项

## 四阶段工作流概览

| 阶段 | 任务 | 核心产出 |
|------|------|---------|
| **阶段一：战略分析** | 程序确定、材料收集、再审事由聚焦分析 | 《再审审理策略方案书》 |
| **阶段二：材料制备** | 起诉状/答辩状、证据目录、类案检索补充 | 诉讼文书+证据目录 |
| **阶段三：庭审交锋** | 开庭提纲、庭审纪要、代理词 | 内部文件+庭审记录 |
| **阶段四：收官** | 裁判分析、后续路径、退费、结案复盘 | 《结案复盘》+路径建议 |

## 再审审理的特殊注意事项

| 事项 | 说明 |
|------|------|
| **审理范围** | 围绕再审请求和再审事由展开，不是全面重审 |
| **新证据** | 可以提交新证据，法院会进行质证和认证；新证据最先出示、醒目标注 |
| **当事人变更** | 法院可能追加或变更当事人 |
| **另行组成合议庭** | 原审审判人员不得参与再审合议庭 |
| **裁判效力** | 按一审程序审理的再审 → 可以上诉；按二审程序审理的再审 → 终审 |
| **调解** | 再审中可以调解，调解书生效后原判决视为撤销 |
| **抗诉案件** | 检察院应当派员出庭；30日内裁定再审 |
