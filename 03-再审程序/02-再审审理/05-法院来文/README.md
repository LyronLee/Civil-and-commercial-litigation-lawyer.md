# 法院来文

> 存放再审审理阶段法院发送的各类法律文书，包括再审裁定书、传票、通知书等。

---

## 常见法院来文类型

| 文书类型 | 触发时机 | 注意事项 |
|---------|---------|---------|
| 再审裁定书 | 法院决定再审时 | 确认再审审理程序（一审/二审） |
| 传票 | 安排开庭时 | 注意开庭时间、地点、审判组织 |
| 举证通知书 | 指定举证期限时 | 注意举证期限，及时提交证据 |
| 裁判文书 | 再审审理终结时 | 确认裁判效力（可上诉/终审） |

---

## 期限提取

收到法院来文后，应立即提取其中指定的所有期限，并更新《关键期限跟踪表》。
