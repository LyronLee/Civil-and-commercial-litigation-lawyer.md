## 三、审理重点预判

### 3.1 本案再审审查范围

> ⚠️ **重要**：再审审理范围限于再审申请书中载明的再审事由。对于当事人已经认可或没有争议的事实，不再审查。

本案再审审查的核心问题：________

### 3.2 因"新证据"再审的审查重点

| 审查维度 | 法官可能关注的问题 | 我方核心论证 |
|---------|-----------------|------------|
| 真实性 | 新证据是否真实、有无篡改 | |
| 取得时间 | 是否原审庭审结束后新发现 | |
| 关联性 | 是否直接指向原审要件事实 | |
| 证明力 | 是否足以推翻原审认定 | |
| 不可归责性 | 原审未提交的客观原因 | |

### 3.3 因"适用法律错误"再审的审查重点

| 审查维度 | 法官可能关注的问题 | 我方核心论证 |
|---------|-----------------|------------|
| 法条引用 | 原审引用的法条是否正确 | |
| 法律解释 | 对争议法律条文的理解是否正确 | |
| 类案规则 | 与最高法院/上级法院类案是否一致 | |
| 司法解释 | 是否遗漏适用相关司法解释 | |

### 3.4 因"认定事实缺乏证据证明"再审的审查重点

| 审查维度 | 法官可能关注的问题 | 我方核心论证 |
|---------|-----------------|------------|
| 基本事实 | 原审认定的哪个基本事实缺乏证据 | |
| 主要证据 | 原审采信的主要证据是否存在问题 | |
| 举证责任 | 举证责任分配是否正确 | |
| 证据链 | 现有证据是否足以支撑原审认定 | |

### 3.5 因"程序违法"再审的审查重点

| 审查维度 | 法官可能关注的问题 | 我方核心论证 |
|---------|-----------------|------------|
| 违法性 | 原审违反了哪项法定程序 | |
| 因果关系 | 程序违法是否可能影响公正裁判 | |
| 损害后果 | 是否实际影响了当事人的诉讼权利 | |

## 八、突发应对预案

| 突发情形 | 应对策略 |
|---------|---------|
| 对方当庭提交新证据 | 申请延期质证/要求复印后书面质证 |
| 对方变更主张/提出反请求 | 指出超出再审范围/申请不予审理 |
| 法官倾向性明显 | 保持冷静，依法陈述，庭后书面补充 |
| 关键证据原件忘带 | 申请庭后补交，说明与复印件一致 |
| 对方代理人未出庭 | 确认是否合法传唤，申请缺席审理 |
| 庭审时间不足 | 申请择期继续开庭，书面补充意见 |

---

> 💡 **再审开庭特别提示**：
> - 再审按一审程序审理的，裁判可以上诉；按二审程序审理的，裁判为终审裁判
> - 再审审理范围限于再审申请书中载明的再审事由
> - 新证据应最先出示、最充分论证，是启动再审的"钥匙"
> - 辩论中应引用原审判决原文后加以批驳，增强针对性
