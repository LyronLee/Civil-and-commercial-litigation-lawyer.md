# 参考模板

本文件夹存放再审程序阶段二（再审审理）的参考模板，供 AI 生成阶段产出时参照使用。

## 模板清单

| 序号 | 模板名称 | 用途 |
|------|---------|------|
| 00-再审开庭提纲模板.md | 再审庭审前内部准备 | 程序核对（另行组成合议庭）、再审重点预判、新证据出示计划、辩论要点 |
| 01-再审庭审纪要模板.md | 再审庭审记录 | 基本信息、程序事项、法庭调查（新证据质证）、法庭辩论、其他安排 |
| 02-再审代理词模板.md | 再审庭后书面意见 | 围绕再审事由展开，引用原审判决原文批驳 |
| 03-再审答辩状模板.md | 被申请人答辩 | 逐条反驳再审事由，维护原审判决，新证据阻击 |

## 使用方式

1. 本文件夹的模板为**参照模板**，AI 在执行 `SKILL.md` 工作流时会自动调用
2. 用户无需手动填写模板，将案件材料按 SKILL.md 要求准备后，唤起 AI 即可
3. 如需了解模板格式，可直接打开查看

## 再审审理的特殊要点

| 特殊规则 | 说明 |
|---------|------|
| 另行组成合议庭 | 原审审判人员不得参与再审合议庭 |
| 审理范围限定 | 限于再审申请书中载明的再审事由 |
| 新证据优先出示 | 新证据在法庭调查中首先出示 |
| 当事人地位变化 | 原告/上诉人→再审申请人；被告/被上诉人→被申请人 |
