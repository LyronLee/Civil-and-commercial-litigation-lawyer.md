# 07-阶段产出

本文件夹用于存放再审审理阶段 AI 生成的工作成果文件。

## 产出文件清单

| 序号 | 文件名 | 说明 |
|------|--------|------|
| 01 | 策略方案书.md | 再审审理策略方案 |
| 02 | 起诉状/答辩状.md | 按原审程序角色确定的诉讼文书 |
| 03 | 证据目录.md | 再审审理证据材料目录 |
| 04 | 开庭提纲.md | 再审庭审发言提纲 |
| 05 | 庭审纪要.md | 再审庭审过程记录 |
| 06 | 代理词.md | 再审庭后代理词 |
| 07 | 类案检索报告.md | 支持再审事由的类案补充检索 |

## 使用方式

AI 根据 `SKILL.md` 工作流生成的文件自动保存于此。用户可直接使用或按需修改。

## 注意事项

- 所有文件生成后标注"初稿，需律师审核后使用"
- 再审审理按原审程序进行（原一审→按一审程序；原二审→按二审程序）
- 另行组成合议庭，原审审判人员不得参与
- 审理范围限于再审申请书中载明的再审事由
- 按一审程序再审的裁判**可以上诉**；按二审程序再审的裁判为终审裁判
