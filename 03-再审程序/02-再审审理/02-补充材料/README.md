# 补充材料

本文件夹用于存放再审审理阶段新增的补充材料，包括法院裁定再审后新发现或新收集的证据、补充说明等。

## 可放入的材料

| 材料类型 | 说明 |
|---------|------|
| **新证据** | 法院裁定再审后新发现或新收集的证据 |
| **补充鉴定/评估报告** | 再审审理期间委托的新的鉴定、评估报告 |
| **法院依职权调取的证据** | 法院依职权调取并送达的新证据 |
| **补充代理意见** | 律师提交的补充书面意见 |
| **其他补充材料** | 再审审理阶段出现的其他新材料 |

## 使用方式

1. 在再审审理过程中，如有新材料产生，放入本文件夹
2. AI 在准备庭审材料时会读取本文件夹中的全部补充材料
3. 新证据应在开庭提纲中规划出示顺序，在庭审中优先出示

## 注意事项

- 本文件夹为**可选输入**，如无可不创建
- 补充材料必须在法定举证期限内提交，逾期可能被法院拒绝接受
- 对方提交的补充材料应放入 `04-对方来文/`，不放入本文件夹
