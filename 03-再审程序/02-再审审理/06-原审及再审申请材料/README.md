# 原审及再审申请材料

本文件夹用于存放原审及再审申请阶段的全部材料，作为再审审理阶段的输入参考。

## 应放入的材料

### 原审材料

| 材料类型 | 说明 |
|---------|------|
| 原审裁判文书 | 一审、二审判决书/裁定书/调解书 |
| 原审起诉状/答辩状 | 了解原审双方主张 |
| 原审证据目录及材料 | 了解原审举证情况 |
| 原审庭审笔录/纪要 | 了解原审庭审争议焦点 |
| 原审代理词 | 了解原审代理观点 |

### 再审申请材料

| 材料类型 | 说明 |
|---------|------|
| 再审申请书 | 明确再审事由和请求 |
| 再审证据目录 | 再审阶段提交的证据 |
| 新证据材料 | 支持再审事由的新证据 |
| 类案检索报告 | 如以适用法律错误为由 |
| 法院再审裁定 | 裁定再审的文书 |

## 使用方式

1. 将原审卷宗和再审申请材料复制到本文件夹
2. AI 将基于这些材料，聚焦再审事由，准备再审审理
3. 所有材料围绕再审事由展开，不引入无关争议

## 注意事项

- 本文件夹材料仅供再审审理阶段参考，不重复生成已在阶段一完成的工作
- 法律上再审按原审程序审理（第218条），但工作流本身独立完整，不再依赖其他程序框架
- 原审已查明且无争议的事实，再审中一般不再审查
