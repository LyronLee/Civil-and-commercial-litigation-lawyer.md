# 03 再审程序：生效裁判的特殊救济

> 🎯 **程序定位**：对已经发生法律效力的判决、裁定、调解书，认为有错误时申请再审。再审是特殊救济程序，**启动门槛高、期限严格**。
>
> 与二审不同，再审不是必然发生的程序。大多数再审申请会被法院裁定驳回。只有符合法定再审事由的案件，才有可能进入再审审理。

---

## 什么时候进入再审程序

| 情形 | 是否适用再审 |
|------|-------------|
| 二审判决已生效，发现新的关键证据 | ✅ 适用 |
| 原判决适用法律确有错误 | ✅ 适用 |
| 审判人员有贪污受贿、枉法裁判行为 | ✅ 适用 |
| 单纯对判决结果不满意 | ❌ 不适用（需有法定事由） |
| 已超过判决生效后 6 个月 | ❌ 不适用（期限已过） |
| 已经过再审审理并作出裁判 | ❌ 一般不再审（一裁终局） |

> ⚠️ **再审 ≠ 三审**：再审不是自动的第三审级，而是对生效裁判的特殊纠错机制。法院对再审申请进行审查后，认为不符合条件的，会裁定驳回再审申请。

## 三阶段流程

### 阶段一：再审申请

**核心任务**：评估是否符合再审条件 → 撰写再审申请书 → 提交法院 → 等待法院裁定

**输入材料**：
- 生效法律文书（判决书/裁定书/调解书）
- 原审全部卷宗（一审+二审）
- 新证据（如有）

**产出文件**：
- 再审申请书
- 再审策略分析
- 新证据评估报告（如有）

**关键期限**：
- 再审申请期限：**判决生效之日起 6 个月内**（硬性 deadline，不可延长）

**可能结果**：
- 法院裁定再审 → 进入阶段二（再审审理）
- 法院裁定驳回 → 可向检察院申请检察监督

---

### 阶段二：再审审理

**核心任务**：按原审程序重新审理案件

**重要规则**：
- 原一审案件的再审 → **按一审程序审理**，作出的裁判可以上诉
- 原二审案件的再审 → **按二审程序审理**，作出的裁判为终审裁判

**工作流特点**：
- 配备独立完整的四阶段工作流，直接按步骤执行
- 法律上按原审程序审理（第218条）：原一审→按一审程序审理；原二审→按二审程序审理
- 再审特殊规则已内嵌于工作流中（另行组成合议庭、审理范围限定、新证据突出展示等）

`02-再审审理/SKILL.md` 提供完整的再审审理操作指引。

---

### 阶段三：收官

**核心任务**：分析再审裁判效力 → 选择后续路径 → 诉讼费退费 → 结案归档与复盘

**关键工作**：
- **再审裁判效力判定**：按一审程序再审的可上诉；按二审程序再审的为终审
- **后续路径选择**：上诉（15 日）/ 检察监督（2 年）/ 执行回转 / 发回重审衔接 / 结案
- **诉讼费退费**：再审案件原则上免交受理费，按改判结果调整
- **结案复盘**：从"纠错"角度诊断原审问题、总结再审经验

## 与相邻程序的衔接

### 从前序程序进入

| 前序程序 | 进入再审的触发条件 |
|---------|-------------------|
| 01-一审程序 | 一审判决生效后，发现符合再审事由 |
| 02-二审程序 | 二审判决生效后，发现符合再审事由 |
| 04-执行程序 | 执行中发现原判决有误，申请再审（一般不停止执行） |

### 再审后的后续路径

| 再审结果 | 后续路径 |
|---------|---------|
| 裁定再审 + 按一审程序改判 | 裁判可上诉（15 日）→ 02-二审程序/ 或 结案 |
| 裁定再审 + 按二审程序改判 | 终审裁判 → 执行回转 或 新裁判执行 → 04-执行程序/ 或 结案 |
| 裁定再审 + 按二审程序维持原判 | 终审裁判 → 检察监督路径 |
| 裁定再审 + 发回重审 | → 01-一审程序/（另行组成合议庭） |
| 裁定驳回再审申请 | → 向检察院申请检察监督 |
| 再审中达成调解 | → 调解书生效 → 04-执行程序/ 或 结案 |

## 再审 vs. 二审 vs. 执行异议之诉

| 对比项 | 二审 | 再审 | 执行异议之诉 |
|--------|------|------|-------------|
| 适用对象 | 未生效的一审判决 | 已生效的判决/裁定/调解书 | 执行程序中的实体权利争议 |
| 启动条件 | 当事人上诉 | 符合法定再审事由 | 对执行异议裁定不服 |
| 期限 | 判决15日/裁定10日 | **6个月** | 收到裁定后15日 |
| 审理范围 | 上诉请求 | 再审请求 | 实体权利争议 |
| 改判难度 | 中 | **高**（维持原判为原则） | 视证据而定 |
| 次数限制 | 一次 | 一般一次 | 一次 |

## 快速开始

### 第一步：评估是否适用再审

问自己三个问题：
1. 判决是否已经生效？（生效后才能申请再审）
2. 是否在判决生效后 6 个月内？（期限不可延长）
3. 是否有明确的再审事由？（民诉法第 211 条规定的 13 种情形之一）

如果三个答案都是"是"，进入第二步。

### 第二步：准备材料

将以下材料放入 `01-再审申请/` 对应位置：
- 生效法律文书
- 原审全部卷宗
- 新证据（如有）

### 第三步：唤起 AI

> **"我要申请再审，请帮我分析是否符合再审条件，起草再审申请书。材料在对应文件夹中。"**

AI 会：
1. 评估是否符合再审条件
2. 确定最优的再审事由
3. 起草再审申请书
4. 提示关键期限

## ❓ 常见问题

**Q1: 再审申请被驳回后还能怎么办？**
> 可以向检察院申请检察监督。检察院审查后，如认为原裁判确有错误，会向法院提出抗诉或再审检察建议。

**Q2: 再审期间原判决还执行吗？**
> 原则上不停止执行。但法院可以裁定中止执行，或当事人提供担保后中止执行。

**Q3: 再审的改判概率有多大？**
> 再审改判率较低。大多数再审申请被驳回，进入再审审理的案件中，相当一部分维持原判。只有在有明确新证据或原判决存在明显法律适用错误时，改判可能性才较高。

**Q4: 我可以同时申请再审和向检察院申诉吗？**
> 可以。向检察院申请检察监督不以再审申请被驳回为前提。但检察院通常会先审查法院是否已经处理过再审申请。
