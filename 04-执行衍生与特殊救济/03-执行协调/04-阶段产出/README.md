# 阶段产出

本文件夹用于存放执行协调阶段 AI 生成的工作成果文件。

## 产出文件清单

| 序号 | 文件名 | 说明 |
|------|--------|------|
| 01 | 执行协调冲突分析报告.md | 分析执行冲突的具体情形 |
| 02 | 协调方案建议书.md | 向上级法院报请协调的文书 |
| 03 | 指定执行申请书.md | 申请指定某一法院执行的文书（如需） |
| 04 | 跟进记录.md | 协调进展跟踪记录 |

## 使用方式

AI 根据 `SKILL.md` 工作流生成的文件自动保存于此。用户可直接使用或按需修改。

## 注意事项

- 所有文件生成后标注"初稿，需律师审核后使用"
- 执行协调由法院内部依职权或依申请启动，无需当事人立案
