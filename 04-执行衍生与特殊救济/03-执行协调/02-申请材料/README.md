# 申请材料

本文件夹用于存放执行协调相关的原始材料。

## 应放入的材料

| 材料类型 | 说明 | 必要性 |
|---------|------|--------|
| 各执行案件信息 | 执行法院、案号、执行标的、执行进展 | 必须 |
| 冲突财产信息 | 被多个法院查封/冻结的财产描述 | 必须 |
| 查封顺位材料 | 各法院查封的时间顺序证明 | 建议 |
| 已提交的协调申请 | 报请共同上级法院协调的文书 | 如有 |

## 使用方式

1. 分析执行冲突的具体情形（冲突法院、冲突财产、冲突原因）
2. 将相关材料放入本文件夹
3. 唤起 AI 协助准备协调方案和报请协调的文书

## 注意事项

- 执行协调由上级法院依职权或依申请进行
- 协调期间，各法院一般应暂停对冲突财产的处置
