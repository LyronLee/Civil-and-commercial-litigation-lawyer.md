# 法院来文

本文件夹用于存放执行回转过程中法院发出的各类文书。

## 应放入的材料

| 材料名称 | 说明 |
|---------|------|
| 受理通知书 | 法院受理执行回转申请的通知 |
| 执行回转裁定 | 法院作出的执行回转裁定 |
| 执行通知书 | 向原申请执行人发出的执行回转通知书 |
| 执行完毕裁定 | 执行回转程序完结的裁定 |
| 财产查控结果 | 法院对原申请执行人财产的查控结果 |

## 使用方式

1. 收到法院文书后，及时存入本文件夹
2. AI 在跟进执行回转进度时会读取本文件夹内容

## 注意事项

- 执行回转立案后，法院会向原申请执行人发出执行通知书
- 如原申请执行人拒不返还，法院可强制执行其财产
- 执行回转不适用执行和解（但可协商折价赔偿方案）
