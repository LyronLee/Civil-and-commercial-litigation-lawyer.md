# 申请材料

本文件夹用于存放执行回转申请所需的原始材料。

## 应放入的材料

| 材料类型 | 说明 | 必要性 |
|---------|------|--------|
| 原执行依据 | 被撤销/变更的判决书/裁定书/调解书 | 必须 |
| 撤销/变更文书 | 证明执行依据被撤销或变更的法律文书（再审判决、撤销裁定等） | 必须 |
| 原执行裁定 | 证明原执行程序已完毕的执行裁定 | 必须 |
| 已执行财产清单 | 原执行中已执行的财产描述及价值 | 必须 |
| 对方财产线索 | 原申请执行人当前的财产状况（便于执行回转） | 建议 |

## 使用方式

1. 确认执行回转条件满足后，将相关材料放入本文件夹
2. 唤起 AI 生成执行回转申请书
3. 执行回转申请提交至原执行法院

## 注意事项

- 执行回转依据：《民事诉讼法》第244条（2023年版）
- 回转标的包括已执行的财产及其孳息
- 如不能返还，应折价赔偿
- 执行回转申请应在执行依据被撤销后及时提出
