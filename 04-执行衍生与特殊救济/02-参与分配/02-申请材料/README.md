# 申请材料

本文件夹用于存放参与分配申请所需的原始材料。

## 应放入的材料

| 材料类型 | 说明 | 必要性 |
|---------|------|--------|
| 执行依据 | 证明债权已取得生效法律文书确认 | 必须 |
| 债权凭证 | 借条、合同、判决书等 | 必须 |
| 被执行人财产状况 | 已知财产线索、查封情况等 | 建议 |
| 其他债权人信息 | 已申请执行的债权人清单（如有） | 可选 |
| 分配方案 | 执行法院作出的分配方案（如已有） | 如有 |

## 使用方式

1. 确认参与分配条件满足后，将相关材料放入本文件夹
2. 唤起 AI 生成参与分配申请书
3. 如对已作出的分配方案有异议，可引导进入 `03-执行异议之诉/04-执行分配方案异议之诉/`

## 注意事项

- 参与分配申请须在**财产执行终结前**提出
- 分配顺序：执行费用 → 优先受偿债权 → 普通债权按比例分配
- 如对分配方案有异议，需先提出异议 + 收到反对意见后，15日内提起分配方案异议之诉
