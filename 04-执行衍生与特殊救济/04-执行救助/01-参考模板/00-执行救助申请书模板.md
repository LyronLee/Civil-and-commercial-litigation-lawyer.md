# 执行救助申请书

> 申请执行人因被执行人无财产可供执行，生活面临急迫困难，可申请执行救助金。

```
申请人：【姓名/名称】
住所地：

被执行人：【姓名/名称】
住所地：

申请事项：
请求贵院依法给予执行救助金人民币【金额】元。

事实与理由：
申请人与被执行人【案由】纠纷一案，业经【法院名称】（【案号】）判决，被执行人应向申请人支付【金额】。案件已进入执行程序（【执行案号】）。

经贵院执行查明，被执行人【无财产可供执行的具体情况】。贵院已于【日期】裁定终结本次执行程序。

申请人因【人身损害/工伤/交通事故等】，【生活困难的具体情况：如医疗费支出、丧失劳动能力、无收入来源等】。被执行人无履行能力，申请人生活面临急迫困难，依据《最高人民法院关于加强和规范人民法院国家司法救助工作的意见》及相关规定，特申请执行救助。

此致
【法院名称】

申请人：【签章】
日期：

附：
1. 身份证明
2. 生效法律文书
3. 终结本次执行裁定书
4. 生活困难证明材料（低保证、医疗费单据等）
```

---

## 重要提示

1. **适用情形**：追索赡养费、扶养费、抚育费、抚恤金、医疗损害赔偿、工伤赔偿、交通事故人身损害赔偿等
2. **前提条件**：被执行人无财产可供执行，申请执行人生活困难
3. **救助金额**：根据困难程度和当地标准确定
4. **审批程序**：法院司法救助委员会审批
5. **注意**：获得救助后，发现被执行人有财产的，仍应继续执行
