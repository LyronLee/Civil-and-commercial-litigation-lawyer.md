# 申请材料

本文件夹用于存放执行救助申请所需的原始材料。

## 应放入的材料

| 材料类型 | 说明 | 必要性 |
|---------|------|--------|
| 终本裁定 | 证明执行案件已终结本次执行 | 必须 |
| 生活困难证明 | 低保证、残疾证、医疗费用证明、收入证明等 | 必须 |
| 被执行人无财产证明 | 法院查控结果、被执行人财产报告等 | 建议 |
| 身份证明 | 申请人身份证/营业执照 | 必须 |
| 生效法律文书 | 判决书/调解书等 | 必须 |

## 使用方式

1. 确认执行救助条件满足后，将相关材料放入本文件夹
2. 提供生活困难证明（如尚未准备，AI 会提示所需材料）
3. 唤起 AI 生成执行救助申请书

## 注意事项

- 执行救助是**司法救助**，非执行程序本身
- 救助金额和标准因地而异，具体以受理法院所在地政策为准
- 救助金一般从法院执行救助专项资金中列支
- 获得救助不意味着放弃对被执行人的追偿权
