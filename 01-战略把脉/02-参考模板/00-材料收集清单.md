# 材料收集清单

> 这份清单帮助你系统性地收集案件材料。逐项对照，缺什么补什么。
>
> ⭐ = 必须项（没有会严重影响案件）
> ○ = 加分项（有了更好，增强说服力）

---

## 一、通用材料（所有民事案件都需要）

### 1. 主体身份材料 ⭐

| 材料名称 | 原告需要 | 被告需要 | 说明 |
|---------|---------|---------|------|
| 本人身份证复印件 | ⭐ | ⭐ | 证明原告/被告身份 |
| 户口本/户籍证明 | ○ | ○ | 涉及亲属关系、继承等案件需要 |
| 企业营业执照复印件 | ⭐ | ⭐ | 公司作为当事人时必须 |
| 法定代表人身份证明 | ⭐ | ⭐ | 公司作为当事人时必须 |
| 授权委托书 | ○ | ○ | 委托他人代理时需要 |

### 2. 合同/协议类材料 ⭐

| 材料名称 | 说明 |
|---------|------|
| 书面合同/协议正本 | 双方签字盖章的原件或清晰复印件 |
| 补充协议/变更协议 | 后续签订的修改或补充文件 |
| 订单/确认单/备忘录 | 虽未签正式合同，但构成合意的文件 |
| 口头约定的佐证 | 录音、聊天记录中关于约定内容的确认 |

### 3. 往来沟通记录 ○

| 材料名称 | 说明 |
|---------|------|
| 微信聊天记录 | 导出完整聊天，包含时间、双方头像和昵称 |
| 短信记录 | 截图保存，保留发送时间 |
| 电子邮件 | 保留完整邮件，包含发件人、收件人、时间 |
| 通话录音 | 转写成文字稿，标注通话时间和双方身份 |
| 会议纪要/往来函件 | 正式沟通的书面记录 |

### 4. 款项往来凭证 ⭐

| 材料名称 | 说明 |
|---------|------|
| 银行转账记录 | 去银行打印带公章的流水，或手机银行截图 |
| 支付宝/微信/云闪付转账记录 | 截图保存，包含转账时间、金额、对方账号 |
| 现金收条/收据 | 原件或清晰复印件 |
| 发票 | 增值税发票、普通发票等 |
| 对账单/结算单 | 双方确认的款项往来明细 |

### 5. 履行证据 ⭐

| 材料名称 | 说明 |
|---------|------|
| 送货单/收货单 | 证明货物已交付 |
| 验收单/验收报告 | 证明对方已验收 |
| 物流/快递单据 | 证明物品已寄出/收到 |
| 现场照片/视频 | 证明履行状态或现场状况 |
| 交接清单/确认书 | 双方签字确认的交接记录 |

### 6. 违约/损害证据 ⭐

| 材料名称 | 说明 |
|---------|------|
| 催告函/律师函及送达证明 | 证明已催告对方履行 |
| 违约通知/解除通知 | 证明已告知对方违约事实 |
| 损害照片/视频 | 证明损失的具体情况 |
| 维修/重做单据及费用 | 证明为减少损失支出的费用 |
| 医疗记录/诊断证明 | 人身损害案件中证明伤情 |
| 报警记录/出警记录 | 证明曾报警处理 |

### 7. 其他辅助材料 ○

| 材料名称 | 说明 |
|---------|------|

## 四、材料缺失补救指南

| 缺失材料 | 补救方法 |
|---------|---------|
| 银行流水 | 本人带身份证去银行柜台打印 |
| 微信聊天记录 | 微信→设置→聊天→聊天记录迁移与备份 |
| 转账记录 | 支付宝/微信账单中导出 |
| 对方身份信息 | 如知道姓名+大致地址，可委托律师调取；企业信息可在"国家企业信用信息公示系统"查询 |
| 合同原件丢失 | 查找扫描件、复印件、拍照；或找对方索要 |
| 通话录音 | 手机自带录音功能；重要通话前开启录音 |
| 事故现场证据 | 调取附近监控（尽快，监控可能定期覆盖） |
| 病历/医疗票据丢失 | 去医院病案室复印，加盖医院公章 |
