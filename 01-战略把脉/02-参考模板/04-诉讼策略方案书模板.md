## 附录：原告方/被告方专项检查清单

### 原告方（Builder）检查清单

```markdown
## 附录A：原告方策略检查清单

- [ ] **请求权基础**已明确且稳固
- [ ] **诉讼请求**具体、明确、可执行（金额计算有依据）
- [ ] **证据链条**完整，能覆盖所有待证事实
- [ ] **被告身份信息**准确（如为公司，确认其存续状态）
- [ ] **管辖法院**正确（合同约定/被告住所地/合同履行地）
- [ ] **诉讼时效**未过（或已找到中断时效的证据）
- [ ] **财产保全**必要性已评估，财产线索已掌握
- [ ] **律师函**是否先发已决策
- [ ] **诉讼费用**已计算，当事人已确认承担
- [ ] **和解底线**已明确
```

### 被告方（Breaker）检查清单

```markdown
## 附录B：被告方策略检查清单

- [ ] 已收到并仔细阅读原告**起诉状及全部证据**
- [ ] **答辩期**已确认（收到起诉状副本之日起15日内）
- [ ] **管辖权异议**已评估（如需要，在答辩期内提出）
- [ ] **诉讼时效**已核查（如已过时效，作为核心抗辩）
- [ ] 原告**主体资格**已审查（是否有权起诉）
- [ ] 原告诉状中的**事实漏洞**已标记
- [ ] 原告证据中的**矛盾点/瑕疵**已标记
- [ ] **抗辩权基础**已明确（程序抗辩+实体抗辩）
- [ ] 是否提起**反诉**已决策（如需反诉，在答辩期内提出）
- [ ] **证据突袭**防范方案已准备
- [ ] **和解空间**已评估
```
